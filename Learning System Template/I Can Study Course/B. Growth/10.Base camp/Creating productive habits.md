# Creating productive habits
## PEER-Peer system

Now, we will introduce a system to ensure we curb our neuroticism. Remember, we want motivation to enhance our performance, but we don’t want to be dependent on motivation.

We will use the PEER-Peer system to execute actions. It has a low learning curve, is easy to set up, and you will see benefits immediately.

### Alert

The PEER-Peer system is considered a core pillar technique of the program.
