# Prechunking

Prechunking is when we create some chunks _before_ engaging in inquiry.

In many cases, this is not advisable as it can bias the chunks we form. However, this is not problematic if we make sure our chunk structures are only hypotheses during the entire chunking process.

_Problems arise when the prechunks do not serve as a springboard for thinking, but rather are assumed to be correct and learners are resistant or hesitant to modify it._

General advice about prechunking

- Prechunking is sometimes acceptable and saves time, but only if the chunks remain hypotheses and continue to be modified.
- If there is any difficulty with avoiding chunking in simply the same ways information is presented (i.e. engaging order control), avoid prechunking.
- If there are any issues with spiderwebbing or the Aim step feels ineffective, avoid prechunking.

In the demonstrations (both in the sublessons below as well as in Live Clinics) you will notice that sometimes prechunking is used and sometimes it is not.

**What matters is that chunks are formed independently and challenged with good inquiry.**
