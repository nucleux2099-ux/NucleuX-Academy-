# _Rapid Start

> [!info] Stage focus
> Stabilise the enablers first. Rapid Start clears urgency traps, rebuilds focus, and sets the practice rhythm that keeps the rest of the program sustainable.

## How to work this stage

- Keep theory blocks light (2–3 hours total) and immediately schedule matching practice time.
- Run the BEDS-M checklist before deep work sessions to normalise good focus hygiene.
- Capture the mindset and scheduling insights in a Kolb log so you can reuse them when workload spikes.

2. BEDS-M adjustments feel normal (environment, distraction sweeps, ritualised start).
