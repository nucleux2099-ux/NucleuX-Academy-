## 🎨 Direct vs Indirect — The Key Distinction

```
                INDIRECT                          DIRECT
                   │                                 │
                   ▼                                 ▼
    ┌──────────────────────────┐    ┌──────────────────────────┐
    │ • LATERAL to epigastrics │    │ • MEDIAL to epigastrics  │
    │ • Through DEEP RING      │    │ • Through FLOOR          │
    │ • CONGENITAL (patent PV) │    │ • ACQUIRED (weakness)    │
    │ • Can descend to scrotum │    │ • Rarely into scrotum    │
    │ • More common overall    │    │ • More common in elderly │
    │ • Sac = peritoneum       │    │ • Sac = peritoneum       │
    └──────────────────────────┘    └──────────────────────────┘
    
    CLINICAL TEST: Deep Ring Occlusion Test
    ─────────────────────────────────────────
    1. Reduce the hernia
    2. Press firmly over deep ring (midpoint of inguinal ligament)
    3. Ask patient to cough
    
    • Hernia controlled → INDIRECT
    • Hernia still appears → DIRECT
```

## 🧪 The Examination Sequence

```
INSPECTION
    │
    ├── Standing position essential
    ├── Both groins exposed
    ├── Look for: Swelling, scars, skin changes
    │
    ▼
COUGH IMPULSE
    │
    ├── Visible impulse on coughing
    ├── Palpable impulse (hand over swelling)
    │
    ▼
PALPATION
    │
    ├── Get above it? (Yes = inguinal, No = inguinoscrotal)
    ├── Reducible? (Yes = simple, No = irreducible/strangulated)
    ├── Tender? (No = simple, Yes = ?strangulation)
    │
    ▼
SPECIAL TESTS
    │
    ├── Deep ring occlusion test
    ├── Invagination test (finger in scrotum)
    ├── Zieman's technique (three-finger test)
    │
    ▼
CONTRALATERAL SIDE
    │
    └── Always examine! (20% bilateral)
```

## 💡 Key Atomic Concepts

### Atom 1: Patent Processus Vaginalis
> The processus vaginalis normally obliterates after testicular descent. If it remains open → indirect inguinal hernia pathway. This is why indirect hernias are considered congenital.

### Atom 2: The Myopectineal Orifice (Fruchaud)
> All groin hernias (inguinal and femoral) occur through ONE weakness — the myopectineal orifice. Understanding this unified concept revolutionized laparoscopic repair.

### Atom 3: Tension-Free Repair
> Suturing tissue under tension → high recurrence. Lichtenstein's mesh repair = tension-free = <1% recurrence. This changed hernia surgery forever.
