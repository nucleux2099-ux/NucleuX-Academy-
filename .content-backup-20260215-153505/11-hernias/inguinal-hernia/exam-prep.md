# 📋 Inguinal Hernia — Exam Prep Mode

> **High-yield facts for NEET-PG, INICET, and university exams**

## 🎯 One-Liners (Most Asked)

| Question | Answer |
|----------|--------|
| Most common hernia | Inguinal hernia |
| Most common type of inguinal hernia | Indirect (60%) |
| Most common hernia in females | Indirect inguinal (NOT femoral!) |
| Content of inguinal canal in males | Spermatic cord |
| Content of inguinal canal in females | Round ligament of uterus |
| Boundaries of Hesselbach's triangle | Inferior epigastric vessels (lateral), Rectus sheath (medial), Inguinal ligament (inferior) |
| Direct hernia occurs through | Hesselbach's triangle (posterior wall weakness) |
| Indirect hernia occurs through | Deep inguinal ring |
| Deep ring location | Midpoint of inguinal ligament |
| Superficial ring location | Above and medial to pubic tubercle |
| Gold standard repair | Lichtenstein tension-free mesh repair |
| Most common complication of mesh repair | Chronic pain |

## 📝 Mnemonics

### NAVEL — Contents of Spermatic Cord (Medial to Lateral)
- **N** — Nerve (ilioinguinal - outside cord, genital branch of genitofemoral - inside)
- **A** — Artery (testicular, cremasteric, artery to vas)
- **V** — Vein (pampiniform plexus)
- **E** — Empty (processus vaginalis - remnant)
- **L** — Lymphatics

### MALT — Boundaries of Inguinal Canal
- **M** — Medial = Conjoint tendon + lateral edge rectus
- **A** — Anterior = External oblique aponeurosis
- **L** — Lateral = Internal oblique (arching fibers)
- **T** — (Top/Trunk) = Floor: Inguinal ligament; Roof: Conjoint tendon

## 🔬 Direct vs Indirect — Quick Comparison

| Feature | Indirect | Direct |
|---------|----------|--------|
| Relation to epigastric vessels | LATERAL | MEDIAL |
| Ring involved | Deep ring | Floor (Hesselbach's) |
| Etiology | Congenital (patent PV) | Acquired (weakness) |
| Age group | Young adults | Elderly |
| Reaches scrotum | YES (common) | Rarely |
| Neck of sac | Narrow | Wide |
| Strangulation risk | Higher | Lower |
| Deep ring occlusion test | Controls hernia | Doesn't control |

## 🏥 Types of Inguinal Hernia Repairs

| Repair | Key Features | Recurrence |
|--------|--------------|------------|
| **Bassini** | Floor repair, sutures | 5-10% |
| **Shouldice** | 4-layer floor repair | 1-2% (expert hands) |
| **Lichtenstein** | Mesh, tension-free | <1% |
| **Desarda** | No mesh, uses EO flap | 1-2% |
| **TEP** | Totally extraperitoneal, laparoscopic | <1% |
| **TAPP** | Transabdominal preperitoneal, laparoscopic | <1% |

## ⚠️ Complications

### Of Hernia
- **Irreducibility** — Cannot reduce, not tender
- **Obstruction** — Intestinal obstruction signs
- **Strangulation** — Compromised blood supply, EMERGENCY
- **Richter's hernia** — Only part of bowel wall (anti-mesenteric border)
- **Littre's hernia** — Meckel's diverticulum in sac
- **Sliding hernia** — Viscus forms part of sac wall

### Of Repair
- **Early:** Hematoma, seroma, wound infection
- **Late:** Chronic pain (most common), recurrence, mesh infection, testicular atrophy

| Type | Description | Key Point |
|------|-------------|-----------|
| **Pantaloon hernia** | Both direct + indirect | Straddles inferior epigastric |
| **Sliding hernia** | Cecum/sigmoid forms sac wall | "Look before you cut" |

## 💊 Management Algorithm

```
INGUINAL HERNIA
      │
      ├── REDUCIBLE
      │       │
      │       └── Elective repair (mesh-based preferred)
      │           • Fit patient → Laparoscopic (TEP/TAPP)
      │           • Open → Lichtenstein
      │
      ├── IRREDUCIBLE (not tender)
      │       │
      │       └── Urgent repair within 24-48 hours
      │
      └── STRANGULATED (tender, signs of ischemia)
              │
              └── EMERGENCY surgery
                  • Resuscitate first
                  • Open approach preferred
                  • Assess bowel viability
                  • Resect if gangrenous
```

## 📝 Quick Revise Before Exam

- Indirect = Lateral to epigastrics, through deep ring, congenital
- Direct = Medial to epigastrics, through floor, acquired
- Hesselbach = Epigastric vessels + Rectus + Inguinal ligament
- Deep ring = Midpoint of inguinal ligament
- Lichtenstein = Gold standard open repair
- TEP = Totally Extraperitoneal (no peritoneum entry)
- TAPP = Transabdominal Preperitoneal (enters peritoneum)
- Strangulation = EMERGENCY
