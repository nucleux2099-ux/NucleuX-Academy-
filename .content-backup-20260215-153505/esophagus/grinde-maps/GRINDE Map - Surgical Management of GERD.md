## R - Recall Mnemonics

### Pre-op Workup - "HEMP"

```
Before fundoplication, do "HEMP":

H - HRM (High-Resolution Manometry)
    → Rule out achalasia, assess motility

E - EGD (Esophagogastroduodenoscopy)
    → Grade esophagitis, rule out Barrett's/mass

M - Monitoring pH (24-hour pH/impedance)
    → Confirm pathological reflux

P - Pictures (Barium swallow)
    → Assess hernia, esophageal length
```

### Floppy Nissen Principles - "SCUFF"

```
┌─────────────────────────────────────────────────┐
│                                                  │
│  S - SHORT wrap (2-3 cm, not >3)                │
│                                                  │
│  C - CALIBRATED (50-56 Fr bougie)               │
│                                                  │
│  U - UNTETHERED fundus (divide short gastrics)  │
│                                                  │
│  F - FUNDUS-to-fundus (not gastric body)        │
│                                                  │
│  F - FIXED to esophagus (include muscle)        │
│                                                  │
└─────────────────────────────────────────────────┘
```

### Wrap Selection - "Nissen Normal, Toupet Tiny"

```
Motility-based selection:

NISSEN (360°) = NORMAL motility
                N for N!

TOUPET (270°) = TINY (weak) motility
                T for T!

DOR (180°) = DOes myotomy
             Used after Heller myotomy
```

### Hiatal Hernia Types - "Simple, Para, Mixed, Monster"

```
TYPE I   = SIMPLE sliding (95%)
           → EGJ slides up

TYPE II  = PARAESOPHAGEAL
           → Fundus beside esophagus, EGJ in place

TYPE III = MIXED (I + II)
           → Most common large hernia

TYPE IV  = MONSTER (other organs)
           → Colon, spleen, omentum herniate
```

## I - Integrative Connections

### Motility-Wrap Selection

```
MOTILITY AND WRAP CHOICE
════════════════════════

HRM Finding                    Recommended Wrap
───────────────────────────────────────────────────
Normal peristalsis       →     NISSEN (360°)
(DCI 450-8000)                 Full wrap, best control

IEM (>70% ineffective)   →     TOUPET (270°)
                               Less dysphagia

Absent contractility     →     TOUPET or DOR
                               Minimal wrap

Post-myotomy             →     DOR (180°)
(achalasia)                    Covers myotomy site

Scleroderma              →     TOUPET
                               Need some wrap but
                               avoid total

Previous failed Nissen   →     TOUPET or DOR
                               Downgrade wrap
```

### Surgical Options by Clinical Scenario

```
CLINICAL DECISION MATRIX
════════════════════════

Scenario                        Procedure
────────────────────────────────────────────────────
Typical GERD, normal motility   Laparoscopic Nissen +
                                crural repair

GERD + weak motility            Laparoscopic Toupet +
                                crural repair

Large hiatal hernia (>5 cm)     Hiatal repair +
                                fundoplication ± mesh

Short esophagus                 Collis gastroplasty +
                                fundoplication

Morbid obesity + GERD           Roux-en-Y gastric bypass
                                (NOT fundoplication)

Barrett's + obesity             Roux-en-Y gastric bypass

Failed fundoplication           Redo with takedown +
                                Toupet (usually)

GERD + achalasia s/p POEM       Medical management OR
                                later fundoplication
```

### Common Complications

```
COMPLICATIONS CHECKLIST
═══════════════════════

INTRAOPERATIVE:
□ Esophageal perforation (1-2%)
□ Gastric perforation
□ Splenic injury (0.5-1%)
□ Pneumothorax
□ Vagal nerve injury

EARLY POSTOPERATIVE:
□ Dysphagia (30-50%, usually resolves)
□ Gas-bloat syndrome
□ Inability to vomit
□ Diarrhea (if vagal injury)

LATE:
□ Recurrent reflux (10-15%)
□ Wrap herniation
□ Slipped Nissen
□ Persistent dysphagia
□ Paraesophageal hernia

"SLIPPED NISSEN":
• Stomach herniates through wrap
• Causes: Not secured to esophagus, too loose
• Presents: Recurrent reflux, dysphagia
• Treatment: Redo fundoplication
```

## E - Elaboration & Examples

### Case-Based Learning

**Case 1: Standard Candidate**

```
SCENARIO:
42-year-old woman with 10-year GERD history
• Heartburn daily despite twice-daily PPI
• Nocturnal regurgitation
• Wants off medications

WORKUP:
• EGD: LA Grade B esophagitis
• pH study: DeMeester 38 (elevated)
• HRM: Normal peristalsis (DCI 2500)
• Barium: 3 cm sliding hiatal hernia

ANALYSIS:
✓ Confirmed pathological reflux
✓ Normal motility
✓ Moderate hernia
✓ Surgical candidate

RECOMMENDATION:
Laparoscopic NISSEN fundoplication
+ Crural repair
+ Normal motility → 360° wrap

EXPECTED OUTCOME:
• 90%+ symptom relief
• 70-80% off PPI long-term
• Watch for transient dysphagia
```

**Case 2: Weak Motility**

```
SCENARIO:
58-year-old man with GERD
• Persistent symptoms on PPI
• Some difficulty swallowing solids (baseline)

WORKUP:
• EGD: Minimal esophagitis
• pH study: Abnormal acid exposure
• HRM: IEM (75% ineffective swallows)
• Barium: 2 cm sliding hernia

ANALYSIS:
✓ Confirmed reflux
✗ WEAK motility (IEM)
✓ Surgical candidate

RECOMMENDATION:
Laparoscopic TOUPET fundoplication
+ Crural repair
+ Weak motility → 270° wrap (less dysphagia)

EXPECTED OUTCOME:
• 85-90% symptom relief
• Lower dysphagia risk than Nissen
• May still have occasional reflux
```

**Case 3: Obesity + GERD**

```
SCENARIO:
45-year-old woman, BMI 42
• Severe GERD on PPI
• Barrett's esophagus (non-dysplastic)
• Wants surgical solution

ANALYSIS:
⚠ Morbid obesity = Fundoplication NOT ideal
• Higher failure rate
• Technical challenges
• Doesn't address weight

RECOMMENDATION:
Roux-en-Y Gastric Bypass
• Addresses obesity AND GERD
• Excellent reflux control
• May improve Barrett's
• NOT fundoplication in this case!
```

### Clinical Pearls

```
CLINICAL PEARLS
═══════════════

🔹 "Never fundoplicate before you manomerate"
   - HRM before surgery is non-negotiable
   - Achalasia would be disaster

🔹 "Short gastrics must go for floppy flow"
   - Divide short gastrics for tension-free wrap
   - Incomplete mobilization = too tight

🔹 "Toupet for the tiny (peristalsis)"
   - Weak motility needs partial wrap
   - Full wrap + weak motility = dysphagia

🔹 "Crural closure is your friend"
   - Always repair the hiatus
   - Prevents wrap herniation
   - Most common failure mode

🔹 "Dysphagia early is okay, late is not"
   - 30-50% have early dysphagia (edema)
   - >3 months = evaluate for problem

🔹 "BMI >35? Think bypass, not wrap"
   - Fundoplication fails in severe obesity
   - RYGB is better for obese GERD
```
