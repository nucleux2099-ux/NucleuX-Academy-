## R - Recall Mnemonics

### LHM Advantages - "WRAP"

```
W - Wrap included (anti-reflux)
R - Reflux rates lower (10-25%)
A - Abdominal access (can repair hernia)
P - Proven long-term (30+ years data)
```

### POEM Advantages - "FAST"

```
F - Fewer incisions (none external)
A - Adaptable length (extend for Type III)
S - Salvage option (after failed LHM/PD)
T - Time shorter (procedure duration)
```

### Key Numbers - "6-2-DOR, 10-2-MORE"

```
┌─────────────────────────────────────────────────┐
│                                                  │
│  LHM: "6-2-DOR"                                 │
│  ─────────────                                  │
│  6 cm = Esophageal myotomy length               │
│  2-3 cm = Gastric myotomy length                │
│  DOR = Most common fundoplication               │
│                                                  │
│  POEM: "10-2-MORE"                              │
│  ────────────────                               │
│  10 cm = Entry above EGJ                        │
│  2-3 cm = Extension onto stomach                │
│  MORE = Can always extend MORE proximally       │
│                                                  │
└─────────────────────────────────────────────────┘
```

### GERD Rates - "Double Trouble"

```
GERD after treatment:

LHM + Dor:  ~20% (one in five)
POEM:       ~40% (two in five) = DOUBLE

Memory: "POEM without wrap = Double the reflux"
```

### Trial Names - "Werner & Ponds"

```
LANDMARK TRIALS:

WERNER 2019 (NEJM):
• European multicenter RCT
• POEM vs LHM+Dor
• 2-year: Non-inferior efficacy
• POEM: More GERD

PONDS 2019 (JAMA):
• Dutch RCT
• POEM vs PD
• 2-year: POEM superior
• 92% vs 54% success
```

### Common Exam Comparisons

```
FREQUENTLY TESTED COMPARISONS
═════════════════════════════

Question: "POEM vs LHM - main advantage of each?"

LHM Advantage:          POEM Advantage:
────────────────        ─────────────────
Lower GERD rates        Extended myotomy (Type III)
Fundoplication          No external incisions
Long-term data          Effective salvage
Can repair hernia       Shorter procedure

Question: "Why is POEM preferred for Type III?"

Answer: Type III has spastic contractions in
        esophageal body (proximal)
        → Need LONGER myotomy
        → LHM limited to 6-8 cm (exposure)
        → POEM can go 15+ cm (endoscopic)

Question: "Post-POEM management?"

Answer: • PPI long-term (all patients)
        • Surveillance EGD (GERD/Barrett's)
        • Lifestyle modifications
        • Consider fundoplication if PPI-refractory
```

### Procedure Details to Know

```
LHM KEY STEPS:
══════════════
1. Port placement (5 ports, supine/split-leg)
2. Divide gastrohepatic ligament
3. Identify/preserve vagus nerves
4. Clear fat pad
5. Myotomy: 6 cm esophageal + 2-3 cm gastric
6. Verify complete (mucosal bulge)
7. Dor fundoplication (180° anterior)

POEM KEY STEPS:
═══════════════
1. Mucosal incision (10-12 cm above EGJ)
2. Submucosal injection (lift/blue dye)
3. Create submucosal tunnel → 2-3 cm past EGJ
4. Selective circular myotomy in tunnel
5. Close mucosal entry (clips/sutures)
6. No fundoplication performed
```

## E - Elaboration & Examples

### Case-Based Learning

**Case 1: Standard Type II**

```
SCENARIO:
48-year-old male, Type II achalasia
No significant GERD symptoms
Wants minimal downtime

ANALYSIS:
• Type II = Best responder (any treatment works)
• "Minimal downtime" → POEM advantage
• No GERD concern → POEM acceptable

OPTIONS DISCUSSION:
LHM + Dor:
  - 90% success
  - 1-3 day stay
  - Lower GERD
  - 5 small incisions

POEM:
  - 90% success
  - 1-2 day stay
  - Higher GERD
  - No incisions

RECOMMENDATION: Either acceptable
Patient chose POEM for cosmesis/recovery
Counsel about PPI need long-term
```

**Case 2: Type III Challenge**

```
SCENARIO:
55-year-old female, Type III achalasia
Chest pain prominent
Previous failed PD 1 year ago

ANALYSIS:
• Type III = POEM preferred
• Failed PD = POEM effective salvage
• Chest pain = spastic component

WHY NOT LHM?
• Type III needs extended myotomy
• LHM limited to 6-8 cm
• Spastic contractions are proximal

RECOMMENDATION: POEM
• Can extend myotomy 15+ cm
• Effective after failed PD
• Address spastic body
• Expected success: 85-90%
```

**Case 3: GERD-Concerned Young Patient**

```
SCENARIO:
28-year-old female, Type I achalasia
Strong family history of Barrett's
Doesn't want to take PPI long-term

ANALYSIS:
• Young = Long life ahead
• Barrett's concern = GERD matters
• Anti-PPI preference = Need good reflux control

RECOMMENDATION: LHM + Dor
• Fundoplication provides barrier
• 15-25% GERD vs 40-50% POEM
• Long-term data available
• Can address any hiatal hernia

Counsel:
• Still may need PPI occasionally
• But lower baseline risk
• Surveillance still recommended
```

### Clinical Pearls

```
CLINICAL PEARLS
═══════════════

🔹 "POEM is not myotomy without consequences"
   - Trade external scars for internal reflux
   - 40-50% will have erosive esophagitis
   - All need PPI consideration

🔹 "Type III is POEM territory"
   - Extended myotomy is the key
   - Can't do 15 cm through laparoscopic ports
   - Even if you're an LHM expert

🔹 "Failed LHM? Think POEM"
   - Re-do LHM is technically challenging
   - POEM uses virgin submucosal plane
   - ~80% salvage success

🔹 "Young + GERD concern = LHM"
   - Decades of reflux ahead
   - Barrett's risk accumulates
   - Fundoplication is protective

🔹 "Check the wrap before blaming the myotomy"
   - Recurrent dysphagia after LHM
   - Wrap migration/too tight common
   - Don't redo myotomy unnecessarily
```
