# 🔍 Shock — Explorer Mode

> **The Big Picture:** Shock = inadequate tissue perfusion. Cells are starving for oxygen. Without rapid recognition and treatment, organ failure and death follow. Time is tissue.

---

```

│                                                             │
│  Recognize early → Treat early → Save lives                 │
│  Miss it → Multi-organ failure → Death                      │
│                                                             │
│  "Shock is easier to prevent than to treat"                 │
```

## 🔄 The Shock Cascade

```
    INITIATING EVENT (Hemorrhage/Sepsis/MI/etc.)
                       │
                       ▼
    ┌─────────────────────────────────┐
    │  COMPENSATED SHOCK              │
    │  • Tachycardia                  │
    │  • Vasoconstriction             │
    │  • Catecholamine surge          │
    │  • BP maintained                │
    └──────────────┬──────────────────┘
                   │ (Compensation fails)
                   ▼
    ┌─────────────────────────────────┐
    │  DECOMPENSATED SHOCK            │
    │  • Hypotension                  │
    │  • Organ hypoperfusion          │
    │  • Lactate rising               │
    │  • REVERSIBLE with treatment    │
    └──────────────┬──────────────────┘
                   │ (Without treatment)
                   ▼
    ┌─────────────────────────────────┐
    │  IRREVERSIBLE SHOCK             │
    │  • Multi-organ failure          │
    │  • Cell death                   │
    │  • IRREVERSIBLE even with Rx    │
    │  • Death                        │
    └─────────────────────────────────┘
```

## 💡 Key Atomic Concepts

### Atom 1: Shock Index (HR/SBP)
> Normal = 0.5-0.7. Shock Index >1.0 = significant shock (even with "normal" BP). More sensitive than BP alone. Quick bedside calculation.

### Atom 2: Lactate = Tissue Hypoxia Marker
> Lactate >2 mmol/L = tissue hypoperfusion. Serial lactate trending is better than single value. Lactate clearance predicts survival.

### Atom 3: Golden Hour
> First hour is critical. Early aggressive resuscitation reduces mortality dramatically. "Treat shock, not BP" — perfusion markers (lactate, urine output) matter more than numbers.
