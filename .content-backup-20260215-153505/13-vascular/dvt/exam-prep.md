# 📋 Deep Vein Thrombosis — Exam Prep Mode

---

## 🎯 One-Liners

| Question | Answer |
|----------|--------|
| Virchow's triad | Stasis, Endothelial injury, Hypercoagulability |
| Most common site | Calf veins (soleal sinusoids) |
| Most dangerous site | Iliofemoral DVT (highest PE risk) |
| Gold standard investigation | Compression ultrasound |
| D-dimer role | Rule OUT DVT (if negative + low clinical probability) |
| Homan's sign | Calf pain on dorsiflexion (unreliable) |
| Phlegmasia alba dolens | Massive DVT, white swollen leg |
| Phlegmasia cerulea dolens | Massive DVT, blue leg, venous gangrene risk |
| First-line anticoagulation | LMWH/DOAC, followed by warfarin/DOAC |
| Duration of anticoagulation | 3-6 months (provoked), longer if unprovoked |

---

## 📝 Wells Score for DVT

| Criteria | Points |
|----------|--------|
| Active cancer | +1 |
| Paralysis/recent cast | +1 |
| Bedridden >3 days / surgery <4 weeks | +1 |
| Localized tenderness along deep veins | +1 |
| Entire leg swollen | +1 |
| Calf swelling >3 cm vs other leg | +1 |
| Pitting edema | +1 |
| Collateral superficial veins | +1 |
| Previously documented DVT | +1 |
| Alternative diagnosis likely | -2 |

**Score interpretation:**
| Score | Risk | DVT Probability |
|-------|------|-----------------|
| ≤0 | Low | 5% |
| 1-2 | Moderate | 17% |
| ≥3 | High | 50% |

---

## ⚠️ Prophylaxis (VTE Prevention)

**Mechanical:**
- Early mobilization
- Graduated compression stockings
- Intermittent pneumatic compression

**Pharmacological:**
- LMWH (enoxaparin)
- UFH
- DOACs (rivaroxaban)
- Fondaparinux
