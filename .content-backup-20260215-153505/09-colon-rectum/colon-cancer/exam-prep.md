# 📋 Colorectal Cancer — Exam Prep Mode

## 🎯 One-Liners
| Question | Answer |
|----------|--------|
| Most common GI malignancy | Colorectal cancer |
| Adenoma-carcinoma sequence | Polyp → Dysplasia → Cancer (10-15 years) |
| Screening age (average risk) | 45-50 years |
| Gold standard diagnosis | Colonoscopy with biopsy |
| Tumor marker | CEA (follow-up, not diagnosis) |
| Duke's A | Limited to bowel wall |
| Duke's C | Lymph node involvement |
| Right colon cancer presents as | Anemia, mass, vague pain |
| Left colon cancer presents as | Obstruction, bleeding, change in bowel habit |
| TME (Total Mesorectal Excision) | Standard for rectal cancer |

## 📊 Staging (TNM simplified)
| Stage | Description |
|-------|-------------|
| I | T1-2, N0 |
| II | T3-4, N0 |
| III | Any T, N+ |
| IV | Metastatic |

## 💊 Treatment
- **Surgery** - Oncological resection with margins
- **Adjuvant chemo** - Stage III, high-risk II
- **Neoadjuvant CRT** - Rectal cancer (T3/4, N+)

📚 **Sources:** Bailey & Love Ch 72 | NCCN Guidelines
