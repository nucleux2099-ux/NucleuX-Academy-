# 📋 Gastric Cancer — Exam Prep Mode

## 🎯 One-Liners
| Question | Answer |
|----------|--------|
| Most common type | Adenocarcinoma (90%) |
| Risk factors | H. pylori, diet (smoked/pickled), smoking, atrophic gastritis |
| Virchow's node | Left supraclavicular lymph node |
| Krukenberg tumor | Ovarian metastasis (signet ring cells) |
| Sister Mary Joseph nodule | Umbilical metastasis |
| Blumer's shelf | Pouch of Douglas metastasis (rectal exam) |
| Lauren classification | Intestinal vs Diffuse type |
| Borrmann classification | Gross morphology (I-IV) |
| Treatment for resectable | D2 gastrectomy + Adjuvant chemo |
| Linitis plastica | Diffuse infiltrating "leather bottle" stomach |

## 📊 Spread
- **Direct** - Adjacent organs
- **Lymphatic** - Perigastric, celiac nodes, Virchow's
- **Hematogenous** - Liver, lung
- **Transcoelomic** - Krukenberg, Blumer's shelf

## 💊 Surgery
- **Total gastrectomy** - Proximal tumors
- **Subtotal gastrectomy** - Distal tumors
- **D2 lymphadenectomy** - Standard
