# Alcoholic Liver Disease — Exam Prep

## Quick Summary
Spectrum: steatosis → alcoholic hepatitis → cirrhosis. Acetaldehyde = key toxic metabolite. Mallory-Denk bodies + neutrophilic infiltration + hepatocyte ballooning = alcoholic hepatitis triad. AST:ALT >2:1. Micronodular cirrhosis. GGT = most sensitive marker for alcohol.

## High Yield Points
- ★ **Mallory-Denk bodies** = eosinophilic ropey inclusions = damaged cytokeratin (CK8/18)
- ★ **AST:ALT >2:1** = characteristic (pyridoxine/B6 deficiency impairs ALT more)
- ★ **Macrovesicular steatosis** = large fat vacuoles, nucleus pushed to periphery
- ★ **Neutrophilic "satellitosis"** = neutrophils around damaged hepatocytes (alcoholic hepatitis)
- ★ **Micronodular cirrhosis** (<3 mm, uniform nodules) = alcoholic cirrhosis
- ★ **"Chicken-wire" fibrosis** = perisinusoidal/pericellular pattern (stellate cell collagen)
- ★ **CYP2E1** induction → ↑ ROS + ↑ acetaldehyde production
- ★ **GGT** = most sensitive marker for chronic alcohol use
- ★ **Maddrey DF ≥32** = severe; treat with prednisolone
- ★ **Steatosis is reversible; cirrhosis is NOT**
- ★ **Women** more susceptible at lower alcohol doses

## Mnemonics
**Alcoholic Hepatitis Histology: "MNS"**
- **M**allory-Denk bodies
- **N**eutrophilic infiltration
- **S**teatosis + ballooning

**Alcohol Metabolism Path:**
- **E**thanol → **A**cetaldehyde → **A**cetate ("EAA")
- Enzymes: ADH → ALDH (both generate NADH)
