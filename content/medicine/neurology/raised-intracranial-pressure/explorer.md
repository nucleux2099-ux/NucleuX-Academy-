# Raised Intracranial Pressure — Explorer

## Overview

**Raised intracranial pressure (ICP)** is a neurological emergency that occurs when ICP exceeds **20 mmHg** (normal: 5-15 mmHg). According to the **Monro-Kellie doctrine**, the cranial vault contains three components: brain (80%), CSF (10%), and blood (10%). Increase in any one must be compensated by decrease in another, or ICP rises.

## Key Points

- Normal ICP: **5-15 mmHg**; raised if >20 mmHg
- **Monro-Kellie doctrine** governs ICP dynamics
- **Cushing's triad** = hypertension + bradycardia + irregular respiration (late, ominous sign)
- **Papilledema** = hallmark of chronic raised ICP
- **CN VI palsy** = false localizing sign (long intracranial course)
- Emergency: Brain herniation syndromes
- Treatment: Head elevation, mannitol/hypertonic saline, treat cause

## Definition & Classification

### Causes

| Category | Examples |
|---|---|
| **Space-occupying lesion** | Tumor, abscess, hematoma |
| **Cerebral edema** | Stroke, trauma, hepatic encephalopathy, hyponatremia |
| **Increased CSF** | Hydrocephalus (obstructive/communicating), choroid plexus tumor |
| **Increased blood volume** | Venous sinus thrombosis, AVM, hypertensive encephalopathy |
| **Idiopathic intracranial hypertension** | Young obese women, normal imaging |

### Types of Cerebral Edema

| Type | Mechanism | Example |
|---|---|---|
| **Vasogenic** | BBB disruption → plasma leak into extracellular space | Tumors, abscess, meningitis |
| **Cytotoxic** | Cell swelling from ionic pump failure | Ischemic stroke, hypoxia |
| **Interstitial** | Transependymal CSF leak | Obstructive hydrocephalus |
| **Osmotic** | Osmolar imbalance | Hyponatremia, DKA (rapid correction) |

## Clinical Features

### Symptoms
- **Headache** — worse in morning, bending, coughing, straining (Valsalva)
- **Nausea and projectile vomiting** (especially morning)
- **Visual disturbances** — transient obscurations, blurred vision, diplopia
- **Altered consciousness** — progressive drowsiness to coma

### Signs

| Sign | Significance |
|---|---|
| **Papilledema** | Hallmark; bilateral disc swelling on fundoscopy |
| **CN VI palsy** | False localizing sign (unilateral or bilateral); diplopia |
| **Cushing's triad** | Hypertension, bradycardia, irregular respiration — LATE sign, impending herniation |
| **Sunset sign** | Infants — eyes deviate downward (hydrocephalus) |
| **Bulging fontanelle** | Infants — before suture closure |
| **Increasing head circumference** | Infants |
| **Altered consciousness** | GCS decline |

### Brain Herniation Syndromes

| Type | Structure | Features |
|---|---|---|
| **Uncal (transtentorial)** | Temporal lobe uncus through tentorial notch | **Ipsilateral CN III palsy** (dilated pupil), contralateral hemiparesis, Kernohan's notch (ipsilateral hemiparesis — false localizing) |
| **Subfalcine (cingulate)** | Cingulate gyrus under falx | ACA compression → contralateral leg weakness |
| **Tonsillar** | Cerebellar tonsils through foramen magnum | **Respiratory arrest, death** — most dangerous |
| **Central** | Bilateral downward herniation | Progressive loss of consciousness, bilateral fixed pupils |
| **Upward** | Cerebellum through tentorial notch | Posterior fossa lesion |

## Diagnosis

| Investigation | Purpose |
|---|---|
| **CT brain (urgent)** | First-line; identifies cause (mass, hemorrhage, hydrocephalus, edema) |
| **MRI brain** | Better anatomical detail; for elective evaluation |
| **Fundoscopy** | Papilledema detection |
| **ICP monitoring** | Intraparenchymal or intraventricular catheter (gold standard in ICU) |
| **LP** | **CONTRAINDICATED if SOL/mass effect** (risk of herniation); use for IIH after imaging |

### Signs of Raised ICP on CT
- Effacement of sulci and basal cisterns
- Midline shift
- Hydrocephalus
- Compressed ventricles
- Space-occupying lesion

## Management

### Emergency Management

| Step | Action |
|---|---|
| **ABCs** | Secure airway (GCS ≤8 → intubate) |
| **Head position** | Elevate head of bed 30°, midline (avoid jugular compression) |
| **Osmotherapy** | **Mannitol** 20% (0.5-1 g/kg IV) or **hypertonic saline** (3%, 23.4%) |
| **Hyperventilation** | Target PaCO₂ 30-35 mmHg (short-term bridge only — causes vasoconstriction) |
| **Steroids** | **Dexamethasone** — for vasogenic edema (tumors, abscess); NOT for cytotoxic edema (stroke) |
| **CSF drainage** | EVD (external ventricular drain) for hydrocephalus |
| **Seizure control** | AEDs if seizures |
| **Surgical** | Decompressive craniectomy, tumor excision, hematoma evacuation |

### Medical ICP Management Targets
- ICP <20-22 mmHg
- CPP (cerebral perfusion pressure) >60 mmHg
- CPP = MAP - ICP

### Idiopathic Intracranial Hypertension (IIH)
- Young obese women; headache, visual obscurations, papilledema, CN VI palsy
- Normal MRI (except empty sella, optic nerve sheath distension)
- LP: Opening pressure >25 cmH₂O, normal CSF
- Treatment: Weight loss, **acetazolamide** (reduces CSF production), serial LP
- Optic nerve sheath fenestration or VP shunt if vision threatened

## Complications & Prognosis

### Complications
- **Brain herniation** (most feared) → death
- **Permanent visual loss** (chronic papilledema)
- Cognitive impairment
- Cranial nerve palsies
- Post-surgical complications

### Prognosis
- Depends entirely on underlying cause
- Early recognition and treatment improve outcomes
- Herniation has very poor prognosis if not rapidly reversed
- IIH has good prognosis with weight loss and medical management
