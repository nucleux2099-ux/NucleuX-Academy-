# Peripheral Neuropathy — Explorer

## Overview

**Peripheral neuropathy** refers to damage to the peripheral nervous system causing sensory, motor, and/or autonomic dysfunction. **Diabetes mellitus** is the most common cause worldwide. Presentation ranges from distal symmetric polyneuropathy (most common pattern) to mononeuropathy or polyradiculopathy.

## Key Points

- **Diabetes** = most common cause (distal symmetric sensorimotor)
- "Glove and stocking" distribution = classic pattern
- Classification: by pattern, fiber type, pathology (axonal vs demyelinating), and etiology
- NCS/EMG is the key diagnostic tool
- Management: treat underlying cause + symptomatic pain management

## Definition & Classification

### By Pattern

| Pattern | Description | Examples |
|---|---|---|
| **Distal symmetric polyneuropathy** | Most common; glove-stocking | DM, alcohol, B12 deficiency |
| **Mononeuropathy** | Single nerve | Carpal tunnel (median), Saturday night palsy (radial) |
| **Mononeuritis multiplex** | Multiple individual nerves | Vasculitis, DM, leprosy |
| **Polyradiculopathy** | Nerve roots | GBS, CIDP |

### By Fiber Type

| Type | Features |
|---|---|
| **Large fiber** | Loss of vibration, proprioception, reflexes; sensory ataxia |
| **Small fiber** | Pain, temperature loss; burning pain; autonomic features |
| **Motor** | Weakness, wasting, foot drop |

### By Pathology

| Type | NCS Finding | Examples |
|---|---|---|
| **Axonal** | Reduced amplitude, normal velocity | DM, alcohol, drugs, uremia |
| **Demyelinating** | Reduced velocity, prolonged distal latency | GBS, CIDP, Charcot-Marie-Tooth |

## Etiology & Pathophysiology

### Common Causes — "DANG THERAPIST"

- **D**iabetes mellitus (most common)
- **A**lcohol
- **N**utritional (B12, B1, B6 excess)
- **G**BS
- **T**oxins (lead, arsenic, mercury)
- **H**ereditary (CMT)
- **E**ndocrine (hypothyroidism)
- **R**enal failure (uremia)
- **A**myloidosis
- **P**orphyria
- **I**nfections (HIV, leprosy, diphtheria)
- **S**ystemic (vasculitis, sarcoidosis)
- **T**umor (paraneoplastic)

### Diabetic Neuropathy Types

| Type | Features |
|---|---|
| **Distal symmetric polyneuropathy** | Most common; sensory > motor; glove-stocking |
| **Diabetic amyotrophy** | Painful proximal leg weakness; femoral nerve; self-limiting |
| **Autonomic neuropathy** | Gastroparesis, orthostatic hypotension, erectile dysfunction |
| **Mononeuropathy** | CN III palsy (pupil-sparing), carpal tunnel |

## Clinical Features

### Sensory Symptoms
- **Positive**: Burning pain, tingling, paresthesias, allodynia
- **Negative**: Numbness, loss of vibration/proprioception
- **Distribution**: Distal > proximal, "glove and stocking"
- **Sensory ataxia**: Romberg positive (large fiber disease)

### Motor Symptoms
- Distal weakness (foot drop, wrist drop)
- Wasting of intrinsic hand/foot muscles
- Absent or reduced reflexes
- Claw hand, pes cavus (in chronic neuropathy like CMT)

### Autonomic Features
- Postural hypotension, resting tachycardia
- Anhidrosis, gustatory sweating
- Gastroparesis, constipation/diarrhea
- Erectile dysfunction, bladder dysfunction

## Diagnosis

### Investigations

| Test | Purpose |
|---|---|
| **NCS/EMG** | Distinguish axonal vs demyelinating; localize lesion |
| **Blood glucose/HbA1c** | Diabetes |
| **B12, folate, methylmalonic acid** | Nutritional deficiency |
| **TFTs** | Hypothyroidism |
| **Renal function** | Uremic neuropathy |
| **ESR, ANA, ANCA, RF** | Vasculitis, autoimmune |
| **SPEP/UPEP** | Paraproteinemia |
| **HIV serology** | HIV neuropathy |
| **Nerve biopsy** | Vasculitis, amyloidosis, leprosy (sural nerve) |
| **Skin biopsy** | Small fiber neuropathy (intraepidermal nerve fiber density) |

## Management

### Treat Underlying Cause
- **DM**: Glycemic control (prevents progression, doesn't reverse)
- **B12 deficiency**: IM cyanocobalamin
- **Alcohol**: Cessation + B1 supplementation
- **GBS/CIDP**: IVIg or plasmapheresis
- **Vasculitis**: Immunosuppression

### Neuropathic Pain Management

| Line | Drug | Notes |
|---|---|---|
| **First-line** | **Pregabalin** or **Gabapentin** | Calcium channel α2δ ligands |
| **First-line** | **Duloxetine** or **Amitriptyline** | SNRI / TCA |
| **Second-line** | **Tramadol** | Weak opioid |
| **Topical** | **Capsaicin** cream, lidocaine patch | For localized pain |
| **Refractory** | Combination therapy, referral to pain clinic | |

### Foot Care (Diabetic Neuropathy)
- Daily foot inspection
- Appropriate footwear
- Regular podiatry
- Avoid walking barefoot
- Early treatment of ulcers

## Complications & Prognosis

### Complications
- **Diabetic foot ulcers** → gangrene → amputation
- Charcot arthropathy (neuropathic joint)
- Falls and injuries (proprioceptive loss)
- Chronic pain and disability
- Autonomic dysfunction

### Prognosis
- Depends on etiology — reversible causes (B12, GBS) have good prognosis
- Diabetic neuropathy: progressive if glycemic control poor
- Hereditary neuropathies: slowly progressive, no cure
- GBS: 85% achieve full functional recovery
