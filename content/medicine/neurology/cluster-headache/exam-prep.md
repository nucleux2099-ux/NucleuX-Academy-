# Cluster Headache — Exam Prep

## Quick Summary

Excruciating strictly unilateral periorbital pain (15-180 min) with ipsilateral autonomic features + restlessness. Male predominance 3-5:1. Acute: high-flow O2 + SC sumatriptan. Preventive: verapamil. Hypothalamic pacemaker dysfunction.

## High Yield Points ★

- ★ **Most painful** primary headache ("suicide headache")
- ★ Strictly **unilateral** periorbital pain with **autonomic features**
- ★ Duration **15-180 min**, 1-8 attacks/day
- ★ **Restless/agitated** (vs migraine = lies still in dark room)
- ★ **Male predominance** (unlike migraine which is female predominant)
- ★ **High-flow O2** (12-15 L/min) = first-line acute treatment
- ★ **SC sumatriptan** (NOT oral — too slow)
- ★ **Verapamil** = first-line preventive (monitor ECG)
- ★ **Hypothalamus** = the pacemaker (posterior hypothalamic grey matter)
- ★ **Alcohol** triggers attack during cluster period only
- ★ Partial **Horner's syndrome** (miosis + ptosis) may persist between attacks

## Mnemonics

- **Cluster = "CRAM"** — Circadian pattern, Restlessness, Autonomic features, Male predominance
- **Autonomic features**: "CRoSS LiP" — Conjunctival injection, Rhinorrhea, Sweating, Smaller pupil (miosis), Lacrimation, Ptosis

## Differential Diagnosis

| Condition | Differentiating Feature |
|---|---|
| **Migraine** | 4-72 hrs, nausea/vomiting, lies still, female predominance |
| **Paroxysmal hemicrania** | Shorter attacks (2-30 min), more frequent (>5/day), responds to **indomethacin** |
| **SUNCT/SUNA** | Very brief (1-600 sec), very frequent, orbital with autonomic features |
| **Trigeminal neuralgia** | Electric shock-like, seconds, triggered by touch/eating |
| **Acute glaucoma** | Red eye, hazy cornea, raised IOP, mid-dilated pupil |
| **Cavernous sinus pathology** | Multiple cranial nerve palsies, proptosis |
