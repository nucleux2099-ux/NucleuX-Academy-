# Chronic Kidney Disease

> 📚 **Sources:** Harrison's 22nd Ed Ch. 311, KDIGO CKD Guidelines 2012
> **Level:** UG | PG

## Core Concept

Chronic Kidney Disease (CKD) is a progressive, irreversible decline in kidney function persisting for ≥3 months. It's defined by GFR <60 mL/min/1.73m² OR markers of kidney damage (albuminuria, structural abnormalities, history of transplant). Diabetes and hypertension account for 60-70% of cases. CKD leads to uremia, fluid/electrolyte imbalances, anemia, bone disease, and cardiovascular complications. Management focuses on slowing progression and managing complications.

## Key Points

- **Definition:** GFR <60 for ≥3 months OR kidney damage markers
- **Leading causes:** Diabetes (40%), Hypertension (25%), Glomerulonephritis (10%)
- **CKD-MBD:** Secondary hyperparathyroidism, renal osteodystrophy
- **Anemia:** EPO deficiency + functional iron deficiency
- **ACEi/ARBs:** First-line for proteinuric CKD, slow progression

## CKD Staging (KDIGO)

### By GFR

| Stage | GFR (mL/min/1.73m²) | Description |
|-------|---------------------|-------------|
| **G1** | ≥90 | Normal or high (with kidney damage) |
| **G2** | 60-89 | Mildly decreased |
| **G3a** | 45-59 | Mildly to moderately decreased |
| **G3b** | 30-44 | Moderately to severely decreased |
| **G4** | 15-29 | Severely decreased |
| **G5** | <15 | Kidney failure (ESKD) |

### By Albuminuria

| Category | ACR (mg/g) | Description |
|----------|------------|-------------|
| **A1** | <30 | Normal to mildly increased |
| **A2** | 30-300 | Moderately increased (microalbuminuria) |
| **A3** | >300 | Severely increased (macroalbuminuria) |

## Etiology

### Leading Causes
| Cause | Percentage |
|-------|------------|
| **Diabetic nephropathy** | 40% |
| **Hypertensive nephrosclerosis** | 25% |
| **Glomerulonephritis** | 10% |
| **Polycystic kidney disease** | 5% |
| **Others** | 20% (interstitial, obstructive, unknown) |

## Clinical Features

### Uremic Syndrome (Stage 4-5)

| System | Manifestation |
|--------|---------------|
| **Neurological** | Encephalopathy, asterixis, neuropathy, restless legs |
| **Cardiovascular** | Pericarditis, cardiomyopathy, accelerated atherosclerosis |
| **Hematological** | Anemia, platelet dysfunction, bleeding |
| **GI** | Anorexia, nausea, vomiting, GI bleeding |
| **Skin** | Pruritus, uremic frost, sallow complexion |
| **Metabolic** | Acidosis, hyperkalemia, hypocalcemia |
| **Bone** | Osteodystrophy (CKD-MBD) |

### CKD-Mineral Bone Disease (CKD-MBD)

**Pathophysiology:**
1. ↓GFR → ↓Phosphate excretion → Hyperphosphatemia
2. ↓1,25-OH Vitamin D → ↓Calcium absorption → Hypocalcemia
3. Hypocalcemia + Hyperphosphatemia → Secondary hyperparathyroidism
4. High PTH → Bone resorption → Renal osteodystrophy

**Types of Bone Disease:**
| Type | Characteristics |
|------|-----------------|
| **Osteitis fibrosa cystica** | High-turnover, ↑↑PTH |
| **Adynamic bone disease** | Low-turnover, over-suppressed PTH |
| **Osteomalite** | Vitamin D deficiency |
| **Mixed uremic osteodystrophy** | Combination |

## Diagnosis

### Workup
- **GFR estimation:** CKD-EPI equation preferred
- **Urinalysis:** Proteinuria, casts, sediment
- **UACR:** Quantify albuminuria
- **Ultrasound:** Kidney size (small = chronic), hydronephrosis
- **Labs:** Cr, BUN, electrolytes, CBC, Ca, PO4, PTH, Vitamin D

### Small Kidneys (<9 cm)
- Suggests chronicity and irreversibility
- Exception: Diabetic nephropathy, APKD, HIV, amyloid → normal/large kidneys

## Treatment

### Slow Progression

| Intervention | Details |
|--------------|---------|
| **BP control** | Target <130/80 (or <120/80 if tolerated) |
| **ACEi/ARBs** | First-line, especially if proteinuria; monitor K+ and Cr |
| **SGLT2 inhibitors** | Dapagliflozin, empagliflozin - renal protection even in non-diabetics |
| **Glycemic control** | HbA1c <7% in diabetics |
| **Dietary protein** | 0.8 g/kg/day in advanced CKD |
| **Avoid nephrotoxins** | NSAIDs, aminoglycosides, contrast |

### Manage Complications

| Complication | Treatment |
|--------------|-----------|
| **Anemia** | ESAs (epoetin, darbepoetin) + Iron; target Hb 10-11.5 |
| **Hyperkalemia** | Low K diet, kayexalate, patiromer, lokelma |
| **Acidosis** | Sodium bicarbonate (target HCO3 >22) |
| **Hyperphosphatemia** | Phosphate binders (sevelamer, calcium acetate) |
| **Secondary hyperPTH** | Calcitriol, cinacalcet |
| **Volume overload** | Loop diuretics, fluid restriction |

### Renal Replacement Therapy (RRT)

**Indications to start dialysis:**
- GFR <10 mL/min (or <15 in diabetics)
- Uremic symptoms despite conservative management
- Refractory hyperkalemia, acidosis, fluid overload

**Options:**
| Modality | Characteristics |
|----------|-----------------|
| **Hemodialysis** | 3-4x/week, 3-4 hours, vascular access (AV fistula best) |
| **Peritoneal dialysis** | Daily, home-based, via Tenckhoff catheter |
| **Transplantation** | Best long-term survival, preemptive if possible |

## High-Yield Points

1. **ACEi/ARBs** can cause acute ↑Cr (up to 30% acceptable) - don't stop unless hyperkalemia
2. **SGLT2i** = new standard of care for CKD (even non-diabetic)
3. **Small kidneys** on ultrasound = CKD (irreversible)
4. **Erythropoietin target:** Hb 10-11.5 (not normal!) - higher increases CV risk
5. **CKD G3a/A3** = same risk as **G4/A1** for progression and CV events
6. **Dialysis access:** AV fistula > AV graft > catheter (infection risk)
