# HIV/AIDS — Explorer

## Overview

**Human Immunodeficiency Virus (HIV)** is a retrovirus that targets **CD4+ T lymphocytes**, leading to progressive immunodeficiency and **Acquired Immunodeficiency Syndrome (AIDS)**. India has the third-largest HIV epidemic globally. ART has transformed HIV into a manageable chronic disease.

📚 Harrison's, 22nd Ed, Ch. 202-203

## Key Points

- **HIV-1** (most common worldwide) and HIV-2 (West Africa; less virulent)
- Binds **CD4 receptor** + co-receptor (CCR5 for macrophage-tropic; CXCR4 for T-cell-tropic)
- **Acute retroviral syndrome:** 2-4 weeks post-infection; mono-like illness; high viral load, low CD4
- **AIDS defined as:** CD4 <200/μL OR AIDS-defining illness (regardless of CD4)
- **Viral load** = best marker of disease activity and treatment response
- **CD4 count** = best marker of immune status and OI risk
- **ART for ALL** — "test and treat" strategy; start regardless of CD4 count (NACO India)
- **First-line ART (India/NACO):** TDF + 3TC + DTG (Tenofovir + Lamivudine + Dolutegravir)

## Natural History

| Stage | CD4 Count | Duration | Features |
|---|---|---|---|
| **Acute infection** | Transient ↓ | 2-4 weeks post-exposure | Fever, rash, lymphadenopathy, pharyngitis (mono-like) |
| **Clinical latency** | Gradual decline (~50-100/yr) | 8-10 years (untreated) | Asymptomatic; PGL (persistent generalized lymphadenopathy) |
| **AIDS** | <200/μL | Variable | Opportunistic infections, malignancies |

## Opportunistic Infections by CD4 Count (HIGH-YIELD!)

| CD4 Count | Opportunistic Infection | Key Features |
|---|---|---|
| **<500** | Kaposi sarcoma, oral candidiasis, TB, herpes zoster | |
| **<200** | **PCP (Pneumocystis jirovecii)** | Bilateral GGO; ↑ LDH; TMP-SMX |
| **<200** | **Toxoplasma encephalitis** | Ring-enhancing lesion(s); empiric TMP-SMX or pyrimethamine+sulfadiazine |
| **<150** | **Histoplasmosis, Coccidioidomycosis** | Endemic mycoses |
| **<100** | **Cryptococcal meningitis** | Headache; CSF CrAg+; India ink; AmB + flucytosine |
| **<100** | **Cryptosporidiosis** | Chronic watery diarrhea |
| **<50** | **CMV retinitis** | "Pizza pie" / "cottage cheese and ketchup" fundus; IV ganciclovir |
| **<50** | **MAC (M. avium complex)** | Disseminated; fever, weight loss, anemia; azithromycin prophylaxis |
| **<50** | **PML (JC virus)** | Demyelination; no enhancement on MRI; no treatment (improve with ART) |

> **Exam Pearl:** **Toxoplasma** = MULTIPLE ring-enhancing lesions. **Lymphoma (CNS)** = SINGLE ring-enhancing lesion + EBV+. Both in HIV with low CD4.

## Diagnosis

| Test | Window Period | Use |
|---|---|---|
| **ELISA/rapid test** (4th gen Ag/Ab) | 2-4 weeks | Screening |
| **Western blot** | Historical | Confirmatory (replaced by differentiation assay) |
| **p24 antigen** | 2 weeks | Early detection (part of 4th gen assay) |
| **HIV RNA PCR (viral load)** | 10-14 days | Earliest detection; monitoring treatment |
| **CD4 count** | — | Immune status; guides OI prophylaxis |

### NACO Testing Strategy (India)
- **3 rapid tests** (sequential) using different principles/antigens
- If all 3 positive → HIV confirmed
- If discordant → repeat after 2 weeks

## Prophylaxis for OIs

| CD4 Threshold | Prophylaxis | Drug |
|---|---|---|
| **<200** | PCP prophylaxis | **TMP-SMX** (1 DS tab daily) |
| **<200** | Toxoplasma prophylaxis | TMP-SMX (same dose covers both) |
| **<100** | Cryptococcal screening | Serum CrAg → treat if positive |
| **<50** | MAC prophylaxis | Azithromycin 1200mg weekly |

## Management — ART

### Principles
- **Treat ALL** HIV+ patients (regardless of CD4/viral load)
- **Goal:** Undetectable viral load (<50 copies/mL) = untransmittable (U=U)
- Never use **monotherapy** or **dual therapy** (resistance)
- Triple drug therapy minimum (2 NRTIs + 1 third agent)

### First-Line Regimen (NACO India, 2021)

**TLD regimen:** Tenofovir (TDF) 300mg + Lamivudine (3TC) 300mg + Dolutegravir (DTG) 50mg — **single daily pill**

### Drug Classes

| Class | Examples | Mechanism | Key Toxicity |
|---|---|---|---|
| **NRTI** | TDF, 3TC, AZT (zidovudine), ABC | Nucleoside analogue; chain termination | TDF: nephrotoxicity, Fanconi; AZT: bone marrow suppression; ABC: hypersensitivity (HLA-B*5701) |
| **NNRTI** | Efavirenz, Nevirapine | Allosteric RT inhibition | EFV: CNS effects, vivid dreams; NVP: hepatotoxicity, rash (Stevens-Johnson) |
| **PI** | Atazanavir, Lopinavir/ritonavir | Protease inhibition | Lipodystrophy, dyslipidemia, jaundice (ATV) |
| **INSTI** | Dolutegravir, Raltegravir | Integrase inhibition | Weight gain; fewest drug interactions; preferred class |

### Special Situations
- **TB + HIV:** Start ATT first → ART within 2-8 weeks (within 2 weeks if CD4 <50); use EFV-based regimen or double-dose DTG (rifampicin interaction)
- **Pregnancy:** TDF + 3TC + DTG (preferred); avoid EFV in 1st trimester if possible
- **PEP (post-exposure prophylaxis):** TDF + 3TC + DTG × 28 days; start within 72 hours
- **PrEP (pre-exposure prophylaxis):** TDF + FTC daily (or on-demand for MSM)

## IRIS (Immune Reconstitution Inflammatory Syndrome)
- Paradoxical worsening after starting ART (immune system recovers → inflammatory response to existing infection)
- Common with TB, Cryptococcus, CMV, PML
- Management: continue ART; treat OI; steroids if severe
