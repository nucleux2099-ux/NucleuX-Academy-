# Firearm Injuries

> 📚 **Sources:** Reddy's Forensic Medicine (Ch. 15), Parikh's Textbook, Apurba Nandy
> **Level:** UG

## Core Concept

**Firearm injuries** are caused by projectiles propelled by explosive force. Understanding wound characteristics helps determine:
- Type of weapon used
- Range of fire
- Direction of fire
- Entry vs exit wound

## Key Points

### Types of Firearms

| Category | Examples | Projectile |
|----------|----------|------------|
| **Rifled weapons** | Revolver, pistol, rifle | Single bullet |
| **Smooth-bore** | Shotgun | Pellets (shots) |
| **Country-made** | Improvised weapons | Variable |

### Components of Cartridge
1. **Bullet/Projectile** — Lead, jacketed
2. **Propellant** — Gunpowder (smokeless/black)
3. **Primer** — Initiates propellant
4. **Case** — Holds components together

### Range of Fire Effects

**Close/Contact Range:**
- Muzzle touching skin
- Stellate/cruciate wound (gas expansion)
- Burning, blackening
- Cherry-red COHb coloring

**Near/Close Range (up to ~60 cm):**
- Tattooing (unburnt powder embedded)
- Singeing of hair
- Soot/blackening present
- Abrasion collar present

**Intermediate Range (60 cm - 90 cm):**
- Tattooing may be present
- No burning/singeing
- Abrasion collar present

**Distant Range (>90 cm):**
- Only bullet hole
- Abrasion collar
- No tattooing, no burning

### Entry Wound Characteristics

| Feature | Description |
|---------|-------------|
| **Shape** | Round/oval (perpendicular), elliptical (angled) |
| **Size** | Usually smaller than bullet |
| **Margins** | Inverted |
| **Abrasion collar** | Present (friction of bullet) |
| **Grease collar** | Dark ring (lubricant/residue) |
| **Tattooing** | Embedded powder (close range) |
| **Blackening** | Soot deposit (close range) |
| **Burning** | Contact/near-contact |

### Exit Wound Characteristics

| Feature | Description |
|---------|-------------|
| **Shape** | Irregular, stellate |
| **Size** | Usually larger than entry |
| **Margins** | Everted |
| **Abrasion collar** | Absent |
| **Tattooing** | Absent |
| **Bone fragments** | May be present in wound |

### Shotgun Wounds

| Range | Pattern |
|-------|---------|
| **Contact** | Single large wound, stellate |
| **Close (<1m)** | Single hole with scalloped margins |
| **Intermediate (1-3m)** | Central hole + satellite pellet holes |
| **Distant (>3m)** | Scattered pellet holes (spread pattern) |

**Spread Rule:** Pellets spread ~2.5 cm per meter distance

## Medico-Legal Aspects

### Documentation for Firearm Cases
1. Entry wound(s) — location, size, features
2. Exit wound(s) — if present
3. Direction of track
4. Range estimation evidence
5. Recovered projectile(s)
6. Clothing examination
7. Photography (with scale)

### Suicide vs Homicide vs Accident

| Feature | Suicide | Homicide | Accident |
|---------|---------|----------|----------|
| **Range** | Contact/near | Variable | Variable |
| **Site** | Temple, mouth, chest | Any | Hands, feet common |
| **Clothing** | Often displaced | As worn | As worn |
| **Weapon** | Found with body | May be absent | Present |
| **Multiple wounds** | Rare | Common | Rare |

### Beveling in Skull Wounds
- **Entry:** Internal beveling (cone inside)
- **Exit:** External beveling (cone outside)
- Helps determine direction when soft tissue absent

### Ricochet Wounds
- Bullet hits surface before striking body
- Irregular wound
- Metal fragments may be present
- Deformed bullet

## Examination Findings

### Systematic Examination
1. **Document all wounds** (entry, exit, graze)
2. **Note range indicators:**
   - Contact: Muzzle imprint, cruciate wound
   - Close: Tattooing, singeing, soot
   - Distant: Clean hole only
3. **Track direction**
4. **Retain bullet** (if found) — handle carefully, no metal instruments
5. **X-ray before autopsy** — locate projectile

### Differentiating Features

| Feature | Entry | Exit |
|---------|-------|------|
| Margins | Inverted | Everted |
| Abrasion collar | Present | Absent |
| Size | Smaller | Larger |
| Shape | Regular | Irregular |
| Tattooing | May be present | Absent |

### Kennedy Phenomenon (Tandem Bullet)
- Bullet pushes bone fragments ahead
- Exit wound shows bone fragments in wound
- Bone may cause additional injury

## High-Yield for Exams

### Must-Know MCQ Points
1. **Abrasion collar present in:** Entry wound only
2. **Tattooing indicates range:** Close/near (powder embedded)
3. **Tattooing vs Blackening:** Tattooing cannot be wiped, blackening can
4. **Contact wound characteristic:** Muzzle imprint, stellate/cruciate
5. **Exit wound feature:** Everted margins, larger, irregular
6. **Beveling pattern:** Internal beveling = entry; External = exit
7. **Shotgun spread:** ~2.5 cm per meter
8. **Cherry pink wound:** CO in contact range

### Common Terms
| Term | Meaning |
|------|---------|
| **Stippling** | Petechial hemorrhages from unburnt powder impact |
| **Tattooing** | Powder particles embedded in skin |
| **Blackening** | Soot deposit (can be wiped) |
| **Singeing** | Burning of hair |
| **Grease ring** | Bullet lubricant ring |
| **Keyhole wound** | Bullet enters at angle (entry + exit combined) |

### Arms Act 1959
- Regulates firearms and ammunition
- License required for possession
- Violations are punishable offenses
- Applies to manufacture, sale, possession, use

### Common Viva Questions
- Q: How to estimate range of fire?
  - A: Contact = muzzle mark, cruciate; Close = tattooing, singeing, soot; Distant = clean hole
  
- Q: How to differentiate entry from exit?
  - A: Entry has inverted margins, abrasion collar, smaller size; Exit has everted margins, larger, irregular

### Atypical Entry Wounds
- **Contact over bone:** Stellate (temple, sternum)
- **Angled entry:** Oval/elliptical
- **Re-entry wound:** Bullet exits and re-enters
- **Shored exit:** Backed by hard surface = looks like entry
