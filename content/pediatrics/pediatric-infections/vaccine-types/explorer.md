# Vaccine Types and Classification

## Overview
Vaccines are biological preparations that provide active immunity against infectious diseases. Understanding vaccine classification — live attenuated vs inactivated, whole-organism vs subunit, conjugate vs polysaccharide — is essential for rational immunization practice. Each type has different immunogenicity, storage requirements, contraindications, and dosing schedules. The choice of vaccine type impacts cold chain management, safety in immunocompromised patients, and immune response quality.

## Key Points
- **Live attenuated:** Strong immune response, fewer doses; contraindicated in immunodeficiency
- **Inactivated/killed:** Safe in immunocompromised; weaker response, need boosters
- **Conjugate vaccines:** Polysaccharide + protein carrier → T-cell dependent → effective in <2 years
- **Polysaccharide vaccines:** T-independent → poor response in <2 years, no memory
- **Toxoid:** Inactivated toxin → antitoxin immunity (DPT, tetanus)
- **Recombinant:** Genetically engineered antigen (Hepatitis B — HBsAg)
- **mRNA vaccines:** Newest platform (COVID-19 vaccines)
- **Live vaccines need 4-week gap** if not given simultaneously

## Definition & Classification

### Classification by Type
| Type | Mechanism | Examples | Key Features |
|------|-----------|---------|-------------|
| **Live attenuated** | Weakened pathogen replicates → immune response | BCG, OPV, MMR, Rotavirus, Varicella, Yellow fever, JE (SA-14-14-2) | Strong response; fewer doses; risk in immunodeficiency |
| **Inactivated (killed)** | Whole killed pathogen | IPV, Hepatitis A, Rabies, Pertussis (whole cell) | Safe in immunocompromised; needs boosters |
| **Toxoid** | Inactivated toxin | Diphtheria, Tetanus | Produces antitoxin antibodies |
| **Subunit/Recombinant** | Purified/engineered antigen | Hepatitis B (HBsAg), HPV, acellular Pertussis (DTaP) | Very safe; specific immune response |
| **Conjugate** | Polysaccharide + protein carrier | PCV, Hib, Meningococcal conjugate | T-cell dependent; effective <2 years; booster response |
| **Polysaccharide** | Pure capsular polysaccharide | PPSV23, Meningococcal polysaccharide, Typhoid Vi | T-independent; poor in <2 years; no memory |
| **mRNA** | Encodes antigen; host cells produce protein | COVID-19 (Pfizer, Moderna) | Rapid development; cold chain demanding |
| **Viral vector** | Uses harmless virus to deliver antigen gene | COVID-19 (Covishield/AstraZeneca), Ebola | Strong response; may have vector immunity issues |

### Live vs Inactivated — Key Differences
| Feature | Live Attenuated | Inactivated/Killed |
|---------|----------------|-------------------|
| Immune response | Humoral + cell-mediated | Mainly humoral |
| Doses needed | Fewer (1-2) | More (3-4 + boosters) |
| Duration of immunity | Long-lasting | Shorter; needs boosters |
| Reversion risk | Possible (VAPP with OPV) | None |
| Use in immunodeficiency | **Contraindicated** | Safe |
| Use in pregnancy | Generally avoided | Generally safe |
| Storage | More sensitive to heat | More stable (except freeze-sensitive) |
| Interval between 2 live | ≥4 weeks if not simultaneous | No restriction |

### Conjugate vs Polysaccharide
| Feature | Conjugate | Polysaccharide |
|---------|-----------|---------------|
| Immune response | T-cell dependent | T-cell independent |
| Effective in <2 years | **Yes** | **No** |
| Immunological memory | Yes (booster response) | No |
| Mucosal immunity | Reduces carriage | Does not |
| Examples | PCV, Hib, MenACWY conjugate | PPSV23, MenACWY polysaccharide, Typhoid Vi |

## Etiology & Pathophysiology

### Immunological Basis
1. **Active immunity:** Vaccine antigen → immune recognition → memory B and T cells
2. **Primary response:** IgM first, then IgG; slow, low titer
3. **Booster (anamnestic) response:** Rapid, high-titer IgG (memory cells activated)
4. **T-cell dependent antigens** (protein/conjugate): Engage T-helper cells → class switching → IgG → memory
5. **T-cell independent antigens** (polysaccharide): Direct B-cell stimulation → IgM predominant → no memory → poor in <2 years (immature B-cells)
6. **Conjugation:** Coupling polysaccharide to protein converts it to T-dependent antigen

### Adjuvants
- Aluminum salts (alum) — most commonly used; enhances immune response
- Enhance antigen presentation and depot effect at injection site

## Clinical Features

### Vaccine-Specific Considerations
- **BCG:** Only intradermal vaccine; forms papule → ulcer → scar
- **OPV vs IPV:** OPV = oral, live, gut immunity, VAPP risk; IPV = injectable, killed, no VAPP
- **DTwP vs DTaP:** Whole cell pertussis more reactogenic; acellular less reactogenic but more expensive
- **PCV vs PPSV23:** PCV (conjugate) for children; PPSV23 for high-risk >2 years

### Cold Chain Sensitivity
- **Freeze-sensitive:** DPT, Pentavalent, HepB, IPV, Td (killed/toxoid — destroyed by freezing)
- **Heat-sensitive:** OPV (store at -20°C); BCG, MR (2-8°C)
- **VVM (Vaccine Vial Monitor):** Color indicator on vial for heat exposure

## Diagnosis
Not applicable — vaccines are preventive interventions.

## Management

### General Principles of Vaccination
1. Multiple vaccines can be given simultaneously at different sites
2. Live vaccines: give same day or ≥4 weeks apart
3. Inactivated vaccines: no minimum interval with other vaccines
4. Minimum interval between doses of same vaccine must be maintained
5. If schedule interrupted: resume from where left off (do NOT restart)
6. Interchangeability: Same type generally interchangeable between brands

### Contraindications
- **All vaccines:** Anaphylaxis to previous dose or component
- **Live vaccines:** Severe immunodeficiency (HIV with severe immunosuppression, primary immunodeficiency, chemotherapy)
- **Pertussis (whole cell):** Encephalopathy within 7 days → switch to acellular
- **Pregnancy:** Avoid live vaccines (except in special circumstances)

### Special Situations
- **Preterm:** Vaccinate at chronological age (same schedule)
- **HIV:** Avoid BCG if symptomatic; give IPV instead of OPV; other vaccines as per schedule
- **Immunosuppressive therapy:** Avoid live vaccines; give inactivated; revaccinate after recovery

## Complications
- **Local reactions:** Pain, swelling, redness (most common)
- **Systemic:** Fever, irritability, malaise
- **VAPP:** OPV — 1 in 2.4 million doses
- **Anaphylaxis:** Any vaccine (rare — 1 in million)
- **BCG dissemination:** In severe immunodeficiency
- **Intussusception:** Rotavirus (rare — 1 in 20,000-100,000)
- **Febrile seizures:** DPT, MMR (benign)
- **Guillain-Barré:** Influenza vaccine (very rare)
