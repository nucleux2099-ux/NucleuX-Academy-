# Achalasia - LHM vs POEM: Current Evidence

## Overview

Achalasia treatment has evolved significantly with the introduction of Per-Oral Endoscopic Myotomy (POEM) in 2010. This topic compares the two definitive surgical approaches: Laparoscopic Heller Myotomy (LHM) with fundoplication versus POEM, examining the current evidence to guide treatment selection.

```
EVOLUTION OF ACHALASIA TREATMENT
════════════════════════════════

1913: Heller - Original open cardiomyotomy (anterior + posterior)
       │
1962: Modified to single (anterior) myotomy
       │
1991: First Laparoscopic Heller Myotomy (LHM)
       │         └── Added partial fundoplication
       │
2010: Inoue - First POEM in humans (Japan)
       │
2020s: LHM vs POEM trials emerge
       │
Today: Both are first-line options for Types I/II
       POEM preferred for Type III
```

## Laparoscopic Heller Myotomy (LHM)

### Surgical Technique

```
LHM WITH DOR FUNDOPLICATION
════════════════════════════

Position: Supine, split-leg, reverse Trendelenburg

Port Placement:
     ┌─────────────────────┐
     │         ○ (10mm)    │  ← Camera
     │        /│\          │
     │       / │ \         │
     │      ○  │  ○        │  ← Working ports (5mm)
     │     /   │   \       │
     │    ○    │    ○      │  ← Assistant/Liver retractor
     │         │           │
     └─────────────────────┘

Key Steps:
1. Divide gastrohepatic ligament
2. Identify EGJ and crura
3. Clear fat pad at EGJ
4. Create myotomy window
5. Myotomy: 6 cm esophageal + 2-3 cm gastric
6. Verify complete (mucosal bulge)
7. Partial fundoplication (Dor or Toupet)
```

```
MYOTOMY ANATOMY
═══════════════

                    Proximal
                       ↑
                       │

    │   ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓        │  (DIVIDE THIS)
    │                                   │

    │                                   │  (PRESERVE)
    │   ════════════════════════       │

```

### Fundoplication Options

| Type | Wrap Degree | Advantages | Disadvantages |
|------|-------------|------------|---------------|
| **Dor (Anterior)** | 180° anterior | Covers myotomy, easier | Less anti-reflux |
| **Toupet (Posterior)** | 270° posterior | Better anti-reflux, keeps edges apart | Doesn't cover myotomy |
| **None** | 0° | Shorter, simpler | Higher GERD rates |

```
DOR vs TOUPET FUNDOPLICATION
════════════════════════════

DOR (180° Anterior):                TOUPET (270° Posterior):

    Esophagus                           Esophagus
        │                                   │
   ┌────┴────┐                         ┌────┴────┐
   │ ←Fundus │                         │         │
   │   wrap  │                     wrap│    wrap │
   └─────────┘                    ←────┘    ────→│
        │                                   │
    Stomach                             Stomach

• Covers myotomy site              • Keeps myotomy edges apart
• Protects if leak                 • Better anti-reflux effect
• Easier technically               • Does not cover myotomy
```

**Consensus:** Dor is most commonly used for achalasia (covers myotomy)

### POEM Parameters

| Parameter | Standard Approach |
|-----------|-------------------|
| Entry point | 10-12 cm above EGJ |
| Tunnel length | Entry to 2-3 cm past EGJ |
| Myotomy start | 2-3 cm below mucosal entry |
| Myotomy extent | 2-3 cm onto gastric cardia |
| Total myotomy | 10-12 cm (can extend for Type III) |
| Approach | Posterior (5-6 o'clock) or Anterior (2 o'clock) |
| Closure | Endoscopic clips or suturing |

## Head-to-Head Comparison

### Key Randomized Controlled Trials

#### 1. Werner et al. (2019) - European Multicenter RCT

| Parameter | POEM (n=112) | LHM+Dor (n=109) | P-value |
|-----------|--------------|-----------------|---------|
| 2-year success (Eckardt ≤3) | 83% | 81.7% | 0.007 (non-inferior) |
| Reflux esophagitis | 44% | 29% | 0.02 |
| Procedure time | 103 min | 146 min | <0.001 |
| Hospital stay | 3 days | 3 days | NS |
| Serious AE | 2.7% | 7.3% | NS |

#### 2. Ponds et al. (2019) - Dutch RCT

| Parameter | POEM (n=60) | PD (n=66) | P-value |
|-----------|-------------|-----------|---------|
| 2-year success | 92% | 54% | <0.001 |
| Reflux esophagitis | 41% | 7% | <0.001 |
| Re-intervention | 0% | 16% | 0.001 |

#### 3. Multiple Meta-analyses (2020-2024)

```

                    ────          ───

### Comparative Summary Table

| Factor | LHM + Fundoplication | POEM | Winner |
|--------|---------------------|------|--------|
| **Short-term efficacy** | 90-95% | 90-95% | Tie |
| **Long-term efficacy (5yr)** | 85-90% | 80-90% | LHM (slightly) |
| **GERD rate** | 10-25% | 35-50% | LHM |
| **Barrett's risk** | Lower | Higher | LHM |
| **Type III achalasia** | 60-70% | 85-95% | POEM |
| **Procedure time** | Longer | Shorter | POEM |
| **Hospital stay** | 1-3 days | 1-2 days | POEM |
| **Repeat procedure** | More difficult | Possible | POEM |
| **Prior treatment failure** | Less effective | Effective | POEM |
| **Learning curve** | Established | Steep | LHM |
| **External incisions** | Yes (5 ports) | No | POEM |
| **Fundoplication** | Included | None | LHM |

## GERD After Myotomy

### The GERD Problem

```
PATHOPHYSIOLOGY OF POST-MYOTOMY GERD
════════════════════════════════════

Normal EGJ Competence:           After Myotomy:

    ────────────                     ────────────
        │                                │
    ╔═══╧═══╗ LES tone              ╔═══╧═══╗ LES disrupted
    ║       ║                       ║ X X X ║
    ╚═══╤═══╝                       ╚═══╤═══╝
        │                                │
    ════╪════ Diaphragm             ════╪════ May still anchor
        │                                │
        ▼                                ▼

    Anti-reflux mechanisms:         Residual protection:
    1. LES intrinsic tone ✗        1. Diaphragm
    2. His angle                    2. His angle
    3. Phrenoesophageal lig.       3. Gravity
    4. Diaphragm                    4. Saliva clearance

    Fundoplication restores:        POEM: No wrap
    - Wrap acts as flap valve       - Higher GERD rates
    - Augments competence           - Needs PPI long-term
```

### GERD Rates Comparison

| Measure | LHM + Dor | POEM | Significance |
|---------|-----------|------|--------------|
| Reflux symptoms | 10-20% | 20-30% | Higher with POEM |
| Erosive esophagitis | 15-25% | 35-50% | Significantly higher |
| Pathological acid exposure | 20-30% | 40-60% | Higher with POEM |
| Barrett's esophagus | <5% | 5-10% | May be higher |
| PPI requirement | 20-30% | 50-70% | Higher with POEM |

### Managing Post-POEM GERD

```
POST-POEM REFLUX MANAGEMENT
═══════════════════════════

All POEM Patients:
┌─────────────────────────────────────────────┐
│ 1. PPI prophylaxis (6-12 months minimum)   │
│ 2. Lifestyle modifications                  │
│ 3. Annual or biennial EGD surveillance     │
└─────────────────────────────────────────────┘
            │
            ▼
┌─────────────────────────────────────────────┐
│ Symptomatic GERD develops                   │
│                                             │
│ • Continue/optimize PPI                    │
│ • Confirm on pH monitoring if needed       │
│ • Consider laparoscopic fundoplication     │
│   if PPI-refractory                        │
└─────────────────────────────────────────────┘

Future Option:
POEM + Fundoplication (POEM-F) being studied
- Full-thickness myotomy + partial wrap
- May reduce GERD rates
- Currently investigational
```

                   │                       │
                   ▼                       ▼
        ┌──────────────────┐      ┌──────────────────┐

                 │                └──────────────────┘

    ┌─────────────────────────────────────────────┐
    │                                              │
    │  Favor LHM if:          Favor POEM if:      │
    │  ─────────────          ──────────────      │
    │  • GERD concern         • No external cuts  │
    │  • Young patient        • Prior failed tx   │
    │  • Barrett's risk       • Redo surgery risk │
    │  • Hiatal hernia        • Shorter stay      │
    │  • Surgeon expertise    • Type III          │
    │                         • Endoscopist expert│
    │                                              │
    └─────────────────────────────────────────────┘
```

### Specific Scenarios

| Clinical Scenario | Preferred Approach | Rationale |
|-------------------|-------------------|-----------|
| Type I/II, young female | LHM + Dor | GERD/Barrett's concern |
| Type III achalasia | POEM | Extended myotomy needed |
| Failed prior PD | POEM | Effective salvage |
| Failed LHM | POEM | Easier redo than re-LHM |
| Failed POEM | LHM | Can still perform |
| Sigmoid esophagus | Either + consider Rx | May need esophagectomy |
| Large hiatal hernia | LHM | Can repair hernia |
| High surgical risk | POEM or Botox | Less invasive |
| No GERD tolerance | LHM + Dor | Anti-reflux protection |

## Complications Comparison

### Intraoperative

| Complication | LHM | POEM | Management |
|--------------|-----|------|------------|
| Mucosal perforation | 1-3% | 3-8% | Clip/suture, usually heal |
| Bleeding | <1% | 1-2% | Coagulation |
| Pneumothorax | Rare | 1-2% | Chest tube if needed |
| Pneumoperitoneum | N/A | Common | Usually resolves, needle decompression |
| Incomplete myotomy | 2-5% | 2-5% | Re-intervention |

### Postoperative

```
COMPLICATION TIMELINE
═════════════════════

EARLY (Days 1-7):
────────────────────────────────────────────────────────
LHM:                          POEM:
• Wound infection (1-2%)      • Delayed leak (1-2%)
• Leak from myotomy (rare)    • Subcutaneous emphysema
• Bleeding                    • Fever (tunnel infection)
• Gastroparesis               • Chest pain

INTERMEDIATE (Weeks 1-6):
────────────────────────────────────────────────────────
Both:
• Dysphagia (transient in 5-10%)
• GERD symptoms emergence

LATE (Months-Years):
────────────────────────────────────────────────────────
LHM:                          POEM:
• GERD (10-25%)               • GERD (35-50%)
• Recurrent dysphagia (5-10%) • Recurrent dysphagia (5-10%)
• Wrap migration/disruption   • Barrett's esophagus
• Incisional hernia (rare)    • No incision issues
```

## Long-term Outcomes

### Durability Data

```
LONG-TERM SUCCESS RATES
═══════════════════════

           1 yr    2 yr    5 yr    10 yr
           ────    ────    ────    ─────
LHM+Dor    94%     90%     85%     75-80%
                    │
POEM       95%     90%     80-85%  Limited data
                    │
                    └──► Similar at 2 years
                         LHM may have edge at 5+ years
                         (pending more POEM long-term data)
```

### Factors Affecting Long-term Success

| Factor | Impact | Notes |
|--------|--------|-------|
| Achalasia subtype | Type II best, Type III worst | Consistent across treatments |
| Pre-op esophageal diameter | >6 cm = worse | Consider timing |
| Myotomy completeness | Critical | Verify intraoperatively |
| Fundoplication (LHM) | Prevents GERD | Dor most common |
| Operator experience | High volume better | Learning curve matters |
| Post-op compliance | PPI, surveillance | Affects outcomes |

## Special Considerations

### Sigmoid Esophagus

```
SIGMOID ESOPHAGUS CLASSIFICATION
════════════════════════════════

Grade I (Mild):              Grade II (Severe):

    │                            │
    │ <6 cm                  ╭───╯
    │                       ╭╯     >6 cm
    └───                   ╭╯
       │                  ╭╯
       │                 ╰╮
       ▼                  ╰╮
                           ╰───▶

• LHM/POEM effective        • Consider esophagectomy
• 80-90% success            • High failure rate
• Watch for progression     • If myotomy fails → Rx

Treatment approach:
1. Try LHM or POEM first
2. If fails or progresses → Esophagectomy
3. End-stage → Esophagectomy primary
```

### Prior Treatment History

| Scenario | Recommendation |
|----------|----------------|
| Treatment-naïve | LHM, PD, or POEM |
| Failed single PD | POEM or LHM |
| Failed multiple PD | POEM (preferred) |
| Failed LHM | POEM |
| Failed POEM | LHM or re-POEM |
| Failed both | Esophagectomy if end-stage |

## Key Evidence Summary

### What the Evidence Shows

```
╔════════════════════════════════════════════════════════════╗
║              CURRENT EVIDENCE CONCLUSIONS                   ║
╠════════════════════════════════════════════════════════════╣
║                                                             ║
║  1. EFFICACY: POEM ≈ LHM at 2 years (Level 1)             ║
║     - Both achieve 85-95% success                          ║
║     - Non-inferior by RCT evidence                         ║
║                                                             ║
║  2. GERD: LHM + Fundoplication < POEM (Level 1)           ║
║     - 15-25% vs 35-50%                                     ║
║     - Fundoplication provides protection                   ║
║                                                             ║
║  3. TYPE III: POEM > LHM (Level 2-3)                       ║
║     - Extended myotomy capability                          ║
║     - 85-95% vs 60-70% success                            ║
║                                                             ║
║  4. REDO/SALVAGE: POEM effective (Level 2-3)              ║
║     - Good option after failed prior treatment             ║
║                                                             ║
║  5. LONG-TERM: More data needed for POEM                   ║
║     - LHM has 20+ years of data                           ║
║     - POEM only since 2010                                ║
║                                                             ║
╚════════════════════════════════════════════════════════════╝
```

### Guideline Recommendations

| Guideline | Recommendation |
|-----------|----------------|
| ACG 2020 | PD, LHM, POEM all first-line for Type I/II |
| SAGES 2022 | Both LHM and POEM acceptable |
| ESGE 2020 | POEM or LHM based on expertise |
| ISDE 2018 | POEM preferred for Type III |
