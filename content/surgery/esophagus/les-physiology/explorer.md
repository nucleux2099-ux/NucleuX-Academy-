# Lower Esophageal Sphincter (LES) Physiology

> **Source**: Shackelford Ch 1 - *Anatomy of the Esophagus*, Ch 7 - *Pathophysiology of Gastroesophageal Reflux*
> **Topic System**: [[Esophagus Index]]
> **Phase**: Shoot
> **Tier**: 1 (Foundation)

---

## Core Concepts

> [!summary] One-liner
> The LES is a 3-4 cm high-pressure zone of tonically contracted smooth muscle that relaxes with swallowing via NO/VIP release, augmented by the crural diaphragm—failure causes GERD (hypotensive) or achalasia (fails to relax).

## Anatomical Components

### The Anti-Reflux Barrier

```
    COMPONENTS OF GE JUNCTION BARRIER
    ══════════════════════════════════

    ┌─────────────────────────────────────────────────┐
    │                                                 │
    │   1. INTRINSIC LES                              │
    │      • Thickened circular smooth muscle         │
    │      • 3-4 cm length                            │
    │      • Tonic contraction (10-30 mmHg)           │
    │                                                 │
    │   2. EXTRINSIC (CRURAL DIAPHRAGM)               │
    │      • Right crus forms sling                   │
    │      • Pinchcock effect on inspiration          │
    │      • Contributes ~50% of barrier              │
    │                                                 │
    │   3. PHRENOESOPHAGEAL LIGAMENT                  │
    │      • Anchors esophagus to diaphragm           │
    │      • Prevents excessive movement              │
    │                                                 │
    │   4. INTRA-ABDOMINAL ESOPHAGUS                  │
    │      • 1-2 cm below diaphragm                   │
    │      • Positive pressure environment            │
    │      • Compressed by increased IAP              │
    │                                                 │
    │   5. ANGLE OF HIS                               │
    │      • Acute angle at cardia                    │
    │      • Creates mucosal flap valve               │
    │                                                 │
    └─────────────────────────────────────────────────┘
```

### LES Parameters

| Parameter | Normal Value | Clinical Significance |
|-----------|--------------|----------------------|
| **Resting pressure** | 10-30 mmHg | <10 = hypotensive (GERD risk) |
| **Length** | 3-4 cm | <2 cm = mechanically defective |
| **Intra-abdominal length** | 1-2 cm | <1 cm = herniated |
| **Relaxation** | To gastric baseline | Incomplete = achalasia |
| **IRP (on HRM)** | <15 mmHg | >15 = impaired relaxation |

## Neural Control

### Resting Tone (Tonic Contraction)

```
    LES RESTING TONE MAINTENANCE
    ════════════════════════════

    MYOGENIC (Intrinsic)
    ├── Spontaneous smooth muscle tone
    └── Ca2+ influx → contraction

    NEUROGENIC
    ├── Excitatory input (maintains tone)
    │   └── Cholinergic (ACh) from vagus
    │
    └── Inhibitory input (modulates)
        └── NANC neurons (NO, VIP)

    HORMONAL
    ├── Increase tone: Gastrin, motilin
    └── Decrease tone: CCK, secretin, progesterone
```

### LES Relaxation (Swallow-Initiated)

```
    SWALLOW-INDUCED LES RELAXATION
    ═══════════════════════════════

    Swallow initiated
          │
          ▼
    Swallowing center (medulla)
          │
          ▼
    Vagal efferent (dorsal motor nucleus)
          │
          ▼
    Myenteric plexus at LES
          │
          ├────────────────────────────┐
          ▼                            ▼
    INHIBITORY neurons           EXCITATORY neurons
    (predominate during          (suppressed)
     relaxation)
          │
          ▼
    Release NO + VIP
          │
          ▼
    Smooth muscle relaxation
    (↓ Ca2+, ↑ K+ channels)
          │
          ▼
    LES OPENS → Food passes
          │
          ▼
    After bolus: Tone restored
```

> [!tip] Clinical Pearl - Achalasia Connection
> In **achalasia**, the inhibitory neurons (NO, VIP) are **lost**. Without them:
> - Swallow signal arrives but relaxation doesn't occur
> - Excitatory input (ACh) is unopposed
> - LES remains contracted → dysphagia

## Transient LES Relaxations (TLESRs)

### Definition and Mechanism

TLESRs are spontaneous LES relaxations **not triggered by swallowing**, lasting 10-60 seconds.

```
    TRANSIENT LES RELAXATION (TLESR)
    ═════════════════════════════════

    TRIGGER: Gastric distension
             │
             ▼
    Vagal afferents (from stomach)
             │
             ▼
    Brainstem reflex center
             │
             ▼
    Vagal efferent to LES
             │
             ▼
    TLESR (without swallow)
             │
             ▼
    Purpose: VENTING (belching)

    ⚠️ PROBLEM: Main mechanism of reflux in GERD
       Acid can reflux during TLESR
```

### TLESR vs Swallow-Induced Relaxation

| Feature | Swallow-Induced | TLESR |
|---------|-----------------|-------|
| **Trigger** | Swallow | Gastric distension |
| **Duration** | 5-10 seconds | 10-60 seconds |
| **Crural inhibition** | No | Yes (diaphragm relaxes too) |
| **Purpose** | Allow bolus passage | Venting (belch) |
| **Role in GERD** | Minor | **Major** (50-70% of reflux) |

## Factors Affecting LES Pressure

### Factors That INCREASE LES Pressure

| Factor | Mechanism |
|--------|-----------|
| **Protein meal** | Gastrin release |
| **Gastrin** | Direct smooth muscle effect |
| **Motilin** | Increases tone |
| **Metoclopramide** | Dopamine antagonist, ↑ ACh |
| **α-adrenergic agonists** | Smooth muscle contraction |
| **Increased IAP** | Compresses intra-abdominal segment |

### Factors That DECREASE LES Pressure

| Factor | Mechanism |
|--------|-----------|
| **Fat meal** | CCK release |
| **Chocolate** | Methylxanthines → ↓ tone |
| **Caffeine** | Same as chocolate |
| **Alcohol** | Direct smooth muscle relaxation |
| **Smoking** | ↓ LES pressure |
| **Peppermint** | Smooth muscle relaxation |
| **CCK, Secretin** | Hormonal |
| **Progesterone** | Smooth muscle relaxation (pregnancy) |
| **Nitrates** | NO donors |
| **Calcium channel blockers** | ↓ Ca2+ influx |
| **Anticholinergics** | ↓ excitatory input |
| **Morphine** | Variable, generally ↓ |

## LES Dysfunction States

### Comparison of LES Disorders

```
    LES DYSFUNCTION SPECTRUM
    ════════════════════════

    NORMAL                    GERD                      ACHALASIA
    ──────                    ────                      ─────────

    Pressure: 10-30 mmHg      Pressure: LOW (<10)       Pressure: HIGH (>30)
                              or normal with ↑TLESRs    or normal

    Relaxation: Complete      Relaxation: Complete      Relaxation: INCOMPLETE
    (to gastric baseline)     (to gastric baseline)     (IRP >15 mmHg)

    Result:                   Result:                   Result:
    Normal function           Acid reflux               Dysphagia
                              Heartburn                 Regurgitation
```

### GERD: LES Incompetence

| Mechanism | Prevalence in GERD |
|-----------|-------------------|
| **Transient LES relaxations** | 50-70% (most common) |
| **Hypotensive LES** | 20-30% |
| **Anatomical disruption** (hiatal hernia) | 40-60% |

### Achalasia: Failed LES Relaxation

| Finding | Normal | Achalasia |
|---------|--------|-----------|
| **IRP** | <15 mmHg | **>15 mmHg** |
| **Basal LES pressure** | 10-30 mmHg | Often elevated |
| **Relaxation** | Complete | **Incomplete** |
| **Cause** | - | Loss of NO/VIP neurons |

## Crural Diaphragm Contribution

### Dual Sphincter Concept

```
    THE DUAL SPHINCTER
    ══════════════════

              ┌──────────────────────────────────┐
              │                                  │
    At REST   │   LES ════════════════ 50%      │
              │   Crural diaphragm ═══ 50%      │
              │                                  │
              └──────────────────────────────────┘

    During INSPIRATION:
    • LES: Unchanged
    • Crural: INCREASES (contracts with diaphragm)
    • Protects against reflux when IAP rises

    During SWALLOW:
    • LES: Relaxes (NO/VIP)
    • Crural: ALSO inhibited
    • Both must relax for bolus to pass

    In HIATAL HERNIA:
    • LES: Displaced above diaphragm
    • Crural: No longer augments LES
    • Loss of dual sphincter protection
```

## Clinical Applications

### Pharmacological Targets

| Drug | Mechanism | Use |
|------|-----------|-----|
| **Baclofen** | GABA-B agonist, ↓ TLESRs | Refractory GERD |
| **Metoclopramide** | ↑ LES pressure, prokinetic | GERD, gastroparesis |
| **Nitrates/CCBs** | ↓ LES pressure | Achalasia (temporizing) |
| **Botox** | Blocks ACh at LES | Achalasia (high-risk patients) |

### Surgical Relevance

| Procedure | Effect on LES |
|-----------|---------------|
| **Fundoplication** | Augments LES, creates valve |
| **Heller myotomy** | Disrupts LES (for achalasia) |
| **POEM** | Divides LES circular muscle |
| **Hiatal hernia repair** | Restores crural contribution |

## Quick Recall Questions

1. Q: What are the two main neurotransmitters responsible for LES relaxation?
   A: Nitric oxide (NO) and vasoactive intestinal peptide (VIP)

2. Q: What is the most common mechanism of reflux in GERD?
   A: Transient LES relaxations (TLESRs) - account for 50-70% of reflux episodes

3. Q: Why does hiatal hernia promote GERD?
   A: Loss of crural diaphragm contribution (no longer augments LES) + loss of positive pressure on intra-abdominal esophagus

4. Q: What is the IRP cutoff for impaired LES relaxation?
   A: IRP >15 mmHg indicates impaired relaxation (suggestive of achalasia)
