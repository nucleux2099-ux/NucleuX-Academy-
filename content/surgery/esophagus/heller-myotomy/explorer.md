# Laparoscopic Heller Myotomy + Fundoplication

> **Source**: Shackelford Ch 19 - *Achalasia Diagnosis and Treatment*
> **Topic System**: [[Esophagus Index]]
> **Phase**: Shoot
> **Tier**: 4 (Surgical Technique)

---

## Core Concepts

> [!summary] One-liner
> Laparoscopic Heller myotomy is the surgical division of LES circular muscle (6-8cm esophageal + 2-3cm gastric) combined with partial fundoplication (Dor/Toupet) for achalasia, achieving 90% long-term success.

---

## Indications & Contraindications

### Indications

| Indication | Notes |
|------------|-------|
| **Achalasia Type I, II** | First-line option |
| **Achalasia Type III** | Extended myotomy; POEM may be preferred |
| **Failed pneumatic dilation** | Rescue therapy |
| **Patient preference** | Over PD or POEM |
| **Young patients** | Better durability than PD |

### Contraindications

| Absolute | Relative |
|----------|----------|
| Unfit for general anesthesia | Prior POEM (can still do) |
| Uncorrectable coagulopathy | Prior esophageal surgery |
| End-stage sigmoid esophagus | Multiple prior Botox injections |

### Step-by-Step Procedure

#### Step 1-3: Exposure and Dissection

1. Insert liver retractor via subxiphoid port
2. Open gastrohepatic ligament (pars flaccida)
3. Identify and preserve vagus nerves
4. Mobilize distal esophagus circumferentially (5-6 cm)

#### Step 4: Myotomy (CRITICAL)

```
    MYOTOMY TECHNIQUE
    ═════════════════

              6-8 cm
          ◄────────────►
    ┌─────────────────────────────┐
    │                             │
    │      ESOPHAGEAL             │
    │      MYOTOMY                │
    │                             │
    │      Start here             │
    │      (proximal esophagus)   │
    │             │               │
    │             ▼               │
    │    Divide circular muscle   │
    │    Preserve longitudinal    │
    │    Expose mucosa (bulges)   │
    │             │               │
    ├─────────────┼───────────────┤ ← GEJ/Fat pad
    │             │               │
    │      GASTRIC                │  2-3 cm
    │      EXTENSION              │  ◄──────►
    │             │               │
    │      MUST divide sling      │
    │      fibers completely      │
    │                             │
    └─────────────────────────────┘

    ⚠️ CRITICAL: Must extend onto gastric cardia
       Incomplete distal myotomy = #1 cause of failure
```

**Myotomy Details**:

| Parameter | Specification |
|-----------|---------------|
| **Location** | Anterior (11 o'clock) |
| **Esophageal length** | 6-8 cm above GEJ |
| **Gastric length** | **2-3 cm onto cardia** (CRITICAL) |
| **Muscle divided** | Circular (inner) completely |
| **Muscle preserved** | Longitudinal (outer) |
| **Endpoint** | Mucosa bulges through |

#### Step 5: Test for Mucosal Perforation

```
1. Fill abdomen with saline (submerge GEJ)
2. Instill air via NG tube (or endoscope)
3. Watch for bubbles at myotomy site
4. If bubbles → identify and repair perforation
5. Alternatively: Inject methylene blue
```

#### Step 6: Fundoplication

**Choice of Fundoplication**:

| Type | Configuration | Use When |
|------|---------------|----------|
| **Dor (Anterior 180°)** | Fundus folded over anterior myotomy | **Preferred** - covers myotomy |
| **Toupet (Posterior 270°)** | Fundus wrapped posteriorly | Alternative - keeps edges apart |

> [!warning] Avoid Nissen (360°)
> Full wrap is too tight for aperistaltic esophagus → dysphagia

**Dor Fundoplication Technique**:

```
    DOR FUNDOPLICATION
    ═══════════════════

           ┌─────────────┐
           │   Myotomy   │
           │   site      │
           └──────┬──────┘
                  │
         ┌────────┴────────┐
         │                 │
    ┌────┴────┐       ┌────┴────┐
    │ Left    │       │ Right   │
    │ edge of │       │ edge of │
    │ fundus  │       │ fundus  │
    └────┬────┘       └────┬────┘
         │                 │
         │    FOLD OVER    │
         │    MYOTOMY      │
         │                 │
         └────────┬────────┘
                  │
                  ▼
         ┌─────────────────┐
         │ Suture to       │
         │ - Left & right  │
         │   myotomy edges │
         │ - Diaphragm     │
         └─────────────────┘
```

## Intraoperative Complications

| Complication | Incidence | Management |
|--------------|-----------|------------|
| **Mucosal perforation** | 5-10% | Primary repair + Dor coverage |
| **Bleeding** | <5% | Cautery, clips |
| **Vagus injury** | <5% | May need pyloroplasty |
| **Pleural entry** | Rare | Close defect, evacuate pneumothorax |

### Managing Mucosal Perforation

```
1. Identify extent of injury
2. Primary repair with absorbable suture (4-0 Vicryl)
3. Interrupted sutures, mucosa-to-mucosa
4. Test repair (air/saline)
5. Complete Dor fundoplication (covers repair)
6. Post-op: Gastrografin swallow POD1
```

## Outcomes

### Success Rates

| Outcome | Rate |
|---------|------|
| **Initial symptom relief** | 90-95% |
| **5-year success** | 85-90% |
| **10-year success** | 80-85% |
| **Need for reoperation** | <10% |

### By Achalasia Type

| Type | Success Rate | Notes |
|------|--------------|-------|
| Type I | 85-90% | Good response |
| Type II | >90% | **Best outcomes** |
| Type III | 70-80% | May need longer myotomy (consider POEM) |

### GERD After Heller

| With Fundoplication | GERD Rate |
|---------------------|-----------|
| Dor (anterior 180°) | 10-15% |
| Toupet (posterior 270°) | 10-15% |
| No fundoplication | 30-50% |

## Quick Recall Questions

1. Q: What is the critical myotomy length onto the gastric cardia?
   A: 2-3 cm onto the stomach; incomplete distal extension is the #1 cause of failure

2. Q: Why is Dor fundoplication preferred over Nissen?
   A: Dor (anterior 180°) covers the myotomy site (protects against leak) and is a partial wrap suitable for aperistaltic esophagus. Nissen (360°) is too tight → dysphagia

3. Q: What is the management of intraoperative mucosal perforation?
   A: Primary repair with absorbable suture + complete Dor fundoplication to cover the repair + post-op contrast study
