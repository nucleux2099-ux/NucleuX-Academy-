# Esophageal Reconstruction: Methods & Outcomes

## Overview

Esophageal reconstruction is required following esophagectomy for malignancy, caustic injury, or end-stage benign disease. The choice of conduit and reconstruction technique significantly impacts outcomes, quality of life, and long-term function. This topic reviews the available options, surgical techniques, and comparative outcomes.

```
INDICATIONS FOR ESOPHAGEAL RECONSTRUCTION
═════════════════════════════════════════

┌────────────────────────────────────────────────────────────┐
│                                                             │
│  MALIGNANCY                                                 │
│  • Esophageal carcinoma (squamous, adenocarcinoma)         │
│  • GEJ tumors (Siewert I, II)                              │
│                                                             │
│  BENIGN CONDITIONS                                          │
│  • Caustic stricture (failed dilation)                     │
│  • End-stage achalasia (sigmoid esophagus)                 │
│  • Refractory peptic stricture                             │
│  • Esophageal perforation (extensive)                      │
│                                                             │
│  CONGENITAL                                                 │
│  • Long-gap esophageal atresia                             │
│                                                             │
└────────────────────────────────────────────────────────────┘
```

## Conduit Options

### Comparison Overview

```
CONDUIT OPTIONS COMPARISON
══════════════════════════

                    GASTRIC        COLON          JEJUNUM
                    ───────        ─────          ───────
Preference          1st choice     2nd choice     3rd choice

Blood Supply        Excellent      Variable       Good
                    (single axis)  (complex)      (segmental)

Reach               Pharynx        Pharynx        Limited
                    (neck)         (neck)         (mid-chest)

Technical           Easier         Complex        Complex
Difficulty

Acid Exposure       Yes            Minimal        None

Redundancy          Rare           Common         Rare

Leak Rate           5-10%          10-15%         5-10%

Mortality           3-5%           5-10%          3-5%
```

### Gastric Conduit (First Choice)

```
GASTRIC PULL-UP - ANATOMY
═════════════════════════

Blood Supply Preserved:                 Conduit Creation:

    ┌─────────────────┐                    ┌────────────┐
    │                 │                    │            │
    │  Left gastric   │ ← DIVIDE          │   Fundus   │
    │  (sacrificed)   │                    │  removed   │
    │                 │                    │            │
    └────────┬────────┘                    │  ┌────────┤
             │                             │  │ Staple │
    ┌────────┴────────┐                    │  │  line  │
    │   STOMACH       │                    │  │        │
    │                 │                    │  │  Tube  │
    │  Right gastric ◄│── PRESERVE        │  │ conduit│
    │                 │                    │  │        │
    │  Right gastro-  │                    │  │        │
    │  epiploic      ◄│── PRESERVE        │  └────────┤
    │                 │                    │            │
    └─────────────────┘                    └────────────┘

Key Points:
• Right gastric and right gastroepiploic arteries preserved
• Forms vascular arcade along greater curve
• Left gastric divided (allows mobilization)
• Short gastrics divided
• Fundus removed (creates narrow tube)
• Pyloroplasty/pyloromyotomy for drainage
```

**Advantages:**
- Single anastomosis
- Reliable blood supply
- Reaches neck easily
- Technical simplicity

**Disadvantages:**
- Acid reflux (no sphincter)
- Loss of reservoir function
- Dumping syndrome risk
- Aspiration risk

### Colon Interposition (Second Choice)

```
COLON INTERPOSITION - OPTIONS
═════════════════════════════

LEFT COLON (Most Common):              RIGHT COLON:

    ┌──────────────────┐               ┌──────────────────┐
    │                  │               │                  │
    │                  │               │    ┌────────┐   │
    │    ┌──────┐      │               │    │        │   │
    │    │      │      │               │    │  Right │   │
    │    │ Left │      │               │    │ Colon  │   │
    │    │Colon │      │               │    │        │   │
    │    │      │      │               │    └────┬───┘   │
    │    └──┬───┘      │               │         │       │
    │       │          │               │     Terminal    │
    │  Left colic      │               │     ileum       │
    │  vessels         │               │     included    │
    │  (marginal)      │               │                 │
    └──────────────────┘               └──────────────────┘

Blood Supply:
• Left colon: Left colic, marginal artery of Drummond
• Right colon: Middle colic, ileocolic

Routes:
• Retrosternal (most common for benign)
• Posterior mediastinal (after esophagectomy)
• Subcutaneous (last resort)
```

**Indications for Colon:**
- Previous gastrectomy (no stomach available)
- Caustic injury involving stomach
- Failed gastric conduit
- Long-gap esophageal atresia (pediatric)

**Advantages:**
- Maintains peristalsis
- Adequate length
- No acid production
- Good for benign disease

**Disadvantages:**
- Three anastomoses
- Complex blood supply
- Redundancy over time
- Colonic complications (odor, redundancy)

### Jejunal Interposition (Third Choice)

```
JEJUNAL INTERPOSITION
═════════════════════

Standard Interposition:               Free Jejunal Transfer:

    ┌────────────────┐                ┌────────────────┐
    │   Esophagus    │                │   Pharynx      │
    │       │        │                │       │        │
    │   ┌───┴───┐    │                │   ┌───┴───┐    │
    │   │Jejunum│    │                │   │Free   │◄── Microvascular
    │   │segment│    │                │   │Jejunum│    anastomosis
    │   └───┬───┘    │                │   └───┬───┘
    │       │        │                │       │        │
    │   Stomach/     │                │   Trachea      │
    │   Jejunum      │                │   stoma        │
    └────────────────┘                └────────────────┘

Blood Supply:
• Segmental (jejunal branches of SMA)
• Mesentery must reach without tension
• Free flap requires microvascular expertise
```

**Indications:**
- Short segment replacement
- Pharyngeal reconstruction (free flap)
- After total gastrectomy
- When colon unavailable

**Advantages:**
- No acid exposure
- Active peristalsis
- No reflux

**Disadvantages:**
- Limited length (mesenteric constraint)
- Multiple anastomoses
- Requires Roux limb
- Free flap complexity

## Surgical Approaches for Esophagectomy

### Route Comparison

```
ESOPHAGECTOMY APPROACHES
════════════════════════

TRANSTHORACIC (Ivor Lewis/McKeown):

    ┌─────────────────────────────────┐
    │          Neck incision          │ ← McKeown only
    │              │                  │   (3-field)
    │     ┌────────┴────────┐         │
    │     │   Esophagus     │         │
    │     │                 │         │
    │ ════╪════ Diaphragm   │         │
    │     │                 │         │
    │     │   Right chest   │ ← Thoracotomy
    │     │   access        │   (Ivor Lewis)
    │     │                 │         │
    │     └────────┬────────┘         │
    │              │                  │
    │         Abdomen                 │ ← Laparotomy
    └─────────────────────────────────┘


TRANSHIATAL (Orringer):

    ┌─────────────────────────────────┐
    │          Neck incision          │
    │              │                  │
    │     ┌────────┴────────┐         │
    │     │   Esophagus     │         │
    │     │   bluntly       │         │ No thoracotomy!
    │     │   dissected     │         │
    │ ════╪════ Diaphragm   │         │
    │     │                 │         │
    │     │   Hiatal        │         │
    │     │   approach      │         │
    │     │                 │         │
    │     └────────┬────────┘         │
    │              │                  │
    │         Abdomen                 │ ← Laparotomy
    └─────────────────────────────────┘
```

### Approach Selection

| Approach | Indications | Anastomosis Level |
|----------|-------------|-------------------|
| **Ivor Lewis** | Mid/distal esophageal Ca | Intrathoracic (chest) |
| **McKeown (3-field)** | Upper esophageal Ca | Cervical (neck) |
| **Transhiatal** | GEJ tumors, benign disease | Cervical (neck) |
| **Left thoracoabdominal** | GEJ/cardia tumors | Low thoracic |
| **Minimally invasive (MIE)** | Selected cases | Varies |

## Anastomotic Techniques

### Cervical vs Intrathoracic Anastomosis

```
ANASTOMOSIS LOCATION COMPARISON
═══════════════════════════════

CERVICAL:                          INTRATHORACIC:

     ┌───────┐                          ┌───────┐
     │ Neck  │ ← Anastomosis            │       │
     │       │                          │ Chest │ ← Anastomosis
     └───┬───┘                          │       │
         │                              └───┬───┘
     ┌───┴───┐                          ┌───┴───┐
     │ Chest │                          │       │
     │       │                          │       │
     └───┬───┘                          └───┬───┘
         │                                  │
     Abdomen                            Abdomen

Leak Rate:      10-15%                  3-5%
Leak Severity:  Less severe             Life-threatening
                (drains to neck)        (mediastinitis)
Stricture:      More common             Less common
Recurrent       Lower risk              Higher risk
Laryngeal Nerve:
```

### Anastomotic Techniques

| Technique | Description | Advantages |
|-----------|-------------|------------|
| **Hand-sewn** | Single or double layer | Versatile, any location |
| **Stapled (EEA)** | Circular stapler | Faster, uniform |
| **Linear stapled** | Side-to-side functional end-to-end | Lower stricture rate |
| **Semi-mechanical** | Stapled back, hand-sewn front | Combines benefits |

```

═════════════════════════════

    ┌───┴───┐                          ┌───┴───┐
    │  ○○○  │ ← Staple line            │       │
    │  ○○○  │                          │ ══════│ ← Staple

Advantages:                        Advantages:
```

## Postoperative Management

### Early Management

```
POST-ESOPHAGECTOMY CARE
═══════════════════════

DAY 0-3:
┌──────────────────────────────────────────────────────────┐
│ • ICU monitoring (1-2 days)                              │
│ • NPO with NG/NJ tube                                    │
│ • IV fluids, electrolyte monitoring                     │
│ • Pain management (epidural or PCA)                     │
│ • Chest tube management (thoracic approach)             │
│ • DVT prophylaxis                                       │
│ • Respiratory therapy (incentive spirometry)            │
└──────────────────────────────────────────────────────────┘

DAY 5-7:
┌──────────────────────────────────────────────────────────┐
│ • Contrast swallow study                                 │
│ • If no leak → start clear liquids                      │
│ • Advance diet slowly                                    │
│ • Chest tubes removed when output low                   │
│ • Early mobilization                                     │
└──────────────────────────────────────────────────────────┘

DISCHARGE (Day 7-14):
┌──────────────────────────────────────────────────────────┐
│ • Tolerating soft/pureed diet                           │
│ • Adequate pain control (oral)                          │
│ • Feeding tube if needed (J-tube)                       │
│ • Patient education on eating                           │
│ • Signs of leak/stricture                               │
└──────────────────────────────────────────────────────────┘
```

### Dietary Progression

| Phase | Timing | Foods | Notes |
|-------|--------|-------|-------|
| NPO | Days 0-5 | None | IV/enteral via J-tube |
| Clear liquids | Day 5-7 | Water, broth, juice | After leak test |
| Full liquids | Week 2 | Protein shakes, soup | Small volumes |
| Pureed | Week 3-4 | Blended foods | 6-8 small meals |
| Soft | Week 5-6 | Soft cooked foods | Chew thoroughly |
| Regular | Week 8+ | Modified regular | Lifelong small meals |

## Complications

### Early Complications

```
EARLY COMPLICATIONS (<30 DAYS)
══════════════════════════════

ANASTOMOTIC LEAK (5-15%)
────────────────────────────────────────────────────────────
Timing: Days 5-10

Risk Factors:              Signs:
• Poor blood supply        • Fever
• Tension                  • Tachycardia
• Technical error         • Leukocytosis
• Malnutrition            • Neck swelling (cervical)
• Neoadjuvant therapy     • Pleural effusion (thoracic)

Management:
• Cervical: Drain, NPO, antibiotics (usually heals)
• Thoracic: Emergency! → Re-operation or covered stent


CONDUIT NECROSIS (1-3%)
────────────────────────────────────────────────────────────
Cause: Vascular compromise

Signs:                     Management:
• Sepsis                   • Emergency surgery
• Rapid deterioration     • Takedown anastomosis
• Dark drainage           • Cervical esophagostomy
• EGD shows necrosis      • Jejunostomy feeding
                          • Delayed reconstruction


PULMONARY COMPLICATIONS (15-30%)
────────────────────────────────────────────────────────────
Types:                     Prevention/Treatment:
• Pneumonia               • Aggressive pulmonary toilet
• Aspiration              • Early mobilization
• ARDS                    • Aspiration precautions
• Respiratory failure     • Antibiotics if infection
```

### Late Complications

```
LATE COMPLICATIONS
══════════════════

ANASTOMOTIC STRICTURE (10-30%)
────────────────────────────────────────────────────────────
Timing: 2-6 months post-op

Causes:                    Symptoms:
• Ischemia                • Progressive dysphagia
• Leak with healing       • Food impaction
• Reflux                  • Weight loss
• Radiation

Management:
1. Endoscopic dilation (first-line)
2. Multiple sessions usually needed
3. Refractory → stent or revision


REFLUX/ASPIRATION
────────────────────────────────────────────────────────────
Cause: No LES after reconstruction

Prevention/Management:
• PPI therapy
• Elevate head of bed
• Small frequent meals
• No eating before bed
• Consider pyloroplasty


DUMPING SYNDROME
────────────────────────────────────────────────────────────
Cause: Rapid gastric emptying, vagotomy effect

Types:
Early (15-30 min): Vasomotor symptoms
Late (1-3 hours): Hypoglycemia

Management:
• Small frequent meals
• Avoid simple sugars
• Acarbose for late dumping
```

## Outcomes by Conduit

### Comparative Outcomes

```
OUTCOMES COMPARISON
═══════════════════

                        GASTRIC     COLON      JEJUNUM
                        ─────────   ────────   ───────
Operative mortality     3-5%        5-10%      3-5%

Anastomotic leak        5-10%       10-15%     5-10%

Conduit necrosis        1-3%        3-5%       2-4%

Stricture rate          10-20%      15-25%     10-15%

Hospital stay           10-14 days  14-21 days 12-16 days

Functional outcome      Good        Good       Good

Long-term survival*     Similar across conduits
                        (depends on disease stage)

*For malignancy - conduit choice doesn't affect cancer survival
```

### Quality of Life

| Aspect | Gastric | Colon | Jejunum |
|--------|---------|-------|---------|
| Eating capacity | Fair | Good | Good |
| Reflux | Common | Rare | Rare |
| Dumping | Common | Rare | Rare |
| Redundancy | Rare | Common | Rare |
| Long-term function | Good | Good | Good |

## Special Situations

### Caustic Injury Reconstruction

```
CAUSTIC INJURY - RECONSTRUCTION PRINCIPLES
══════════════════════════════════════════

Timing:
• Acute: Supportive care, assess extent
• Subacute (2-6 weeks): Avoid surgery (friable tissue)
• Chronic (>6 months): Reconstruction if indicated

Assessment:
• EGD (initial and follow-up)
• Contrast study
• CT for extent
• Gastric involvement?

Conduit Selection:
┌────────────────────────────────────────────────────────┐
│                                                         │
│  Stomach spared → GASTRIC PULL-UP (preferred)          │
│                                                         │
│  Stomach involved → COLON INTERPOSITION                │
│                    (most common for caustic)           │
│                                                         │
│  Route: RETROSTERNAL (bypass damaged mediastinum)      │
│                                                         │
└────────────────────────────────────────────────────────┘
```

### Supercharged Gastric Conduit

```
SUPERCHARGED CONDUIT
════════════════════

Indication: Concerns about gastric tip perfusion

Technique:
    ┌─────────────────────────────────────────┐
    │                                          │
    │  Left internal mammary or              │
    │  internal jugular vein                  │
    │           │                              │
    │     ┌─────┴─────┐                       │
    │     │Anastomosis│ ← Microvascular      │
    │     └─────┬─────┘   anastomosis        │
    │           │                              │
    │  ┌────────┴────────┐                    │
    │  │   Gastric tip   │                    │
    │  │   (enhanced)    │                    │
    │  └─────────────────┘                    │
    │                                          │
    └─────────────────────────────────────────┘

Purpose:
• Augments blood supply to conduit tip
• Reduces anastomotic complications
• Consider for long conduit pulls
```
