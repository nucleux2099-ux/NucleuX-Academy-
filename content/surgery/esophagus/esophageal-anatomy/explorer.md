# Esophageal Anatomy

> **Source**: Shackelford Ch 1 - *Anatomy of the Esophagus and the Gastroesophageal Reflux Barrier*
> **Topic System**: [[Esophagus Index]]
> **Phase**: Shoot
> **Tier**: 1 (Foundation)

---

## Core Concepts

> [!summary] One-liner
> The esophagus is a 25cm muscular tube extending from C6 to T11, with three anatomical narrowings, transitioning from striated to smooth muscle, and lacking a serosa—making anastomotic leaks more common.

---

## Gross Anatomy

### Length and Position

| Parameter | Value |
|-----------|-------|
| **Total length** | 25 cm (range 18-26 cm) |
| **Cervical** | 5 cm (C6 to thoracic inlet) |
| **Thoracic** | 17 cm (thoracic inlet to diaphragm) |
| **Abdominal** | 2-3 cm (diaphragm to GEJ) |

### Distance from Incisors

| Landmark | Distance from Incisors |
|----------|----------------------|
| **Cricopharyngeus (UES)** | 15-18 cm |
| **Aortic arch** | 22-25 cm |
| **Left main bronchus** | 27-30 cm |
| **Diaphragmatic hiatus** | 38-40 cm |
| **GEJ** | 40-42 cm |

### Three Anatomical Narrowings

```
    ESOPHAGEAL NARROWINGS
    ═════════════════════

    1. CRICOPHARYNGEAL (UES) ─── 15 cm ─── C6 level
       Narrowest point (14mm)
       Common site for Zenker diverticulum
       Foreign body impaction
                │
                │
    2. AORTIC/BRONCHIAL ─────── 25 cm ─── T4-5 level
       Aortic arch + left main bronchus
       External compression
                │
                │
    3. DIAPHRAGMATIC ──────────  40 cm ─── T10-11 level
       Esophageal hiatus
       Site of hiatal hernia
       GEJ location
```

**Clinical Significance**:
- **Corrosive injury**: Most severe at narrowings (prolonged contact)
- **Foreign body impaction**: Most common at cricopharyngeus
- **Cancer**: 50% occur in middle third (bronchial narrowing)

## Wall Layers

### Histological Layers

```
    ESOPHAGEAL WALL (Inside → Out)
    ══════════════════════════════

    ┌─────────────────────────────────────┐
    │         MUCOSA                      │
    │  ┌───────────────────────────────┐  │
    │  │ Epithelium (stratified        │  │
    │  │ squamous, non-keratinized)    │  │
    │  ├───────────────────────────────┤  │
    │  │ Lamina propria                │  │
    │  ├───────────────────────────────┤  │
    │  │ Muscularis mucosae            │  │
    │  └───────────────────────────────┘  │
    ├─────────────────────────────────────┤
    │         SUBMUCOSA                   │
    │  • Blood vessels, lymphatics        │
    │  • Submucosal (Meissner's) plexus   │
    │  • Esophageal glands                │
    ├─────────────────────────────────────┤
    │         MUSCULARIS PROPRIA          │
    │  ┌───────────────────────────────┐  │
    │  │ Inner CIRCULAR muscle         │  │ ← LES fibers
    │  ├───────────────────────────────┤  │
    │  │ Myenteric (Auerbach's) plexus │  │ ← Lost in achalasia
    │  ├───────────────────────────────┤  │
    │  │ Outer LONGITUDINAL muscle     │  │
    │  └───────────────────────────────┘  │
    ├─────────────────────────────────────┤
    │         ADVENTITIA                  │
    │  • NO SEROSA (above diaphragm)      │ ← Leak risk
    │  • Loose connective tissue          │
    │  • Allows spread of infection       │
    └─────────────────────────────────────┘
```

> [!warning] No Serosa = Clinical Implications
> - Anastomotic leaks more common (no serosal seal)
> - Cancer spreads earlier to adjacent structures
> - Perforations spread easily to mediastinum
> - Sutures don't hold as well

### Muscle Type Transition

| Level | Muscle Type | Significance |
|-------|-------------|--------------|
| **Upper 1/3** | Striated (voluntary) | UES control |
| **Middle 1/3** | Mixed | Transition zone |
| **Lower 1/3** | Smooth (involuntary) | LES, affected in achalasia |

## Blood Supply

### Arterial Supply (Segmental)

```
    ESOPHAGEAL ARTERIAL SUPPLY
    ═══════════════════════════

    CERVICAL
    └── Inferior thyroid artery (thyrocervical trunk)

    THORACIC
    ├── Bronchial arteries
    ├── Aortic esophageal branches (4-6 direct)
    └── Right intercostal arteries

    ABDOMINAL
    ├── Left gastric artery (main supply)
    ├── Left inferior phrenic artery
    └── Short gastric arteries

    ⚠️ Watershed area: Lower thoracic esophagus
       (Between aortic and left gastric territories)
```

### Venous Drainage

| Level | Drainage | Systemic/Portal |
|-------|----------|-----------------|
| **Upper** | Inferior thyroid → Brachiocephalic | Systemic |
| **Middle** | Azygos/Hemiazygos | Systemic |
| **Lower** | Left gastric (coronary) vein | **Portal** |

> [!tip] Clinical Pearl
> The lower esophageal veins drain into the **portal system** via left gastric vein. In portal hypertension, these become **esophageal varices** (portosystemic collateral).

## Lymphatic Drainage

### Longitudinal Submucosal Plexus

```
    ESOPHAGEAL LYMPHATICS
    ═════════════════════

    Key feature: EXTENSIVE LONGITUDINAL SPREAD
    (Tumor can skip levels → aggressive nodes)

    CERVICAL ────► Deep cervical nodes
                   Paratracheal nodes

    THORACIC ────► Tracheobronchial nodes
                   Posterior mediastinal nodes
                   Subcarinal nodes

    ABDOMINAL ───► Left gastric nodes
                   Celiac nodes
                   Para-aortic nodes

    ⚠️ 6cm longitudinal spread in submucosa
       → Wide margins needed for cancer
```

**Implications for Cancer Surgery**:
- 8-10 cm proximal margin recommended for squamous cell carcinoma
- Skip metastases can occur
- Two-field or three-field lymphadenectomy may be needed

## Nerve Supply

### Extrinsic Innervation

| Nerve | Origin | Function |
|-------|--------|----------|
| **Vagus (CN X)** | Brainstem | Parasympathetic - peristalsis, LES relaxation |
| **Sympathetic** | T1-T10 | Inhibits peristalsis, contracts LES |
| **Recurrent laryngeal** | Vagus | Motor to upper esophagus (striated) |

### Intrinsic (Enteric) Plexuses

```
    MYENTERIC (AUERBACH'S) PLEXUS
    ═════════════════════════════
    Location: Between circular and longitudinal muscle

    ┌─────────────────────────────────────────────┐
    │                                             │
    │   EXCITATORY neurons ──► Acetylcholine      │
    │   (Cause contraction)    Substance P        │
    │                                             │
    │   INHIBITORY neurons ──► NO (nitric oxide)  │
    │   (Cause relaxation)     VIP                │
    │                                             │
    │   ⚠️ In ACHALASIA: Inhibitory neurons LOST  │
    │      → Unopposed excitatory → LES fails     │
    │        to relax                             │
    │                                             │
    └─────────────────────────────────────────────┘

    SUBMUCOSAL (MEISSNER'S) PLEXUS
    ═══════════════════════════════
    Location: Submucosa
    Function: Regulates secretion, blood flow
```

## Sphincters

### Upper Esophageal Sphincter (UES)

| Component | Details |
|-----------|---------|
| **Main muscle** | Cricopharyngeus (inferior pharyngeal constrictor) |
| **Level** | C5-C6 |
| **Length** | 2-4 cm |
| **Resting pressure** | 40-100 mmHg |
| **Muscle type** | Striated (voluntary) |
| **Relaxation** | Swallow-initiated, vagal mediated |

**Clinical correlates**:
- Zenker diverticulum (posterior weakness above cricopharyngeus)
- Cricopharyngeal dysfunction → dysphagia

### Lower Esophageal Sphincter (LES)

See detailed note: [[LES Physiology]]

| Component | Details |
|-----------|---------|
| **Intrinsic** | Thickened circular smooth muscle (3-4 cm) |
| **Extrinsic** | Right crus of diaphragm |
| **Level** | T10-T11, at diaphragmatic hiatus |
| **Length** | 3-4 cm (1-2 cm intra-abdominal) |
| **Resting pressure** | 10-30 mmHg (above gastric) |
| **Muscle type** | Smooth (involuntary) |
| **Relaxation** | Swallow-initiated, NO/VIP mediated |

## Clinical Correlations

| Anatomical Feature | Clinical Relevance |
|-------------------|-------------------|
| **No serosa** | Higher leak rates, cancer spread |
| **Three narrowings** | Foreign body, corrosive injury, cancer sites |
| **Submucosal lymphatics** | Skip metastases, wide margins needed |
| **Portal venous connection** | Esophageal varices in portal HTN |
| **Myenteric plexus** | Achalasia (neuron loss), Chagas |
| **Striated to smooth transition** | Different pathology upper vs lower |
| **Right crus** | Hiatal hernia when disrupted |
| **Recurrent laryngeal nerve** | Hoarseness with esophageal surgery/cancer |

## Quick Recall Questions

1. Q: What is the clinical significance of the esophagus lacking a serosa?
   A: Higher anastomotic leak rates, earlier cancer spread to adjacent structures, easier spread of infection to mediastinum

2. Q: At what distance from the incisors is the GEJ typically located?
   A: 40-42 cm

3. Q: Which esophageal veins connect to the portal system and cause varices?
   A: Lower esophageal veins → left gastric (coronary) vein → portal vein
