# Surgical Management of GERD

## Overview

Gastroesophageal reflux disease (GERD) affects 10-20% of adults in Western populations. While most patients respond to lifestyle modifications and medical therapy, approximately 30-40% require long-term PPI use, and 10-15% may benefit from surgical intervention. Anti-reflux surgery (ARS), primarily laparoscopic fundoplication, remains the definitive treatment for appropriately selected patients.

```
ANTI-REFLUX BARRIER - ANATOMY
═════════════════════════════

Normal Anti-Reflux Mechanisms:

    Esophagus
        │
        │   1. LES intrinsic tone (10-35 mmHg)
        │
    ╔═══╧═══╗ ←── 2. Acute angle of His
    ║  LES  ║
    ╚═══╤═══╝
        │     ←── 3. Phrenoesophageal ligament
    ════╪════     (anchors esophagus)
        │
    ┌───┴───┐ ←── 4. Crural diaphragm
    │       │     (external sphincter)
    │Stomach│
    │       │ ←── 5. Intra-abdominal esophagus
    └───────┘     (positive pressure segment)

Failure of any component → GERD
Fundoplication recreates/augments these mechanisms
```

## Pathophysiology of GERD

### Mechanisms of Reflux

```
GERD PATHOPHYSIOLOGY
════════════════════

PRIMARY MECHANISMS:
─────────────────────────────────────────────────────────
1. Transient LES Relaxations (TLESRs)
   • Most common mechanism (50-70%)
   • Physiologic but excessive
   • Not related to swallowing

2. Hypotensive LES
   • Resting pressure <10 mmHg
   • Scleroderma, post-myotomy

3. Anatomic disruption
   • Hiatal hernia
   • Loss of angle of His
   • Short intra-abdominal esophagus

CONTRIBUTING FACTORS:
─────────────────────────────────────────────────────────
• Impaired esophageal clearance
• Delayed gastric emptying
• Increased intra-abdominal pressure (obesity)
• Diet/lifestyle factors
• Medications (CCB, nitrates, anticholinergics)
```

## Indications for Anti-Reflux Surgery

### Established Indications

```
INDICATIONS FOR ANTI-REFLUX SURGERY
═══════════════════════════════════

DEFINITE INDICATIONS:
┌─────────────────────────────────────────────────────────┐
│ 1. Refractory to medical therapy                        │
│    • Persistent symptoms despite optimal PPI            │
│    • Must confirm ongoing reflux (pH study)            │
│                                                         │
│ 2. PPI-responsive but desires surgery                  │
│    • Long-term PPI concerns                            │
│    • Medication non-compliance                          │
│    • Quality of life preference                         │
│                                                         │
│ 3. Large hiatal hernia                                  │
│    • Type II, III, IV (paraesophageal)                │
│    • Symptomatic Type I >5 cm                          │
│                                                         │
│ 4. Extra-esophageal manifestations                      │
│    • Recurrent aspiration                              │
│    • Chronic laryngitis                                │
│    • Asthma exacerbated by reflux                      │
│                                                         │
│ 5. Barrett's with intractable symptoms                 │
│    (does NOT prevent progression to cancer)            │
└─────────────────────────────────────────────────────────┘

RELATIVE INDICATIONS:
• Young patients facing lifelong PPI
• Volume regurgitation despite PPI
• Medication side effects
```

### Pre-operative Evaluation

| Test | Purpose | Key Findings |
|------|---------|--------------|
| **EGD** | Mucosal assessment | Esophagitis grade, Barrett's, stricture, mass |
| **pH/Impedance** | Confirm reflux | DeMeester >14.7, AET >6% |
| **HRM** | Motility assessment | Rule out achalasia, assess peristalsis |
| **Barium swallow** | Anatomy | Hernia size, esophageal length |
| **Gastric emptying** | Gastroparesis | If delayed, consider pyloroplasty |

```

═══════════════════════

         ▼
```

## Surgical Options

### Fundoplication Types

```

═════════════════════════

NISSEN (360° TOTAL)              TOUPET (270° POSTERIOR)
────────────────────             ───────────────────────

    Esophagus                        Esophagus
        │                                │
   ┌────┴────┐                      ╱────┴────╲
   │ ╭─────╮ │                    ╱            ╲
   │ │     │ │                   │              │
   │ │ Eso │ │                   │     Eso     │
   │ │     │ │                   │              │
   │ ╰─────╯ │                    ╲            ╱
   └─────────┘                      ╲────┬────╱
                                        │

• Complete wrap               • Partial posterior wrap
• Best reflux control         • Less dysphagia
• Higher dysphagia           • Used with weak motility
• Standard for normal        • Good reflux control
  motility


DOR (180° ANTERIOR)              WATSON (120° ANTERIOR)
───────────────────              ─────────────────────

    Esophagus                        Esophagus
        │                                │
   ┌────┴────┐                      ────┴────
   │ ╭─────╮ │                     ╭─────╮
   │ │     │                       │     │
   │ │ Eso │                       │ Eso │
   │ │     │                       │     │
   │ ╰─────╯                       ╰─────╯
   └─────────

• Partial anterior wrap      • Minimal wrap
• Used with myotomy          • Rarely used alone
  (Heller procedure)         • Lowest dysphagia
• Covers myotomy site        • Lowest reflux control
```

### Selection Criteria

| Patient Factor | Recommended Wrap |
|---------------|------------------|
| Normal peristalsis | Nissen (360°) |
| Weak peristalsis (IEM) | Toupet (270°) |
| Very weak/absent motility | Dor (180°) or Toupet |
| Post-myotomy (achalasia) | Dor (180°) |
| Failed prior Nissen | Toupet or Dor |
| Morbid obesity | Consider RYGB instead |

## Laparoscopic Nissen Fundoplication

### Surgical Technique

```
LAPAROSCOPIC NISSEN - KEY STEPS
═══════════════════════════════

POSITION: Supine, split-leg, reverse Trendelenburg (15-20°)

PORT PLACEMENT:
     ┌─────────────────────────┐
     │          ○              │  Camera (10-12mm)
     │         /│\             │
     │        / │ \            │
     │       ○  │  ○           │  Working ports (5mm)
     │      /   │   \          │
     │     ○    │    ○         │  Liver retractor + Assistant
     └─────────────────────────┘

CRITICAL STEPS:

1. HIATAL DISSECTION
   ┌───────────────────────────────────────────┐
   │ • Divide gastrohepatic ligament          │
   │ • Identify right crus                     │
   │ • Dissect posterior to esophagus         │
   │ • Identify left crus                      │
   │ • Create retroesophageal window          │
   │ • Preserve vagus nerves                   │
   └───────────────────────────────────────────┘

2. FUNDIC MOBILIZATION
   ┌───────────────────────────────────────────┐
   │ • Divide short gastric vessels           │
   │ • Mobilize fundus completely              │
   │ • Ensure "floppy" fundus                 │
   │ • No tension on wrap                      │
   └───────────────────────────────────────────┘

3. CRURAL CLOSURE
   ┌───────────────────────────────────────────┐
   │ • Approximate crura posteriorly          │
   │ • Non-absorbable sutures (0 silk/Ethibond)│
   │ • Snug but not tight                      │
   │ • Mesh if large defect (controversial)   │
   └───────────────────────────────────────────┘

4. FUNDOPLICATION
   ┌───────────────────────────────────────────┐
   │ • Pass fundus posterior to esophagus     │
   │ • "Shoeshine" maneuver to confirm        │
   │ • Create 2-3 cm wrap                      │
   │ • Include esophageal wall in sutures     │
   │ • 50-56 Fr bougie for calibration        │
   │ • 2-3 sutures (non-absorbable)           │
   └───────────────────────────────────────────┘
```

### The "Floppy Nissen" Principles

```
FLOPPY NISSEN - KEY PRINCIPLES
══════════════════════════════

    ┌─────────────────────────────────────────────┐
    │                                              │
    │  1. SHORT WRAP: 2-3 cm (not >3 cm)          │
    │     • Longer wrap = more dysphagia          │
    │                                              │
    │  2. TENSION-FREE                             │
    │     • Complete fundic mobilization          │
    │     • Divide all short gastrics             │
    │                                              │
    │  3. CALIBRATED                               │
    │     • 50-56 Fr bougie during construction   │
    │     • Prevents too-tight wrap               │
    │                                              │
    │  4. FUNDUS-TO-FUNDUS                         │
    │     • True fundus wraps around              │
    │     • Not body of stomach                   │
    │     • "Shoeshine" confirms proper position  │
    │                                              │
    │  5. SECURE TO ESOPHAGUS                      │
    │     • Include esophageal muscle in sutures  │
    │     • Prevents slippage                      │
    │                                              │
    └─────────────────────────────────────────────┘
```

### Intraoperative Verification

| Maneuver | Purpose |
|----------|---------|
| "Shoeshine" test | Confirm fundus passes freely |
| Bougie placement | Calibrate wrap tightness |
| Intraoperative EGD | Verify wrap position, no twist |
| Retroflexion view | Confirm symmetric wrap |

## Toupet (Partial Posterior) Fundoplication

### Technique

```
TOUPET FUNDOPLICATION
═════════════════════

Steps 1-3 same as Nissen (dissection, mobilization, crural closure)

FUNDOPLICATION:
                        ┌─────────────┐
                        │  Esophagus  │
                        │      │      │
                    ╱───┘      │      └───╲
                   │           │           │
                   │    270°   │    wrap   │
                   │           │           │
                    ╲──────────┴──────────╱
                            │
                         Stomach

Construction:
1. Pass fundus POSTERIOR to esophagus
2. Suture fundus to LEFT crus (2-3 sutures)
3. Suture fundus to LEFT esophageal wall
4. Suture fundus to RIGHT crus (2-3 sutures)
5. Suture fundus to RIGHT esophageal wall
6. Creates 270° posterior wrap
7. Anterior esophagus remains free (90° gap)

Advantages:
• Less dysphagia than Nissen
• Maintains ability to belch/vomit
• Good for weak peristalsis
• Effective reflux control (85-90%)
```

## Special Situations

### Large Paraesophageal Hernia

```
PARAESOPHAGEAL HERNIA REPAIR
════════════════════════════

KEY PRINCIPLES:
┌──────────────────────────────────────────────────────┐
│                                                       │
│  1. COMPLETE REDUCTION                               │
│     • Reduce all herniated contents                  │
│     • Excise hernia sac (controversial)             │
│                                                       │
│  2. ADEQUATE ESOPHAGEAL LENGTH                       │
│     • Need ≥2-3 cm intra-abdominal esophagus        │
│     • If short → Collis gastroplasty                │
│                                                       │
│  3. CRURAL REPAIR                                    │
│     • Primary closure if possible                   │
│     • Mesh reinforcement for large defects          │
│     • Biologic > Synthetic (erosion risk)          │
│                                                       │
│  4. FUNDOPLICATION                                   │
│     • Anchors stomach below diaphragm               │
│     • Nissen or Toupet based on motility           │
│                                                       │
│  5. GASTROPEXY (optional)                           │
│     • Fixes stomach to anterior abdominal wall     │
│     • Reduces recurrence                             │
│                                                       │
└──────────────────────────────────────────────────────┘
```

### Short Esophagus - Collis Gastroplasty

```
COLLIS GASTROPLASTY
═══════════════════

Indication: Short esophagus (<2 cm intra-abdominal)

Technique:

    Normal                    After Collis
    ───────                   ────────────

        │                          │
        │                          │
    ════╪════ Diaphragm       ════╪════
        │                          │
    ┌───┴───┐                  │   │
    │       │                  │   │ ← Neo-esophagus
    │Stomach│              ┌───┴───┴───┐
    │       │              │           │
    └───────┘              │  Stomach  │
                           │           │
                           └───────────┘

• Stapled division of proximal stomach
• Creates 3-5 cm "neo-esophagus"
• Allows tension-free wrap below diaphragm
• Fundoplication around neo-esophagus
```

### Mesh in Hiatal Repair

| Factor | Consideration |
|--------|---------------|
| When to use | Large defect (>5 cm), recurrent hernia |
| Mesh type | Biologic preferred (lower erosion) |
| Placement | Onlay reinforcement, not bridging |
| Controversy | Mixed evidence on recurrence benefit |
| Complications | Mesh erosion into esophagus (rare but serious) |

## Outcomes and Complications

### Expected Outcomes

```
FUNDOPLICATION OUTCOMES
═══════════════════════

SUCCESS RATES (symptom control):
─────────────────────────────────────────────────
                    Short-term    Long-term
                    (1-2 years)   (5-10 years)
─────────────────────────────────────────────────
Heartburn relief    90-95%        80-85%
Regurgitation       95%           85-90%
Overall satisfaction 85-90%       75-85%
─────────────────────────────────────────────────

FUNCTIONAL OUTCOMES:
─────────────────────────────────────────────────
Dysphagia (early)           30-50% (transient)
Dysphagia (persistent)      3-5%
Bloating                    10-15%
Inability to vomit          5-10%
PPI discontinuation         70-80%
─────────────────────────────────────────────────
```

### Complications

```
COMPLICATIONS OF FUNDOPLICATION
═══════════════════════════════

INTRAOPERATIVE:
┌──────────────────────────────────────────────────┐
│ • Esophageal/gastric perforation (1-2%)         │
│ • Splenic injury (0.5-1%)                        │
│ • Pneumothorax (1-2%)                           │
│ • Vagal nerve injury (rare)                     │
│ • Bleeding                                       │
└──────────────────────────────────────────────────┘

EARLY POSTOPERATIVE:
┌──────────────────────────────────────────────────┐
│ • Dysphagia (common, usually resolves)          │
│ • Bloating/gas-bloat syndrome                   │
│ • Diarrhea (vagal injury)                       │
│ • Wrap disruption/leak                          │
│ • Pulmonary complications                        │
└──────────────────────────────────────────────────┘

LATE:
┌──────────────────────────────────────────────────┐
│ • Recurrent reflux (10-15%)                     │
│ • Wrap herniation (5-10%)                       │
│ • Wrap disruption (slipped Nissen)             │
│ • Persistent dysphagia                          │
│ • Paraesophageal hernia                         │
└──────────────────────────────────────────────────┘
```

### "Slipped" Nissen

```
SLIPPED NISSEN FUNDOPLICATION
═════════════════════════════

Normal Wrap:              Slipped Wrap:

    │                          │
    │                          │
┌───┴───┐                     │
│ Wrap  │               ┌─────┴─────┐
│       │               │   Wrap    │
│  EGJ  │               │           │
│       │               │   Stomach │ ← Stomach
└───────┘               │   hernia- │   above wrap
    │                   │   ted     │
 Stomach                └───────────┘
                             │
                          Stomach

Causes:
• Wrap not secured to esophagus
• Wrap too loose
• Inadequate crural repair
• Poor tissue quality

Symptoms:
• Recurrent reflux
• Dysphagia
• Early satiety
• Epigastric pain

Management:
• Re-do fundoplication
• May need conversion to Toupet
```

## Alternative/Emerging Procedures

### Magnetic Sphincter Augmentation (LINX)

```
LINX DEVICE
═══════════

┌─────────────────────────────────────────────┐
│                                              │
│  Device: Ring of titanium beads with        │
│          magnetic cores                      │
│                                              │
│      ┌─○─○─○─○─○─○─○─○─┐                    │
│      │                 │  ← Magnetic ring   │
│      └─────────────────┘    around LES      │
│                                              │
│  Mechanism:                                  │
│  • Magnets attract = closed sphincter       │
│  • Swallowing overcomes magnets = opens     │
│  • Allows burp/vomit (unlike tight Nissen)  │
│                                              │
│  Outcomes:                                   │
│  • 85-90% symptom relief                    │
│  • Less dysphagia than Nissen               │
│  • Less gas-bloat                           │
│                                              │
│  Concerns:                                   │
│  • MRI compatibility (newer versions OK)    │
│  • Device erosion (rare)                    │
│  • Long-term data limited                   │
│  • Cost                                      │
│                                              │
└─────────────────────────────────────────────┘
```

### Transoral Incisionless Fundoplication (TIF)

| Aspect | Details |
|--------|---------|
| Approach | Endoscopic (no incisions) |
| Mechanism | Creates 2-3 cm valve at EGJ |
| Device | EsophyX device |
| Efficacy | 60-70% (lower than surgical) |
| Best for | Small hiatal hernia (<2 cm), mild GERD |
| Limitations | Cannot address large hernia |

### Roux-en-Y Gastric Bypass for GERD

**When to consider:**
- Morbid obesity + GERD
- Failed fundoplication in obese patient
- Barrett's esophagus in obese patient

**Advantages:**
- Addresses both obesity and GERD
- Excellent reflux control
- May improve Barrett's

## Redo Surgery

### Indications for Reoperation

```
INDICATIONS FOR REDO FUNDOPLICATION
═══════════════════════════════════

1. ANATOMIC FAILURE
   • Wrap herniation
   • Wrap disruption
   • Slipped Nissen
   • Twisted/malpositioned wrap

2. FUNCTIONAL FAILURE
   • Recurrent reflux (confirmed on pH study)
   • Too-tight wrap (persistent dysphagia)

3. NEW PATHOLOGY
   • Development of paraesophageal hernia
   • Esophageal stricture

WORKUP BEFORE REDO:
┌──────────────────────────────────────────────┐
│ • Detailed symptom history                   │
│ • EGD (anatomy, esophagitis, Barrett's)     │
│ • Barium swallow (wrap position, hernia)    │
│ • HRM (rule out motility disorder)          │
│ • pH/impedance (confirm ongoing reflux)     │
│ • Gastric emptying if indicated             │
└──────────────────────────────────────────────┘
```

### Redo Surgery Principles

| Principle | Rationale |
|-----------|-----------|
| Complete takedown | Must fully dismantle old wrap |
| Re-assess esophageal length | May need Collis |
| Consider partial wrap | Toupet if Nissen failed |
| Thorough crural repair | Often needs mesh |
| Higher complication rate | 10-15% vs 2-3% primary |
