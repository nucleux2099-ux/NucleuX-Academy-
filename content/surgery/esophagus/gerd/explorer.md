# GERD - Gastroesophageal Reflux Disease

> **Source**: Shackelford Ch 7 - *Pathophysiology of Gastroesophageal Reflux*, Ch 8 - *GERD*, Ch 9 - *Barrett's Esophagus*, Ch 10 - *Antireflux Procedures*
> **Topic System**: [[Esophagus Index]]
> **Phase**: Shoot

---

## Core Concepts

> [!summary] One-liner
> GERD results from excessive esophageal acid exposure (>4% time pH <4) primarily due to transient LES relaxations (TLESRs), with diagnosis confirmed by symptoms + endoscopy/pH monitoring, and managed by PPIs first-line with fundoplication for refractory cases or patient preference.

## Pathophysiology

### Anti-Reflux Barrier Components

```
    THE ANTI-REFLUX BARRIER
    ═══════════════════════

    ┌─────────────────────────────────────────────────┐
    │                                                 │
    │   1. INTRINSIC LES                              │
    │      • Tonically contracted smooth muscle       │
    │      • Resting pressure 10-30 mmHg              │
    │      • 3-4 cm length                            │
    │                                                 │
    │   2. CRURAL DIAPHRAGM (Extrinsic)               │
    │      • Right crus forms sling                   │
    │      • Augments LES during inspiration          │
    │      • Contributes ~50% of barrier              │
    │                                                 │
    │   3. INTRA-ABDOMINAL ESOPHAGUS                  │
    │      • 1-2 cm below diaphragm                   │
    │      • Positive pressure environment            │
    │                                                 │
    │   4. ANGLE OF HIS                               │
    │      • Acute cardioesophageal angle             │
    │      • Creates mucosal flap valve               │
    │                                                 │
    │   5. PHRENOESOPHAGEAL LIGAMENT                  │
    │      • Anchors GEJ to diaphragm                 │
    │                                                 │
    └─────────────────────────────────────────────────┘
```

### Mechanisms of Reflux

```
    THREE MECHANISMS OF REFLUX
    ══════════════════════════

    1. TRANSIENT LES RELAXATIONS (TLESRs) ──── 50-70% of episodes
       │
       ├── NOT triggered by swallowing
       ├── Triggered by gastric distension
       ├── Vagal reflex (for venting/belch)
       └── Duration: 10-60 seconds

    2. HYPOTENSIVE LES ──────────────────────── 20-30% of patients
       │
       ├── Resting LES pressure <10 mmHg
       ├── Free reflux can occur
       └── Often with hiatal hernia

    3. ANATOMICAL DISRUPTION ────────────────── 40-60% with hiatal hernia
       │
       ├── Hiatal hernia separates LES from crura
       ├── Loss of dual sphincter effect
       └── "Acid pocket" in hernia sac
```

### The "Acid Pocket" Phenomenon

```
    ACID POCKET IN HIATAL HERNIA
    ════════════════════════════

    NORMAL                      WITH HIATAL HERNIA
    ──────                      ──────────────────

    Esophagus                   Esophagus
        │                           │
        │                           │
    ────┼──── Diaphragm        ────┼──── Diaphragm
        │                       ╭───┴───╮
        │                       │░░░░░░░│ ← "Acid pocket"
    ────┴────                   │░░░░░░░│   unbuffered acid
    ╱        ╲                  ╰───┬───╯   in hernia sac
   │ Stomach  │                     │
                                ────┴────
                               ╱         ╲
                              │  Stomach  │

    ░░░ = Unbuffered gastric acid
          (escapes meal buffering)
```

## Clinical Presentation

### Typical Symptoms

| Symptom | Description |
|---------|-------------|
| **Heartburn** | Retrosternal burning, postprandial, worse supine |
| **Regurgitation** | Effortless return of gastric contents |
| **Dysphagia** | May indicate stricture, severe esophagitis, or cancer |
| **Chest pain** | Can mimic angina; must rule out cardiac first |

### Atypical/Extraesophageal Symptoms

| Symptom | Mechanism |
|---------|-----------|
| **Chronic cough** | Microaspiration, vagal reflex |
| **Hoarseness** | Laryngeal inflammation |
| **Asthma exacerbation** | Bronchospasm from reflux |
| **Dental erosions** | Acid effect on enamel |
| **Globus sensation** | "Lump in throat" |
| **Throat clearing** | Posterior laryngeal irritation |

### Alarm Features (Require EGD)

| Red Flag | Concern |
|----------|---------|
| **Dysphagia** | Stricture, malignancy |
| **Odynophagia** | Ulceration, severe esophagitis |
| **Weight loss** | Malignancy |
| **Anemia/GI bleeding** | Ulcer, cancer |
| **Age >50 with new symptoms** | Barrett's, malignancy |

    GRADE C                    GRADE D
    ─────────                  ─────────

    │             │            │ ░░░░░░░░░░░░░░░ │
    │   ───────   │            │ ░░░░░░░░░░░░░░░ │

### 24-Hour pH Monitoring

| Technique | Use |
|-----------|-----|
| **Catheter-based** | Off PPI x 7 days; measures acid exposure |
| **Wireless (Bravo)** | Capsule attached to esophagus; more comfortable |
| **Impedance-pH** | Detects non-acid reflux; use ON PPI |

#### Key Parameters

| Parameter | Normal | Abnormal |
|-----------|--------|----------|
| **Total acid exposure time** | <4% | **>6%** |
| **DeMeester score** | <14.7 | **>14.7** |
| **Number of reflux episodes** | <50/day | >50/day |
| **Symptom correlation (SI)** | - | >50% = positive |
| **SAP** | - | >95% = positive |

```
    pH MONITORING INTERPRETATION
    ════════════════════════════

    24-hour pH tracing:

    pH 8 ─┬──────────────────────────────────
          │
    pH 4 ─┼─────┬─────┬─────┬─────┬─────────  ← pH 4 threshold
          │     │     │     │     │
    pH 2 ─┤     └─────┘     └─────┘
          │     ▲     ▲
          │  Reflux episodes
          └────────────────────────────────→
              0    6    12   18   24 hours

    Total time pH <4 / Total time = % acid exposure
```

## Complications of GERD

### Esophageal Complications

| Complication | Description | Management |
|--------------|-------------|------------|
| **Erosive esophagitis** | LA Grade A-D | PPI, healing in 8-12 weeks |
| **Stricture** | Fibrotic narrowing | Dilation + PPI |
| **Barrett's esophagus** | Intestinal metaplasia | Surveillance, ablation if dysplasia |
| **Ulceration** | Deep mucosal injury | High-dose PPI |
| **Adenocarcinoma** | Complicates Barrett's | Staging, treatment |

### Barrett's Esophagus

```
    BARRETT'S ESOPHAGUS
    ═══════════════════

    Definition: Replacement of normal squamous epithelium
                with intestinal metaplasia (goblet cells)

    NORMAL                      BARRETT'S
    ──────                      ─────────

    ┌─────────────┐             ┌─────────────┐
    │ Squamous    │             │░░░░░░░░░░░░░│ ← Columnar
    │ (pink)      │             │░░░░░░░░░░░░░│   (salmon)
    │             │             │             │
    │             │             │             │
    ├─────────────┤ Z-line      │░░░░░░░░░░░░░│
    │ Columnar    │             │░░░░░░░░░░░░░│
    │ (stomach)   │             ├─────────────┤
    └─────────────┘             │ Columnar    │
                                └─────────────┘

    ⚠️ Requires INTESTINAL METAPLASIA on biopsy
       (goblet cells) for diagnosis

    PROGRESSION:
    Barrett's → Low-grade dysplasia → High-grade dysplasia → Adenocarcinoma
       │              │                      │
    Surveillance    ↑Surveillance        Ablation/Resection
```

### Cancer Risk in Barrett's

| Finding | Annual Cancer Risk |
|---------|-------------------|
| **Non-dysplastic Barrett's** | 0.5% per year |
| **Low-grade dysplasia** | 1-2% per year |
| **High-grade dysplasia** | 5-10% per year |

## Medical Management

### Lifestyle Modifications

| Modification | Evidence |
|--------------|----------|
| **Weight loss** | Strong - reduces symptoms |
| **Elevate head of bed** | Moderate - for nocturnal symptoms |
| **Avoid late meals** | Moderate - 3 hours before bed |
| **Avoid trigger foods** | Weak - individualized |
| **Smoking cessation** | Moderate - reduces LES pressure |
| **Limit alcohol** | Weak - may help |

### Pharmacological Therapy

#### Proton Pump Inhibitors (PPIs)

```
    PPI MECHANISM
    ═════════════

    Parietal Cell:

    ┌─────────────────────────────────────────┐
    │                                         │
    │   H+/K+-ATPase ────► H+ into lumen      │
    │        ▲                                │
    │        │                                │
    │        ✕ ← PPI (irreversible block)     │
    │                                         │
    │   Proton pump is IRREVERSIBLY inhibited │
    │   New pumps must be synthesized         │
    │                                         │
    └─────────────────────────────────────────┘

    Key points:
    • Take 30-60 min before meals (pumps active when stimulated)
    • Once daily often sufficient
    • Twice daily for erosive esophagitis or EoE
```

| PPI | Standard Dose | Notes |
|-----|---------------|-------|
| **Omeprazole** | 20 mg daily | OTC available |
| **Esomeprazole** | 20-40 mg daily | S-isomer |
| **Pantoprazole** | 40 mg daily | Fewer drug interactions |
| **Lansoprazole** | 30 mg daily | - |
| **Rabeprazole** | 20 mg daily | - |

#### Other Medications

| Drug Class | Example | Use |
|------------|---------|-----|
| **H2 blockers** | Famotidine | Nocturnal acid breakthrough |
| **Antacids** | Tums, Maalox | Acute relief |
| **Prokinetics** | Metoclopramide | Gastroparesis component |
| **Baclofen** | GABA-B agonist | Reduces TLESRs (refractory) |

## Surgical Management

### Indications for Surgery

| Indication | Details |
|------------|---------|
| **Refractory to medical therapy** | Symptoms despite optimal PPI |
| **Patient preference** | Avoid lifelong medication |
| **Volume regurgitation** | Not controlled by acid suppression |
| **Medication intolerance** | PPI side effects |
| **Large hiatal hernia** | Symptomatic Type II-IV |
| **Documented GERD** | Must confirm with pH/impedance |

> [!warning] Pre-operative Requirements
> - **Confirm GERD**: pH monitoring (especially if no esophagitis)
> - **Evaluate motility**: HRM to rule out achalasia, severe dysmotility
> - **EGD**: Rule out Barrett's, stricture, cancer
> - **Appropriate expectations**: Surgery may not cure all symptoms

### Pre-operative Evaluation

```
    PRE-FUNDOPLICATION WORKUP
    ═════════════════════════

    1. EGD
       • Confirm no Barrett's needing ablation first
       • Grade esophagitis
       • Rule out stricture, mass
       • Assess hiatal hernia size

    2. ESOPHAGEAL MANOMETRY (HRM)
       • Rule out achalasia (fails to relax)
       • Assess peristaltic function
       • Guides wrap type:
         - Normal motility → Nissen (360°)
         - Poor motility → Toupet (270°)

    3. 24-HOUR pH MONITORING
       • Essential if no esophagitis on EGD
       • Confirms pathologic reflux
       • Correlates symptoms with reflux

    4. BARIUM SWALLOW
       • Optional
       • Anatomy (hiatal hernia size, short esophagus)
       • Stricture evaluation
```

### Surgical Options

See detailed note: [[Fundoplication]]

| Procedure | Description | Best For |
|-----------|-------------|----------|
| **Nissen (360°)** | Complete wrap | Normal motility, best reflux control |
| **Toupet (270°)** | Posterior partial | Impaired motility, redo |
| **Dor (180°)** | Anterior partial | After Heller myotomy |
| **LINX** | Magnetic sphincter | Alternative to fundoplication |

### LINX Device

```
    LINX MAGNETIC SPHINCTER AUGMENTATION
    ════════════════════════════════════

    • Ring of magnetic titanium beads
    • Placed around LES laparoscopically
    • Beads separate with swallow pressure
    • Close to prevent reflux

          Rest                    Swallow
         ─────                    ───────

      ┌───●●●───┐              ┌───●─●─●───┐
      │   LES   │       →      │    LES    │
      │ closed  │              │   opens   │
      └─────────┘              └───────────┘

    Advantages: Preserves belch, reversible
    Contraindications: Large hiatal hernia, Barrett's
```

## Outcomes

### Medical Therapy

| Outcome | Rate |
|---------|------|
| **Symptom relief with PPI** | 70-80% |
| **Healing of esophagitis (8 weeks)** | 85-95% |
| **Long-term PPI dependence** | 50-80% relapse off PPI |

### Surgical Therapy

| Outcome | Rate |
|---------|------|
| **Symptom control at 5 years** | 85-90% |
| **Off all acid suppression** | 70-80% |
| **Need for redo surgery** | 5-10% |
| **Post-op dysphagia (persistent)** | 3-5% |

## Follow-up and Surveillance

### Non-Barrett's GERD

- No routine surveillance needed
- Follow-up based on symptoms
- Consider EGD if symptoms change

### Barrett's Esophagus Surveillance

| Finding | Surveillance Interval |
|---------|----------------------|
| **Non-dysplastic** | EGD every 3-5 years |
| **Low-grade dysplasia** | EGD every 6-12 months |
| **High-grade dysplasia** | Endoscopic treatment (ablation/EMR) |

## Quick Recall Questions

1. Q: What is the most common mechanism of reflux in GERD?
   A: Transient lower esophageal sphincter relaxations (TLESRs) - account for 50-70% of reflux episodes

2. Q: What pH monitoring value confirms pathologic acid exposure?
   A: Total acid exposure time >6% (time pH <4) or DeMeester score >14.7

3. Q: When is esophageal manometry required before fundoplication?
   A: Always - to rule out achalasia (would make reflux worse) and assess motility (guides wrap type - partial if poor motility)

4. Q: What is the annual cancer risk in non-dysplastic Barrett's esophagus?
   A: Approximately 0.5% per year
