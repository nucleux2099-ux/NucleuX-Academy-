## R - Recall Mnemonics

### Chicago Classification Categories - "ICE MAP"

```
┌──────────────────────────────────────────────────────┐
│                                                       │
│   I - IRP first (≥15 or <15?)                        │
│   C - Contractility pattern (present/absent/spastic) │
│   E - EGJ outflow obstruction                        │
│                                                       │
│   M - Major disorders (if IRP normal)                │
│   A - Absent, DES, hypercontractile                  │
│   P - Peristaltic minor disorders (IEM, fragmented)  │
│                                                       │
└──────────────────────────────────────────────────────┘
```

### Achalasia Subtypes - "123 = ABC"

```
TYPE 1 = "A"bsent (no esophageal activity)
TYPE 2 = "B"oth (body + pan-pressurization) - BEST outcome
TYPE 3 = "C"ontraction premature (spastic) - worst prognosis
```

### Key Numbers to Remember - "FIVE-FOUR-EIGHT-ONE"

```
┌─────────────────────────────────────────────────────┐
│                                                      │
│  15  → IRP cutoff (≥15 = EGJ outflow obstruction)   │
│                                                      │
│  4.5 → DL cutoff (<4.5s = premature = spastic)      │
│                                                      │
│  8000 → DCI upper limit (>8000 = Jackhammer)        │
│                                                      │
│  100 → DCI <100 = failed contraction                │
│                                                      │
│  450 → DCI <450 = weak (ineffective)                │
│                                                      │
└─────────────────────────────────────────────────────┘

Memory: "15 at the junction, 4.5 for timing,
         8000 is hammer-time, 100 is failure,
         450 is weakness"
```

### Major vs Minor Disorders - "FAD vs IF"

```
MAJOR DISORDERS (FAD):
  F - Failed (Absent contractility)
  A - Angry esophagus (DES - distal spasm)
  D - Dynamite (Hypercontractile/Jackhammer)

MINOR DISORDERS (IF):
  I - Ineffective esophageal motility
  F - Fragmented peristalsis
```

### EGJOO Differential - "MATOS"

```
When you find EGJOO, think "MATOS":

  M - Medication effect (opioids!)
  A - Artifact/Technical issue
  T - Tumor/Mass (pseudoachalasia)
  O - Obstruction (stricture, hiatal hernia)
  S - Scleroderma/Early achalasia
```

```

### Version Changes (v3.0 → v4.0)

```
KEY CHANGES IN CHICAGO v4.0 (2021)
══════════════════════════════════

v3.0                          →    v4.0
────                               ────
Position: Supine only         →    Supine primary + upright
IRP cutoff variable           →    Standardized ≥15 mmHg
EGJOO = definitive diagnosis  →    EGJOO needs confirmation
Fragmented: >50% with breaks  →    Break must be >5 cm

## E - Elaboration & Examples

### Case-Based Learning

**Case 1: Classic Presentation**

```
CASE: 45-year-old woman
─────────────────────────────────────

History:
• 3-year progressive dysphagia to solids, then liquids
• Regurgitation of undigested food
• Weight loss 15 lbs
• Occasional chest discomfort

EGD: Dilated esophagus, retained food, tight EGJ
     Biopsies normal

HRM Results:
• IRP: 22 mmHg (elevated)
• 100% failed peristalsis
• Panesophageal pressurization in 40% of swallows
• No normal contractions

DIAGNOSIS: Type II Achalasia

Treatment options:
1. Pneumatic dilation (good candidate)
2. LHM + Dor fundoplication
3. POEM

EXPECTED OUTCOME: ~90% response rate (best subtype)
```

**Case 2: Diagnostic Dilemma**

```
CASE: 68-year-old man
─────────────────────────────────────

History:
• 2-month dysphagia to solids
• 25-lb weight loss
• No prior GI history

HRM Results:
• IRP: 18 mmHg (elevated)
• Variable - some preserved peristalsis
• Not meeting full achalasia criteria

INITIAL DIAGNOSIS: EGJOO

RED FLAGS:
• Age >55
• Rapid onset
• Significant weight loss

WORKUP:
• CT Chest/Abdomen: 3 cm mass at EGJ
• EUS: Adenocarcinoma at EGJ

FINAL DIAGNOSIS: Pseudoachalasia
                 (secondary to EGJ malignancy)

LESSON: Always rule out malignancy in EGJOO,
        especially with red flags!
```

### Clinical Pearls

```
CLINICAL PEARLS
═══════════════

🔹 "Paradoxical dysphagia" (worse with liquids than solids)
   suggests achalasia - solids may push through by gravity

🔹 "Bird's beak" on barium swallow is classic for achalasia,
   but HRM is gold standard for diagnosis

🔹 Opioids can cause EGJOO - always check medication list!

🔹 Post-fundoplication dysphagia may show EGJOO pattern -
   consider wrap too tight vs. other causes

🔹 Chest pain in young patient + normal cardiac workup →
   consider DES or Jackhammer esophagus

🔹 Scleroderma = Absent contractility + severe GERD
   (incompetent LES + no clearance)

🔹 TBE at 1 and 5 minutes is excellent for f/u after
   achalasia treatment - objective emptying assessment
```
