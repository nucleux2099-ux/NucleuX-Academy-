# POEM - Peroral Endoscopic Myotomy

> **Source**: Shackelford Ch 19 - *Achalasia Diagnosis and Treatment*
> **Topic System**: [[Esophagus Index]]
> **Phase**: Shoot
> **Tier**: 4 (Surgical Technique)

---

## Core Concepts

> [!summary] One-liner
> POEM is an incisionless endoscopic myotomy for achalasia performed through a submucosal tunnel, allowing longer myotomies (ideal for Type III), but with higher GERD rates (20-50%) due to lack of fundoplication.

---

## Indications & Advantages

### Indications

| Indication | Notes |
|------------|-------|
| **Achalasia Type I, II** | Equivalent to LHM |
| **Achalasia Type III** | **Preferred** - longer myotomy possible |
| **Failed prior LHM** | Can approach from opposite side (posterior) |
| **Sigmoid achalasia** | Technically easier than redo surgery |
| **Patient preference** | Incisionless approach |

### POEM Advantages Over Heller

| Feature | POEM | Heller |
|---------|------|--------|
| **Incisions** | None | 5 ports |
| **Myotomy length** | Flexible (10-25 cm possible) | Standard 6-8 cm |
| **Approach** | Anterior or posterior | Anterior only |
| **Type III suitability** | Excellent | Suboptimal |
| **Recovery** | Faster | Slightly longer |
| **Redo capability** | Can use opposite side | Limited |

### POEM Disadvantages

| Disadvantage | Details |
|--------------|---------|
| **Higher GERD** | 20-50% (no fundoplication) |
| **Requires expertise** | Steep learning curve |
| **Long-term data** | Limited (10 years max) |
| **Gas complications** | Pneumomediastinum, pneumoperitoneum |

## Operative Technique

### Setup

```
    POEM SETUP
    ══════════

    • General anesthesia with intubation
    • Supine position
    • CO2 insufflation (not air - absorbs faster)
    • High-definition gastroscope with cap
    • Electrosurgical generator (spray coag, soft coag)
    • Coagulation forceps, injection needle
    • Clips for closure
```

### Step-by-Step Procedure

#### Step 1: Mucosal Entry

```
    MUCOSAL INCISION SITE
    ═════════════════════

    Location: 10-15 cm above GEJ
    Position: Posterior (5-6 o'clock) or Anterior (2 o'clock)

           10-15 cm above GEJ
                  │
                  ▼
         ┌───────────────┐
         │               │
         │    ●          │ ← Mucosal incision
         │  (entry)      │   (2 cm longitudinal)
         │               │
         │               │
         │               │
         └───────────────┘
                  │
                  ▼
              Toward GEJ

    Technique:
    1. Submucosal injection (saline + methylene blue)
    2. Create mucosal bleb
    3. Incise mucosa longitudinally (2 cm)
    4. Enter submucosal space
```

#### Step 2: Submucosal Tunnel

```
    SUBMUCOSAL TUNNELING
    ═══════════════════

              Entry ●
                  │
                  │  Tunnel created
                  │  in submucosal
                  │  plane
                  │
                  │  (Inject, spread,
                  │   dissect)
                  │
              ────┼──── Diaphragm
                  │
                  │  Continue 2-3 cm
                  │  onto gastric cardia
                  │
                  ▼
              End of tunnel

    Key Points:
    • Use spray coagulation
    • Inject saline to maintain plane
    • Blue staining from methylene blue guides
    • Tunnel must extend onto stomach (like Heller)
```

#### Step 3: Myotomy

```
    SELECTIVE CIRCULAR MYOTOMY
    ══════════════════════════

    Cross-sectional view:

    ┌─────────────────────────────────┐
    │         MUCOSA                  │ ← Intact (in tunnel)
    ├─────────────────────────────────┤
    │       SUBMUCOSA                 │ ← Tunnel space
    ├─────────────────────────────────┤
    │ ████ CIRCULAR MUSCLE ████       │ ← DIVIDED (myotomy)
    ├─────────────────────────────────┤
    │    Longitudinal muscle          │ ← PRESERVED
    └─────────────────────────────────┘

    Technique:
    1. Identify circular muscle fibers (transverse)
    2. Hook and coagulate fibers
    3. Divide from proximal to distal
    4. Preserve longitudinal muscle (usually)
    5. Complete division at gastric cardia
```

**Myotomy Length**:

| Achalasia Type | Recommended Length |
|----------------|--------------------|
| Type I, II | 6-10 cm esophageal + 2-3 cm gastric |
| Type III | 10-20+ cm (extend to cover spastic segment) |

#### Step 4: Mucosal Closure

```
    CLOSURE TECHNIQUES
    ═══════════════════

    1. ENDOCLIPS
       • Multiple clips to approximate mucosal edges
       • Most common method
       • 5-10 clips typically

    2. ENDOSCOPIC SUTURING
       • OverStitch device
       • More secure closure

           ┌─●─●─●─●─●─┐
           │   Clips   │
           │   close   │
           │   entry   │
           └───────────┘
```

## Post-operative Care

### Immediate Management

| Item | Protocol |
|------|----------|
| **Observation** | Monitor for chest pain, fever, subcutaneous emphysema |
| **NPO** | Overnight |
| **Imaging** | Gastrografin swallow POD1 |
| **Antibiotics** | Continue 24-48 hours |
| **PPI** | Start immediately (high GERD rates) |

### Expected Gas-Related Findings

| Finding | Management |
|---------|------------|
| **Subcutaneous emphysema** | Common, self-resolving |
| **Pneumomediastinum** | Usually benign if no leak |
| **Pneumoperitoneum** | Needle decompression if tense |
| **Capnoperitoneum** | From CO2, resolves quickly |

---

## Complications

### Post-operative

| Complication | Incidence | Management |
|--------------|-----------|------------|
| **Pneumomediastinum** | 10-20% | Observe if stable |
| **Pneumoperitoneum** | 5-10% | Needle decompression if tense |
| **Leak** | 0.5-1% | NPO, antibiotics, drainage, stent |
| **GERD** | **20-50%** | PPI, may need late fundoplication |
| **Delayed bleeding** | <1% | Endoscopic hemostasis |

> [!warning] GERD After POEM
> Unlike Heller (with fundoplication), POEM has **no anti-reflux procedure**
> - GERD rate: 20-50%
> - All patients should be on PPI
> - May need fundoplication later if severe

## Outcomes

### Success Rates

| Outcome | Rate |
|---------|------|
| **Initial clinical success** | 90-98% |
| **2-3 year success** | 90-95% |
| **5-year success** | ~85-90% |

### By Achalasia Type

| Type | Success Rate | Notes |
|------|--------------|-------|
| Type I | 90-95% | Excellent |
| Type II | 95%+ | Best |
| Type III | **85-90%** | Better than Heller for this type |

### POEM vs Heller Head-to-Head

| Outcome | POEM | Heller |
|---------|------|--------|
| Short-term symptom relief | 95% | 93% |
| 2-year clinical success | 92% | 90% |
| GERD (objective) | 44% | 29% |
| Serious adverse events | 3% | 3% |

## Special Situations

### POEM After Failed Heller

- Approach from **posterior** (opposite to prior anterior myotomy)
- Tunnel through unscarred tissue
- Success rate: 80-90%

### POEM for Type III Achalasia

- **Preferred over Heller** for Type III
- Can extend myotomy 15-20 cm or more
- Address entire spastic segment
- Improved chest pain relief

## Quick Recall Questions

1. Q: What is the main advantage of POEM for Type III achalasia?
   A: Can perform a longer myotomy (10-20+ cm) to address the entire spastic segment

2. Q: What is the main disadvantage of POEM compared to Heller?
   A: Higher GERD rates (20-50%) because no fundoplication is performed

3. Q: How do you approach a failed Heller with POEM?
   A: Perform POEM from the posterior approach (opposite side from the anterior Heller myotomy)
