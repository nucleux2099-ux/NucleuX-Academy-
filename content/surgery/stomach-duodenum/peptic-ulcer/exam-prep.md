# 📋 Peptic Ulcer Disease — Exam Prep Mode

## 🎯 One-Liners
| Question | Answer |
|----------|--------|
| Two major causes | H. pylori and NSAIDs |
| Duodenal vs Gastric ulcer pain | DU: relieved by food; GU: worsened by food |
| Most common site for DU | First part of duodenum (anterior wall) |
| Most common complication | Bleeding |
| Johnson classification | Types I-V based on location |
| Test for H. pylori (non-invasive) | Urea breath test, Stool antigen |
| Triple therapy duration | 14 days |
| Indication for surgery now | Perforation, bleeding (failed endoscopy), obstruction |
| Graham patch | Omental patch for perforated DU |
| Truncal vagotomy + drainage | Reduces acid + ensures gastric emptying |

## 📊 Complications
- **Bleeding** (most common) - Melena, hematemesis
- **Perforation** - Acute abdomen, pneumoperitoneum
- **Obstruction** - Gastric outlet obstruction
- **Malignancy** - Gastric ulcer can harbor cancer

## 💊 Management
- PPI therapy
- H. pylori eradication (triple therapy)
- Stop NSAIDs
- Surgery for complications
