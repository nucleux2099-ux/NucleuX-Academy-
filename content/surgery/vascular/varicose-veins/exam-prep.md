# 📋 Varicose Veins — Exam Prep Mode

---

## 🎯 One-Liners

| Question | Answer |
|----------|--------|
| Definition | Dilated, tortuous superficial veins |
| Most commonly affected | Great saphenous vein (GSV) |
| Underlying mechanism | Valve incompetence |
| Trendelenburg test assesses | SFJ competence |
| Tap test (Schwartz) assesses | Valve competence along vein |
| Brodie-Trendelenburg test | Double tourniquet test |
| SFJ location | 4 cm below & lateral to pubic tubercle |
| SPJ location | Popliteal fossa |
| CEAP C6 | Active venous ulcer |
| Gold standard investigation | Duplex ultrasound |

---

## 📝 Clinical Examination

**Inspection (standing):**
- Distribution of veins
- Skin changes (pigmentation, lipodermatosclerosis)
- Ulceration (medial malleolus = venous)

**Palpation:**
- Saphena varix (cough impulse)
- Tenderness (thrombophlebitis)

**Special Tests:**
- Trendelenburg/Tourniquet test
- Tap test (Schwartz)
- Perthe's test (deep vein patency)

---

## 🔬 CEAP Classification (Clinical)

| Grade | Description |
|-------|-------------|
| C0 | No visible disease |
| C1 | Telangiectasia/spider veins |
| C2 | Varicose veins |
| C3 | Edema |
| C4 | Skin changes (pigmentation, eczema) |
| C5 | Healed ulcer |
| C6 | Active ulcer |

---

## 💊 Treatment

**Conservative:**
- Compression stockings
- Leg elevation
- Weight loss, exercise

**Interventional:**
- EVLA (Endovenous Laser)
- RFA (Radiofrequency Ablation)
- Foam sclerotherapy
- Surgery (SFJ ligation + stripping)
