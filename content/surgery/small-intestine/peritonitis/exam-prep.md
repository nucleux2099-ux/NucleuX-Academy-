# 📋 Peritonitis — Exam Prep Mode

## 🎯 One-Liners
| Question | Answer |
|----------|--------|
| Most common cause of secondary peritonitis | Appendicitis |
| Primary vs Secondary peritonitis | Primary: no viscus perforation; Secondary: perforation |
| Classic presentation | Severe abdominal pain, board-like rigidity |
| X-ray finding in perforation | Air under diaphragm (pneumoperitoneum) |
| Typhoid perforation site | Terminal ileum (antimesenteric border) |
| SBP (Spontaneous Bacterial Peritonitis) | In ascites, usually E. coli |
| Most common organism in secondary | E. coli (gram-negative) |
| Treatment of secondary peritonitis | Source control (surgery) + Antibiotics |
| Murphy's triad in SBP | Fever + abdominal pain + altered mentation |
| Ascitic fluid WBC in SBP | >250 neutrophils/mm³ |

## 📊 Classification
| Type | Cause | Example |
|------|-------|---------|
| Primary | No perforation | SBP in cirrhosis |
| Secondary | GI perforation | Appendicitis, PUD, trauma |
| Tertiary | Persistent infection | Post-surgical, ICU patients |

## 💊 Management
1. Resuscitation (IV fluids, catheter)
2. Antibiotics (broad-spectrum)
3. Source control (SURGERY for secondary)
4. ICU if severe

**Antibiotics:** Cover gram-negative + anaerobes
- Ceftriaxone + Metronidazole OR
- Piperacillin-Tazobactam
