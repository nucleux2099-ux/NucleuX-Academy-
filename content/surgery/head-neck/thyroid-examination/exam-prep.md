# 📋 Thyroid Examination — Exam Prep Mode

> **High-yield facts for NEET-PG, INICET, and clinical exams**

---

## 🎯 One-Liners (Most Asked)

| Question | Answer |
|----------|--------|
| How to confirm thyroid origin | Moves with SWALLOWING |
| How to differentiate from thyroglossal cyst | Thyroglossal moves with TONGUE PROTRUSION |
| Why thyroid moves with swallowing | Attached to pretracheal fascia |
| Position to examine thyroid | Standing BEHIND the patient |
| Berry's sign indicates | Malignancy (carotid encasement) |
| Pemberton's test indicates | Retrosternal extension (SVC compression) |
| Thyroid bruit indicates | Thyrotoxicosis (increased vascularity) |
| Kocher's test indicates | Malignancy (sympathetic invasion) |

## 📝 Examination Sequence (OSCE Format)

```
1. INTRODUCTION + CONSENT
2. GENERAL INSPECTION
   □ Build (thin/obese)
   □ Anxiety/tremor
   □ Eye signs
3. LOCAL INSPECTION
   □ Swelling location
   □ Movement with swallowing
   □ Movement with tongue protrusion
   □ Scars, skin changes
4. PALPATION (from behind)
   □ Temperature
   □ Size, shape, surface
   □ Consistency
   □ Tenderness
   □ Mobility
   □ Trachea position
5. PERCUSSION
   □ Retrosternal extension
6. AUSCULTATION
   □ Bruit
7. THYROID STATUS
   □ Pulse, tremor
   □ Eye signs
   □ Reflexes
8. LYMPH NODES
   □ All cervical groups
```

## 🔬 Clinical Signs to Know

### Eye Signs (Thyrotoxicosis)
| Sign | Description |
|------|-------------|
| **Lid retraction** | Sclera visible above iris at rest |
| **Lid lag** | Sclera appears on downward gaze |
| **Exophthalmos** | Protrusion of eyeballs |
| **Diplopia** | Extraocular muscle involvement |
| **Chemosis** | Conjunctival edema |

### Special Tests
| Test | Technique | Positive Finding |
|------|-----------|------------------|
| **Pemberton's** | Arms raised above head 1 min | Facial congestion, JVP rise |
| **Berry's** | Palpate carotid against thyroid | Absent carotid pulsation |
| **Kocher's** | Compress carotid against mass | Ipsilateral pupil dilation |

## 📊 Thyroid Status Comparison

| Feature | Hyperthyroid | Hypothyroid |
|---------|--------------|-------------|
| **Pulse** | Fast, bounding, AF | Slow, weak |
| **Skin** | Warm, sweaty | Cold, dry |
| **Hands** | Tremor, warm palms | Cold hands |
| **Weight** | Loss | Gain |
| **Bowels** | Diarrhea | Constipation |
| **Reflexes** | Brisk | Slow relaxing |
| **Voice** | Normal | Hoarse |
| **Face** | Anxious | Puffy (myxedema) |

## 🎨 Characteristics of Thyroid Swelling

### By Consistency
| Consistency | Suggests |
|-------------|----------|
| Soft | Colloid goitre, cyst |
| Firm | MNG, adenoma |
| Hard | Carcinoma, calcification |
| Variable | MNG (different areas) |

### By Surface
| Surface | Suggests |
|---------|----------|
| Smooth | Diffuse goitre, cyst |
| Nodular | MNG, adenoma |
| Irregular | Carcinoma |

## ⚠️ Red Flags for Malignancy

```
HIGH SUSPICION OF THYROID CANCER:
□ Hard nodule
□ Fixed to surrounding structures
□ Cervical lymphadenopathy
□ Rapid growth
□ Hoarseness (RLN involvement)
□ Dysphagia
□ Stridor
□ Berry's sign positive
□ Age <20 or >60
□ Male sex (higher risk)
□ History of radiation
□ Family history of MEN/thyroid cancer
```

## 📝 Differential Diagnosis of Neck Swellings

| Swelling | Movement with Swallowing | Movement with Tongue Protrusion |
|----------|--------------------------|--------------------------------|
| Thyroid | YES | NO |
| Thyroglossal cyst | YES | YES |
| Lymph node | NO | NO |
| Laryngocele | NO | NO (but increases with Valsalva) |

## 🔬 Goitre Classification

### By Function
| Type | TSH | T3/T4 | Examples |
|------|-----|-------|----------|
| Euthyroid | Normal | Normal | Simple goitre, MNG |
| Toxic | Low | High | Graves', toxic MNG |
| Hypothyroid | High | Low | Hashimoto's |

### By Morphology
- **Diffuse** — Graves', simple goitre, Hashimoto's
- **Uninodular** — Adenoma, cyst, cancer
- **Multinodular (MNG)** — Most common goitre

## 📝 Quick Revise Before Exam

- **Swallowing movement** = thyroid/thyroglossal
- **Tongue protrusion** = ONLY thyroglossal
- Examine from BEHIND the patient
- **Pemberton's** = retrosternal extension
- **Berry's** = carotid engulfed = malignancy
- **Bruit** = thyrotoxicosis
- **Hard + fixed + nodes** = MALIGNANCY
- Always check THYROID STATUS + LYMPH NODES

---

## 💡 Practical Exam Tips

```
✓ Ask patient to swallow (provide water)
✓ Stand BEHIND patient for palpation
✓ Assess BOTH lobes + isthmus
✓ Always complete with:
  - Thyroid status assessment
  - Cervical lymph node examination
  - Complications check
  
✗ DON'T forget:
  - To check tongue protrusion (thyroglossal DDx)
  - Eye signs if suspecting thyrotoxicosis
  - Tracheal position
```
