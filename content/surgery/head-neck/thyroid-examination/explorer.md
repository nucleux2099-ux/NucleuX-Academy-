## 👁️ Inspection — What to Look For

```
GENERAL INSPECTION (From End of Bed)
──────────────────────────────────────
• Body habitus (thin = ?thyrotoxic, obese = ?hypothyroid)
• Agitation/restlessness (thyrotoxicosis)
• Slow movements (hypothyroidism)
• Obvious neck swelling
• Facial features (puffy face = myxedema)

LOCAL INSPECTION (Neck Exposed)
───────────────────────────────
• Site of swelling (midline vs lateral)
• Size (small, visible, large, massive)
• Shape (diffuse, nodular, solitary nodule)
• Skin changes (redness, dilated veins, scars)
• Movement with SWALLOWING (confirms thyroid origin)
• Movement with TONGUE PROTRUSION (excludes thyroglossal cyst)

    ┌───────────────────────────────────────┐
    │    KEY DIAGNOSTIC MOVEMENTS           │
    │                                       │
    │    SWALLOWING: Thyroid moves UP       │
    │    (attached to pretracheal fascia)   │
    │                                       │
    │    TONGUE OUT: Thyroglossal cyst      │
    │    moves UP (attached to hyoid)       │
    │                                       │
    │    Both move with swallowing, but     │
    │    ONLY thyroglossal moves with       │
    │    tongue protrusion                  │
    └───────────────────────────────────────┘
```

```
──────────────────────────────────

    ┌─────────────────────────────┐

    │     EXAMINER (behind)       │
    │         ╭─────╮             │
    │        │░░░░░░│             │
    │        │░░░░░░│             │    Fingers wrap around
    │   ╭────┤░░░░░░├────╮        │    from both sides
    │   │    │░░░░░░│    │        │
    │   │    └──┬───┘    │        │
    │   │       │        │        │
    │   │   PATIENT      │        │
    │   │   (sitting)    │        │
    │                             │
    └─────────────────────────────┘

PALPATION CHECKLIST:
────────────────────
□ Temperature (warm = inflammation/thyrotoxicosis)
□ Tenderness (thyroiditis)
□ Size (each lobe separately, isthmus)
□ Shape (diffuse vs nodular)
□ Surface (smooth vs nodular vs irregular)
□ Consistency (soft, firm, hard, variable)
□ Margins (well-defined vs ill-defined)
□ Mobility (moves with swallowing, can get below it?)
□ Fixity (to skin, muscle, trachea)
□ Tracheal position (deviation = large goitre/cancer)
```

## 🩺 Thyroid Status — Look for These Signs

```
┌────────────────────┬────────────────────┬───────────────────┐
│   HYPERTHYROID     │     EUTHYROID      │    HYPOTHYROID    │
├────────────────────┼────────────────────┼───────────────────┤
│ • Anxiety, tremor  │                    │ • Slow, lethargic │
│ • Warm, sweaty     │     NO SYMPTOMS    │ • Cold, dry skin  │
│ • Tachycardia      │        OR          │ • Bradycardia     │
│ • Weight loss      │      SIGNS         │ • Weight gain     │
│ • Lid lag, stare   │                    │ • Puffy face      │
│ • Proximal myopathy│                    │ • Slow relaxing   │
│ • Atrial fibrillation                   │   reflexes        │
│ • Thyroid bruit    │                    │ • Hoarse voice    │
└────────────────────┴────────────────────┴───────────────────┘

SPECIFIC SIGNS TO ELICIT:
─────────────────────────
• FINE TREMOR: "Stretch your hands out" + paper on hands
• LID LAG: Follow finger moving down, look for sclera above iris
• LID RETRACTION: Visible sclera above iris at rest
• EXOPHTHALMOS: Stand behind, look from above for proptosis
• REFLEXES: Ankle jerk - slow relaxing = hypothyroid
• PULSE: Rate, rhythm (AF in thyrotoxicosis)
• PALMS: Warm/sweaty vs cold/dry
```

## 🎯 Special Maneuvers

```
1. PEMBERTON'S TEST (Retrosternal Extension)
   ──────────────────────────────────────────
   • Patient raises both arms above head
   • Hold for 1 minute
   • Positive: Facial congestion, cyanosis, distended neck veins
   • Indicates: Large goitre causing thoracic inlet obstruction

2. BERRY'S SIGN (Carotid Pulsation)
   ─────────────────────────────────
   • Normally carotid pulsation felt lateral to thyroid
   • If NOT palpable → Thyroid engulfing carotid
   • Suggests: Anaplastic carcinoma (highly invasive)

3. KOCHER'S TEST (Carotid Compression)
   ────────────────────────────────────
   • Compress carotid against thyroid mass
   • Positive: Dilation of ipsilateral pupil
   • Suggests: Malignancy invading sympathetic chain

4. TRACHEAL POSITION
   ─────────────────────
   • Feel trachea in suprasternal notch
   • Deviation away from large goitre = space-occupying
   • Deviation towards lesion = fibrosis/malignancy
```

## 💡 Key Atomic Concepts

### Atom 1: Movement with Swallowing
> The thyroid gland is attached to the pretracheal fascia. When you swallow, the larynx rises, taking the thyroid with it. This is pathognomonic for thyroid origin. NO other neck swelling does this (except thyroglossal cyst).

### Atom 2: Hard = Cancer Until Proven Otherwise
> Soft/firm = likely benign. HARD nodule = think malignancy (papillary, follicular, medullary, anaplastic). Fixed to surrounding structures is even more concerning.

### Atom 3: Always Check Regional Lymph Nodes
> Papillary carcinoma spreads to cervical lymph nodes early. Examine: Submental → Submandibular → Jugular chain → Posterior triangle → Supraclavicular. Enlarged nodes = staging and prognosis implications.
