# 🔧 Thyroid Examination — Clinical Examination Procedure

## Overview

| Aspect | Details |
|--------|---------|
| **Examination Type** | Clinical bedside/OSCE |
| **Duration** | 5-8 minutes |
| **Common in** | Short case, Long case, OSCE, Viva |

---

## 🔷 Step-by-Step Examination Sequence

### STEP 0: Introduction & Consent
```
"Good morning, I'm Dr. [Name]. 
May I examine your neck, please?"

✓ Wash hands / use sanitizer
✓ Adequate exposure (neck fully visible)
✓ Patient seated comfortably
```

---

### STEP 1: General Inspection (From End of Bed)

**What to observe:**
- Build (thin → thyrotoxic? obese → hypothyroid?)
- Apparent age vs actual age
- Anxiety, restlessness (thyrotoxicosis)
- Slow, lethargic (hypothyroidism)
- Obvious neck swelling
- Eye prominence

**Say:** "On general inspection, the patient appears [well/unwell, appropriate build, no obvious distress]."

---

### STEP 2: Local Inspection of Neck

**Position:** Face the patient, neck exposed

**Look for:**
- Site of swelling (midline, lateral, diffuse)
- Size (small, moderate, large)
- Shape (diffuse, nodular, solitary)
- Surface (smooth, nodular)
- Skin over swelling (scars, redness, dilated veins)
- Trachea position (visible deviation?)

**Special maneuvers:**

#### A. Movement with Swallowing
```
"Please take this sip of water and hold it in your mouth."
"Now swallow."

Watch the swelling move UP → Confirms thyroid origin
```

#### B. Movement with Tongue Protrusion
```
"Please stick your tongue out."

If swelling moves UP → Thyroglossal cyst
If swelling stays → Thyroid swelling
```

---

### STEP 3: Palpation

**Position:** Stand BEHIND the patient

```
┌─────────────────────────────────┐
│                                 │
│    EXAMINER (standing behind)   │
│                                 │
│         ↓ Fingers wrap around   │
│        ╭─────╮                  │
│       │      │                  │
│  ╭────┤THYROID├────╮            │
│  │    │      │    │            │
│  │    └──────┘    │            │
│  │                │            │
│  │    PATIENT     │            │
│  │   (sitting)    │            │
│                                 │
└─────────────────────────────────┘
```

**Palpation sequence:**

#### A. Temperature
"I'm going to feel your neck. Tell me if there's any tenderness."

#### B. Size
- Palpate each lobe separately
- Palpate isthmus
- Estimate size (normal lobe = thumb-sized)

#### C. Shape & Surface
- Diffuse enlargement
- Single nodule
- Multiple nodules

#### D. Consistency
- Soft (cyst, colloid)
- Firm (adenoma, MNG)
- Hard (carcinoma, calcification)
- Variable (MNG)

#### E. Tenderness
- Tender = thyroiditis, hemorrhage into cyst

#### F. Mobility
Ask patient to swallow → Feel movement
Try to move swelling side to side → Assess fixation

#### G. Tracheal Position
Feel in suprasternal notch
- Central
- Deviated

#### H. Can you get below it?
- Yes → Not retrosternal
- No → Retrosternal extension

#### I. Lower Border
If cannot feel lower border → ?Retrosternal extension

---

### STEP 4: Percussion (Retrosternal Extension)

**Position:** Face the patient

```
Percuss over manubrium from below upward
Resonant → Normal
Dull → Retrosternal goitre
```

---

### STEP 5: Auscultation

**Position:** Place stethoscope over thyroid lobes

```
"Please hold your breath for a moment."

Listen for:
- BRUIT (continuous murmur) → Thyrotoxicosis
- Transmitted sounds from carotid
```

---

### STEP 6: Special Tests

#### Pemberton's Test
```
"Please raise both arms above your head and keep them there."
Wait 1 minute

Positive if:
- Facial congestion/plethora
- Cyanosis
- JVP distension
- Stridor

Indicates: Retrosternal extension with thoracic inlet obstruction
```

#### Berry's Sign
```
Palpate carotid artery lateral to thyroid mass

Positive if:
- Cannot feel carotid pulsation

Indicates: Malignancy encasing carotid
```

---

### STEP 7: Thyroid Status Assessment

#### Hands
- Temperature (warm/cold)
- Sweating (moist/dry)
- Tremor (stretch out hands, place paper)
- Pulse (rate, rhythm, character)
- Palmar erythema
- Thyroid acropachy

#### Eyes
- Proptosis/Exophthalmos (look from above/behind)
- Lid retraction (sclera visible above iris at rest)
- Lid lag (follow finger downward)
- Chemosis (conjunctival swelling)
- Ophthalmoplegia (check eye movements)

#### Face
- Myxedematous facies (puffy, coarse features)
- Loss of outer 1/3 eyebrows

#### Reflexes
- Ankle jerk
- Slow relaxing = Hypothyroid
- Brisk = Hyperthyroid

---

### STEP 8: Lymph Node Examination

**Always complete with:**

```
Cervical lymph node examination:
□ Submental
□ Submandibular
□ Pre-auricular
□ Post-auricular
□ Occipital
□ Jugulodigastric
□ Upper, middle, lower jugular chain
□ Posterior triangle
□ Supraclavicular

Note: Papillary carcinoma spreads early to cervical nodes
```

---

### STEP 9: Completion

```
"To complete my examination, I would like to:
1. Examine the cardiovascular system
2. Check for proximal myopathy
3. Request thyroid function tests
4. Arrange ultrasound neck"

"Thank you. You can get dressed now."
```

## 🔷 Presenting Findings (Example)

```
"This is a [age]-year-old [male/female] with a swelling in the 
neck of [duration].

On inspection, there is a [size] swelling in the [location] that 
moves with swallowing but not with tongue protrusion.

On palpation, the swelling is [consistency], [tender/non-tender], 
with [smooth/nodular] surface. The trachea is [central/deviated].

I [can/cannot] get below the swelling.

There is [no bruit/a bruit] on auscultation.

Pemberton's test is [positive/negative].

On thyroid status examination, the patient appears [euthyroid/
hyperthyroid/hypothyroid] based on [findings].

There are [no/palpable] cervical lymph nodes.

My clinical impression is [diagnosis] and I would like to 
confirm with TFTs and ultrasound neck."
```
