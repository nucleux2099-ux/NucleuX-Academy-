# 📋 Fluid & Electrolytes — Exam Prep Mode

## 🎯 One-Liners
| Question | Answer |
|----------|--------|
| Total body water (adult male) | 60% of body weight |
| ICF : ECF ratio | 2:1 (40%:20%) |
| Main ICF cation | Potassium (K+) |
| Main ECF cation | Sodium (Na+) |
| Daily maintenance fluid | 30-40 mL/kg/day |
| 4-2-1 rule for hourly maintenance | 4 mL/kg (first 10kg) + 2 mL/kg (next 10kg) + 1 mL/kg (remaining) |
| Normal saline composition | 154 mEq Na, 154 mEq Cl |
| Ringer's lactate composition | Na 130, K 4, Cl 109, Ca 3, Lactate 28 |
| Fluid of choice in hemorrhage | Crystalloid (RL) initially |
| Signs of dehydration | Dry mucosa, reduced skin turgor, tachycardia, oliguria |

## 📊 Daily Requirements
| Component | Amount |
|-----------|--------|
| Water | 30-40 mL/kg |
| Sodium | 1-2 mEq/kg |
| Potassium | 0.5-1 mEq/kg |
| Glucose | 100-150 g (to prevent ketosis) |

## 📝 Hyponatremia
- **Symptoms:** Confusion, seizures, coma
- **Causes:** SIADH, water excess, diuretics
- **Treatment:** Fluid restriction; Hypertonic saline if severe

## 📝 Hyperkalemia
- **ECG:** Tall peaked T waves → Wide QRS → Sine wave
- **Treatment:** Ca gluconate → Insulin/glucose → K binders → Dialysis
