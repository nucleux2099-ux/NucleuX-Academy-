# 📋 Wound Healing — Exam Prep Mode

> **High-yield facts for NEET-PG, INICET, and university exams**

---

## 🎯 One-Liners (Most Asked)

| Question | Answer |
|----------|--------|
| Most important cell in wound healing | **Macrophage** |
| First cell to arrive at wound | Platelet (hemostasis) |
| First inflammatory cell | Neutrophil (24-48 hrs) |
| Main cell of proliferation phase | Fibroblast |
| Predominant collagen in early wound | Type III |
| Predominant collagen in mature scar | Type I |
| Maximum tensile strength achieved | 80% of original (never 100%) |
| Time for maximum strength | 3 months (12 weeks) |
| Vitamin essential for collagen synthesis | Vitamin C (ascorbic acid) |
| Mineral essential for wound healing | Zinc |
| Most common cause of impaired healing | Infection |

---

## 📝 Phases of Wound Healing

| Phase | Duration | Key Events | Cells |
|-------|----------|------------|-------|
| **Hemostasis** | Minutes | Clot formation, vasoconstriction | Platelets |
| **Inflammation** | Day 1-4 | Phagocytosis, debridement | Neutrophils → Macrophages |
| **Proliferation** | Day 4-21 | Granulation, epithelialization | Fibroblasts, Keratinocytes |
| **Remodeling** | Day 21-1 year | Collagen maturation, contraction | Fibroblasts → Myofibroblasts |

---

## 📊 Types of Wound Healing

| Type | Description | Example |
|------|-------------|---------|
| **Primary intention** | Wound edges approximated | Clean surgical wound |
| **Secondary intention** | Left open, heals by granulation | Abscess cavity, burns |
| **Tertiary intention** | Delayed primary closure | Contaminated wound cleaned first |

## 🔬 Collagen Facts (High-Yield)

| Feature | Early Wound | Mature Scar |
|---------|-------------|-------------|
| Collagen type | Type III | Type I |
| Fibers | Thin, disorganized | Thick, organized |
| Strength | Weak | Strong (80% max) |
| Ratio III:I | 3:1 | 1:3 |

**Key enzyme:** Collagen synthesis requires:
- **Prolyl hydroxylase** — needs Vitamin C, Fe, O2
- Deficiency → Scurvy → impaired healing

## ⚠️ Factors Affecting Wound Healing

### Local Factors
| Factor | Mechanism |
|--------|-----------|
| **Infection** | Most common cause; bacteria compete for resources |
| **Blood supply** | Ischemia = no oxygen = no collagen |
| **Foreign body** | Persistent inflammation |
| **Radiation** | Damages fibroblasts, vasculature |
| **Wound tension** | Disrupts healing |
| **Hematoma/Seroma** | Medium for infection |

### Systemic Factors
| Factor | Mechanism |
|--------|-----------|
| **Diabetes** | Microangiopathy + impaired immunity |
| **Malnutrition** | Protein, Vit C, Zinc deficiency |
| **Steroids** | Inhibit inflammation + collagen |
| **Smoking** | Vasoconstriction + CO binding |
| **Age** | Slower fibroblast proliferation |
| **Immunosuppression** | Impaired inflammatory phase |

---

## 🎨 Abnormal Healing

| Type | Location | Timing | Features |
|------|----------|--------|----------|
| **Hypertrophic scar** | Within wound margins | 1-2 months | Raised, regresses with time |
| **Keloid** | Beyond wound margins | Months-years | Raised, does NOT regress |

**High-risk sites for keloid:** Earlobes, sternum, deltoid, upper back

|---------------|--------|----------|

| **EGF** | Platelets, macrophages | Epithelial proliferation |
| **VEGF** | Macrophages | Angiogenesis |
| **FGF** | Macrophages, fibroblasts | Angiogenesis, fibroblast proliferation |

## 📊 Wound Strength Timeline

| Time | Tensile Strength | Clinical Significance |
|------|------------------|----------------------|
| Day 0-5 | ~0% | Held by sutures only |
| Week 1 | 3-5% | Suture removal (face) |
| Week 2 | 10-15% | Suture removal (body) |
| Week 6 | 50% | Normal activities |
| Week 12 | 80% | Maximum strength |
| >1 year | 80% | Never reaches 100% |
