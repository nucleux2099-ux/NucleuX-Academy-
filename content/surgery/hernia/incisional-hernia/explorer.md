# Incisional Hernia: Repair Strategies

> **Source**: Maingot's Part I Ch 13-14, Fischer's Ch 23
> **Topic System**: [[Hernia Index]]
> **Phase**: Shoot

---

## Core Concepts (Shoot Phase)

### Definition & Overview

> [!summary] One-liner
> Incisional hernia is a fascial defect at a previous surgical incision site, occurring in 10-15% of laparotomies, with risk factors including obesity, wound infection, and poor surgical technique.

### Key Points

1. **Incidence**: 10-15% after midline laparotomy; up to 23% at 10 years
2. **Risk factors**: Obesity, wound infection, emergency surgery, re-operation, malnutrition
3. **Mesh is essential**: Suture repair recurrence 30-50%; mesh repair 5-15%
4. **Sublay preferred**: Retromuscular/preperitoneal mesh has best outcomes
5. **Complex hernias**: May need component separation or TAR for closure

### Clinical Pearl
> [!tip] "Mesh wins"
> Always use mesh for incisional hernia repair. Suture-only repair has unacceptable recurrence rates (30-50%).

## Detailed Notes

### Risk Factors for Incisional Hernia

| Category | Risk Factors |
|----------|--------------|
| **Patient** | Obesity (BMI >30), diabetes, smoking, malnutrition, steroids, connective tissue disorders |
| **Wound** | Infection, hematoma, seroma |
| **Technical** | Emergency surgery, midline incision, suture type/technique, re-operation |
| **Postoperative** | Coughing, straining, early physical activity |

### Prevention

**Fascial Closure Technique:**
- Small bites (5-8 mm from edge)
- Small steps (5 mm apart)
- Suture length:wound length ratio ≥4:1
- Slowly absorbable or non-absorbable suture
- Continuous technique (no difference vs interrupted if done properly)

### Classification

**EHS (European Hernia Society) Classification:**
- Location: Midline (M) or Lateral (L)
- Subgroups: M1-M5, L1-L4 (based on anatomic zone)
- Width categories: W1 (<4cm), W2 (4-10cm), W3 (>10cm)
- Recurrence: R (yes/no)

### Preoperative Optimization

| Factor | Intervention |
|--------|--------------|
| **Obesity** | Weight loss (goal BMI <35) |
| **Smoking** | Cessation 4-6 weeks before |
| **Diabetes** | HbA1c optimization (<8%) |
| **Malnutrition** | Nutritional supplementation |
| **Skin condition** | Treat dermatitis, ulceration |
| **Large hernia** | Consider preoperative botox, PPP |

### Repair Options

#### Mesh Placement Positions

```
MESH PLACEMENT LAYERS:

    ONLAY          INLAY           SUBLAY          UNDERLAY
   ┌──────┐      ┌──────┐        ┌──────┐         ┌──────┐
   │ MESH │      │      │        │      │         │      │
   ├──────┤      ├─MESH─┤        │      │         │      │
   │Fascia│      │Fascia│        ├──────┤         ├──────┤
   │      │      │      │        │ MESH │         │      │
   │      │      │      │        ├──────┤         │ MESH │
   │Perit.│      │Perit.│        │Perit.│         │Perit.│
   └──────┘      └──────┘        └──────┘         └──────┘
   
   Risk:SSI     Recurrence      PREFERRED        IP adhesions
                high            Low recurrence   (need composite)
```

**Sublay (Retromuscular/Retrorectus)** = Gold standard
- Mesh behind rectus muscle, above posterior sheath
- Rives-Stoppa technique
- Best incorporation, lowest recurrence

#### Open Repair Techniques

**1. Retrorectus Repair (Rives-Stoppa)**
- Mesh in retromuscular space
- Excellent for midline hernias
- Can extend preperitoneally

**2. Onlay Repair**
- Mesh on top of closed fascia
- Simpler but higher SSI and recurrence
- Larger contact with subcutaneous fat

**3. Intraperitoneal Onlay Mesh (IPOM)**
- Mesh placed intraperitoneally
- Requires composite mesh (e.g., Physiomesh, Proceed)
- Can be open or laparoscopic

#### Laparoscopic Repair

**Laparoscopic IPOM:**
- Ports placed away from hernia
- Reduce contents, lyse adhesions
- Place composite mesh intraperitoneally
- Fixation: Transfascial sutures + tacks

**Advantages:** Less wound complications, faster recovery
**Disadvantages:** Mesh-bowel interface, requires composite mesh

#### Component Separation (for Large Defects)

**When fascia cannot be closed primarily:**

**Anterior Component Separation:**
- Release external oblique lateral to linea semilunaris
- Allows 3-5 cm advancement per side at waist
- Can close defects up to 10-20 cm

**Posterior Component Separation (TAR - Transversus Abdominis Release):**
- Release transversus abdominis from posterior rectus sheath
- Allows similar advancement
- Creates retromuscular space for large mesh
- Less subcutaneous dissection (preserves perforators)

```
TRANSVERSUS ABDOMINIS RELEASE (TAR):

        Rectus abdominis
              │
    ┌─────────┤
    │Post.    │
    │rectus   │──── Release transversus abdominis
    │sheath   │     here
    │         │
    └─────────┘
              │
          Large retromuscular space
          for mesh placement
```

### Outcomes

| Technique | Recurrence | SSO (Surgical Site Occurrence) |
|-----------|------------|-------------------------------|
| Suture repair | 30-50% | 10-15% |
| Onlay mesh | 10-20% | 15-25% |
| Sublay mesh | 5-10% | 5-10% |
| Laparoscopic IPOM | 5-15% | 5-10% |

### Complications

| Complication | Risk Factors | Management |
|--------------|--------------|------------|
| **Seroma** | Large dead space | Drains, expectant |
| **SSI** | Obesity, diabetes, onlay mesh | Antibiotics, may need mesh removal |
| **Recurrence** | Inadequate overlap, tension, infection | Re-repair with mesh |
| **Chronic pain** | Nerve entrapment | May improve; neurectomy rarely |
| **Mesh infection** | Contaminated field | May need staged approach |

## Retrieval Practice

### Quick Recall Questions

1. Q: What is the preferred mesh position for incisional hernia repair?
   A: Sublay (retromuscular/retrorectus) - lowest recurrence, best mesh incorporation

2. Q: What is the suture length to wound length ratio recommended for fascial closure?
   A: ≥4:1 (to reduce incisional hernia risk)

3. Q: When is component separation needed?
   A: When the fascial defect is too large to close primarily (typically >8-10 cm), allowing advancement of the abdominal wall muscles
