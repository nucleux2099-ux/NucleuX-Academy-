# Heart Sounds & Murmurs

> 📚 **Sources:** Guyton & Hall 14th Ed (Ch 23), Bates' Guide to Physical Examination
> **Level:** UG

## Core Concept

Heart sounds are produced by **valve closure** and blood flow turbulence. Normal hearts produce S1 and S2. Additional sounds (S3, S4) and murmurs indicate pathology in adults. Murmurs are characterized by timing, location, radiation, pitch, and response to maneuvers.

## Key Points

- **S1:** Mitral and Tricuspid closure (lub) - Start of systole
- **S2:** Aortic and Pulmonic closure (dub) - End of systole
- **S3:** Early diastole, rapid ventricular filling (pathological in adults)
- **S4:** Late diastole, atrial contraction into stiff ventricle
- **Murmurs:** Turbulent flow through abnormal valves or shunts

## Detailed Content

### Normal Heart Sounds

```
        S1                  S2
        │                   │
   SYSTOLE                  DIASTOLE
        │                   │
    ┌───┴───┐           ┌───┴───┐
   LUB     │           DUB     │
(M1 + T1) │         (A2 + P2) │
        │                   │
  AV valves close    Semilunar valves close
```

| Sound | Timing | Components | Best Heard |
|-------|--------|------------|------------|
| **S1** | Onset of systole | M1 (mitral), T1 (tricuspid) | Apex (mitral) |
| **S2** | End of systole | A2 (aortic), P2 (pulmonic) | Base (aortic/pulmonic areas) |

### S2 Splitting

```
Normal (Physiological) Splitting:
EXPIRATION:  S1 ──────── S2 (A2P2 together)
INSPIRATION: S1 ──────── A2──P2 (split widens)

Why? Inspiration → ↑ Venous return to RV → Delayed P2
```

| Type | Mechanism | Causes |
|------|-----------|--------|
| **Physiological** | ↑ Inspiration widens split | Normal |
| **Wide splitting** | Delayed P2 | RBBB, Pulmonic stenosis |
| **Fixed splitting** | Constant split (no respiratory variation) | **ASD** |
| **Paradoxical** | Split narrows on inspiration | LBBB, Aortic stenosis |

### Extra Heart Sounds

```
S1          S2    S3   S4    S1
│           │     │    │     │
LUB ─────── DUB ──•────•──── LUB
        │         │    │
    SYSTOLE   DIASTOLE │
              │        │
          Rapid    Atrial
          filling  kick
          
S3 = "Kentucky" (ken-TUC-ky)
S4 = "Tennessee" (TEN-nes-see)
S3+S4 = "Gallop" rhythm
```

| Sound | Timing | Mechanism | Pathology |
|-------|--------|-----------|-----------|
| **S3** | Early diastole | Rapid filling into dilated ventricle | HF, MR, Volume overload |
| **S4** | Late diastole | Atrial contraction into stiff ventricle | HTN, HCM, Ischemia |
| **Opening snap** | After S2 | Stenotic mitral valve opening | Mitral stenosis |
| **Ejection click** | After S1 | Stenotic semilunar valve | AS, PS |

### Murmurs: Classification

**By Timing:**
```
         S1              S2              S1
          │               │               │
SYSTOLIC: │──────────────│               │
          │  ejection/   │               │
          │  pansystolic │               │
                                         
DIASTOLIC:               │──────────────│
                         │  early/mid/  │
                         │  presystolic │
                                         
CONTINUOUS: ─────────────────────────────
```

### Systolic Murmurs

| Murmur | Character | Location | Radiation | Maneuvers |
|--------|-----------|----------|-----------|-----------|
| **Aortic Stenosis** | Crescendo-decrescendo, harsh | RUSB (2nd R ICS) | Carotids | ↑ with squatting |
| **Pulmonic Stenosis** | Crescendo-decrescendo | LUSB (2nd L ICS) | Left shoulder | ↑ with inspiration |
| **Mitral Regurgitation** | Holosystolic, blowing | Apex | Left axilla | ↑ with handgrip |
| **Tricuspid Regurgitation** | Holosystolic | LLSB | - | ↑ with inspiration |
| **VSD** | Holosystolic, harsh | LLSB | - | ↑ with handgrip |
| **HCM (HOCM)** | Crescendo-decrescendo | LLSB | - | ↑ with Valsalva, standing |
| **MVP** | Late systolic + click | Apex | - | Click earlier with standing |

### Diastolic Murmurs

| Murmur | Character | Location | Key Features |
|--------|-----------|----------|--------------|
| **Aortic Regurgitation** | Decrescendo, blowing | LUSB/LLSB | Wide pulse pressure, water-hammer pulse |
| **Mitral Stenosis** | Low-pitched rumble | Apex (left lateral) | Opening snap, presystolic accentuation |
| **Pulmonic Regurgitation** | Decrescendo | LUSB | Graham Steell murmur (with pulm HTN) |
| **Tricuspid Stenosis** | Low-pitched rumble | LLSB | ↑ with inspiration |

### Continuous Murmurs

| Cause | Character | Location |
|-------|-----------|----------|
| **Patent Ductus Arteriosus** | "Machinery" murmur | Left infraclavicular |
| **AV fistula** | Continuous bruit | Variable |
| **Venous hum** | Disappears lying down | Supraclavicular |

### Auscultation Areas

```
        ┌───────────────────────┐
        │    A         P       │
        │   (R)       (L)      │  A = Aortic (2nd R ICS)
        │    2nd ICS  2nd ICS  │  P = Pulmonic (2nd L ICS)
        │                      │  T = Tricuspid (4th L ICS - LLSB)
        │         T            │  M = Mitral (5th ICS MCL - Apex)
        │        4th ICS       │
        │                      │
        │              M       │
        │             5th ICS  │
        └───────────────────────┘
```

### Dynamic Maneuvers

| Maneuver | Effect | Increased | Decreased |
|----------|--------|-----------|-----------|
| **Inspiration** | ↑ RV filling | Right-sided murmurs | Left-sided murmurs |
| **Valsalva (strain)** | ↓ Preload | HCM, MVP | AS, MR |
| **Standing** | ↓ Preload | HCM, MVP | AS, MR |
| **Squatting** | ↑ Preload, ↑ Afterload | AS, MR | HCM, MVP |
| **Handgrip** | ↑ Afterload | MR, VSD, AR | AS, HCM |
| **Leg raise** | ↑ Preload | AS, MR | HCM |

## Clinical Correlations

### Aortic Stenosis
- **Causes:** Calcific (elderly), Bicuspid valve (younger)
- **Triad:** Angina, Syncope, Heart failure
- **Exam:** Pulsus parvus et tardus, sustained apex beat
- **Murmur:** Harsh crescendo-decrescendo, radiates to carotids

### Mitral Regurgitation
- **Causes:** MVP, Rheumatic, Ischemic, Endocarditis
- **Exam:** Displaced apex (LV dilation), S3
- **Murmur:** Holosystolic, apex → axilla

### HCM (Hypertrophic Cardiomyopathy)
- **Pathology:** Asymmetric septal hypertrophy, LVOT obstruction
- **Key feature:** Murmur INCREASES with ↓ preload (Valsalva, standing)
- **Warning:** Sudden cardiac death in young athletes

### Mitral Valve Prolapse
- **Epidemiology:** 2-3% population, F > M
- **Exam:** Mid-systolic click + late systolic murmur
- **Click moves earlier** with standing/Valsalva

## High-Yield for Exams

1. **S1 loud in:** Mitral stenosis, short PR interval, tachycardia
2. **S1 soft in:** Mitral regurgitation, long PR interval, poor LV function
3. **Fixed split S2** = ASD (pathognomonic)
4. **Paradoxical split S2** = LBBB or Aortic stenosis
5. **S3** = Volume overload / systolic HF (normal in children, pregnancy)
6. **S4** = Stiff ventricle (HCM, HTN, ischemia) - never normal
7. **Opening snap** closer to S2 = More severe mitral stenosis
8. **Pulsus parvus et tardus** = Aortic stenosis
9. **Austin Flint murmur** = Mid-diastolic rumble in AR (functional MS)
10. **All diastolic murmurs are pathological**

### Grading Murmurs (Levine Scale)

| Grade | Description |
|-------|-------------|
| I | Barely audible |
| II | Soft but readily audible |
| III | Moderately loud |
| IV | Loud with palpable thrill |
| V | Very loud, thrill, audible with edge of stethoscope |
| VI | Audible without stethoscope |

### Memory Aid

- **A**ortic regurgitation
- **I**nfective endocarditis (AR)
